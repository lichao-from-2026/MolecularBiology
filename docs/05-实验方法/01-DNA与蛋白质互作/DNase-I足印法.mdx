# 一、DNase I足印法

## 1. 定义与核心概念

**DNase I足印法**（DNase I Footprinting）是一种用于精确定位蛋白质在DNA上结合位点的实验方法。该技术利用DNase I在DNA骨架上随机切割的特性，当蛋白质与DNA特定区域结合后，会保护该区域免受DNase I切割，从而在测序凝胶上形成"足印"（保护区域），通过分析保护区域可确定蛋白质结合的精确DNA序列。

## 2. 分子机制/原理

DNase I足印法的基本原理是：
1. **DNA片段制备**：制备带有末端标记的单链DNA片段（通常使用放射性标记）
2. **蛋白-DNA结合**：与待测蛋白质在适宜条件下孵育，使其与DNA上的特定位点结合
3. **DNase I有限切割**：加入适量的DNase I进行有限酶切，切割未被保护的DNA区域
4. **DNA纯化**：解交联并纯化DNA片段
5. **变性问题**：使DNA片段变性，形成单链
6. **电泳分析**：在变性聚丙烯酰胺测序凝胶上电泳分离
7. ** autoradiography**：通过放射自显影显示条带模式
8. **足印识别**：比较有蛋白和无蛋白条件下的切割模式，保护区域即为蛋白质结合位点

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| DNA模板（含有待测区域） | PCR扩增或合成获取 |
| [α-³²P]dCTP或其他放射性标记 | 末端标记DNA |
| PCR引物（一个5'端标记） | 扩增DNA片段 |
| 纯化的DNA结合蛋白 | 用于足印分析 |
| DNase I（0.1U/μL新鲜配制） | 切割DNA |
| 10× DNase I缓冲液 | 提供适宜的切割条件 |
| 蛋白酶K | 降解蛋白 |
| 酚/氯仿/异戊醇 | 抽提DNA |
| 乙醇 | 沉淀DNA |
| 甲酰胺上样缓冲液 | 准备变性凝胶上样 |
| 变性聚丙烯酰胺凝胶（6-8%） | DNA片段分离 |
| DNA测序分子量标准 | 读取序列 |
| TBE缓冲液 | 凝胶电泳 |

## 4. 实验步骤

### 步骤1：DNA片段制备与标记

- **目的**：制备带有末端放射性标记的DNA片段，用于后续足印分析
- **操作**：
  1. 设计PCR引物，一条引物的5'端带有放射性磷酸标记（[γ-³²P]ATP）
  2. 以含有目标区域的水稻基因组DNA或质粒为模板
  3. 进行PCR扩增，得到150-300bp的DNA片段
  4. 用2%琼脂糖凝胶电泳分离，切胶回收纯化DNA片段
  5. 测定放射性强度（cpm），确保标记效率
- **注意事项**：
  - DNA片段长度以150-300bp为宜，太短影响分辨率，太长足印定位困难
  - 标记引物时使用高比活性的[³²P]ATP
  - 切胶回收时注意避免污染和放射线暴露

### 步骤2：蛋白质与DNA结合反应

- **目的**：让蛋白质与标记的DNA片段在适宜条件下充分结合
- **操作**：
  1. 准备两个反应管：
     - 实验组：DNA + 蛋白 + DNase I缓冲液
     - 对照组：DNA + 缓冲液（无蛋白）
  2. 配制结合反应体系（20μL）：
     - 10×结合缓冲液：2μL
     - 标记的DNA片段：~10,000 cpm
     - 纯化蛋白或核提取物：适量
     - 水：补足至20μL
  3. 室温孵育20-30分钟
- **注意事项**：
  - 蛋白浓度需要优化，确保有足够的结合但不过量
  - 可设置一系列蛋白浓度梯度
  - 如使用核提取物，应预先用核酸酶处理去除内源DNA

### 步骤3：DNase I有限酶切

- **目的**：对蛋白-DNA复合物进行有限酶切，使被蛋白保护的区域不被切割
- **操作**：
  1. 将DNase I用1× DNase I缓冲液新鲜稀释至0.1U/μL
  2. 向实验组和对照组反应管中各加入适量DNase I（通常1-2μL）
  3. 室温孵育1-2分钟
  4. 立即加入终止缓冲液（含有EDTA和SDS）终止反应
  5. 加入蛋白酶K，55°C孵育15分钟降解蛋白
- **注意事项**：
  - DNase I浓度和孵育时间是关键参数，需要优化
  - DNase I用量过多会导致DNA完全降解，用量过少则切割不足
  - 酶切时间必须精确控制
  - 需确保两组反应的DNase I处理条件完全一致

### 步骤4：DNA纯化

- **目的**：去除蛋白质和其他杂质，获得纯净的DNA片段用于电泳分析
- **操作**：
  1. 向终止反应的样品中加入等体积酚/氯仿/异戊醇
  2. 充分混匀，12000rpm离心5分钟
  3. 收集上层水相
  4. 加入1/10体积3M醋酸钠（pH 5.2）和2倍体积无水乙醇
  5. -20°C沉淀过夜或-80°C沉淀1小时
  6. 12000rpm离心15分钟，弃上清
  7. 用75%乙醇洗涤沉淀2次
  8. 晾干后溶于20μL无菌水中
- **注意事项**：
  - 纯化过程中避免剧烈操作，防止DNA断裂
  - 乙醇沉淀后的DNA沉淀要充分晾干但不能过度干燥
  - 纯化后尽快进行电泳分析

### 步骤5：变性凝胶电泳

- **目的**：分离不同长度的DNA片段，确定蛋白保护区域的位置
- **操作**：
  1. 配制6-8%变性聚丙烯酰胺凝胶（含有8M尿素）
  2. 将凝胶板组装好，加入1×TBE缓冲液
  3. 预电泳30-60分钟（60W）使凝胶升温至约50°C
  4. 向DNA样品中加入等体积甲酰胺上样缓冲液
  5. 95°C加热5分钟使DNA变性
  6. 立即冰浴冷却
  7. 上样（每个样品5-10μL）
  8. 电泳（60W，1-3小时，根据片段大小调整）
  9. 电泳结束后，小心剥离凝胶
  10. 将凝胶转移到滤纸上
  11. 用真空干燥器在80°C干燥30-60分钟
  12. 放置X光片或磷屏进行曝光（-80°C，12-24小时）
- **注意事项**：
  - 凝胶必须完全聚合，否则条带会出现畸变
  - 变性条件必须充分，确保DNA完全解链
  - 电泳过程中要保持恒温，尿素浓度要均匀

### 步骤6：结果分析

- **目的**：读取序列，确定蛋白质在DNA上的精确结合位点
- **操作**：
  1. 显影X光片或读取磷屏
  2. 比较实验组（有蛋白）和对照组（无蛋白）的条带模式
  3. 在实验组中出现的条带缺失或减弱的区域即为"足印"
  4. 与DNA测序分子量标准对比，确定足印的精确位置
  5. 读取足印区域的DNA序列
- **注意事项**：
  - 足印通常为15-30bp的保护区域
  - 某些蛋白可能同时保护多个区域
  - 阴性对照（无蛋白）的切割模式应该均匀随机

## 5. 应用场景

- **转录因子结合位点精确定位**：确定OsWRKY、OsNAC等转录因子在水稻基因启动子上的精确结合序列
- **DNA-蛋白相互作用结构研究**：研究蛋白质与DNA的主沟和次沟相互作用
- **顺式作用元件鉴定**：鉴定水稻基因启动子中的转录因子结合位点
- **突变分析**：验证点突变对蛋白质结合的影响
- **化合物结合位点研究**：筛选与DNA结合的小分子化合物

## 6. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **随机引物延伸足印法** | 使用随机引物延伸 | 更大范围的DNA区域分析 |
| **镁离子依赖核酸酶足印法** | 使用核酸酶替代DNase I | 提高切割特异性 |
| **光交联足印法** | 使用光交联剂 | 共价交联蛋白-DNA |
| **毛细管电泳足印法** | 使用毛细管电泳 | 高通量分析 |
| **湿热法足印法** | 使用高温使DNA断裂 | 无需酶 |

## 7. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| DNA完全降解 | DNase I用量过大 | 减少酶量，缩短酶切时间 |
| 无明显足印 | 蛋白未与DNA结合 | 优化结合条件，验证蛋白活性 |
| 条带不清晰 | DNA降解或电泳问题 | 检查样品质量，检查凝胶 |
| 重复性差 | DNase I处理不一致 | 精确控制酶切条件 |
| 背景高 | 非特异性切割或污染 | 使用新鲜DNase I |

## 8. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 可精确确定蛋白质在DNA上的结合位点（单碱基分辨率） | 操作复杂，需要高活性的DNase I |
| 可同时分析较大范围的DNA区域 | 需要放射性操作 |
| 结果直观，可直接读取序列 | 对蛋白纯度要求较高 |
| 可用于研究多个蛋白结合位点 | 耗时较长（3-4天） |
| 可定量分析结合强度 | DNase I切割存在序列偏好性 |
