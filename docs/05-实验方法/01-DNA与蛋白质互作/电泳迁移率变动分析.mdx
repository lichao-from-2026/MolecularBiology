# 一、电泳迁移率变动分析（EMSA）

## 1. 定义与核心概念

**电泳迁移率变动分析**（Electrophoretic Mobility Shift Assay, EMSA）是一种用于研究**体外**DNA与蛋白质相互作用的实验方法。该技术基于蛋白质与DNA结合后会改变DNA在凝胶电泳中的迁移率这一原理，通过非变性聚丙烯酰胺凝胶电泳分离结合与未结合的DNA片段，观察蛋白质-DNA复合物形成的滞后条带，从而鉴定蛋白质的DNA结合活性。

## 2. 分子机制/原理

EMSA技术的基本原理是：
1. **DNA探针标记**：用放射性同位素（如³²P）或非放射性标记物（如生物素、荧光素）标记DNA探针
2. **蛋白质与DNA结合**：将标记的DNA探针与蛋白质样品在适宜条件下孵育，形成蛋白质-DNA复合物
3. **非变性凝胶电泳分离**：通过非变性聚丙烯酰胺凝胶电泳分离复合物与游离DNA
4. **检测分析**：通过放射自显影、化学发光或荧光检测观察电泳结果

当DNA与蛋白质结合后，形成的复合物分子量增大且形状改变，在凝胶中的迁移速度减慢，从而在凝胶上形成滞后于游离DNA的条带。

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| DNA探针（20-50bp） | 含有目标结合位点的双链DNA |
| [γ-³²P]ATP或生物素标记试剂 | 标记DNA探针 |
| T4多核苷酸激酶 | 催化放射性同位素标记 |
| 蛋白质样品（纯化蛋白或细胞核提取物） | 待检测的DNA结合蛋白 |
| 结合缓冲液（10× Binding Buffer） | 提供适宜的结合条件 |
| Poly(dI-dC)（1μg/μL） | 竞争DNA，减少非特异性结合 |
| 非特异性竞争DNA（如鲑鱼精DNA） | 消除非特异性结合 |
| 非变性聚丙烯酰胺凝胶（4-6%） | 分离蛋白质-DNA复合物 |
| 电泳缓冲液（0.5×TBE） | 用于凝胶电泳 |
| 上样缓冲液（6×） | 准备样品上样 |
| 放射自显影设备或化学发光检测系统 | 检测标记的DNA |

## 4. 实验步骤

### 步骤1：DNA探针制备与标记

- **目的**：获得带有可检测标记的DNA探针，用于追踪蛋白质的结合
- **操作**：
  1. 合成单链寡核苷酸（20-50bp），包含目标转录因子结合位点
  2. 将互补的单链寡核苷酸等摩尔比混合
  3. 95°C加热5分钟，然后缓慢冷却至室温进行退火，形成双链DNA探针
  4. **放射性标记**：使用T4多核苷酸激酶和[γ-³²P]ATP标记探针末端
  5. **生物素标记**：使用Terminal Deoxynucleotidyl Transferase将生物素标记到探针3'末端
  6. 使用离心柱或乙醇沉淀纯化标记的DNA探针，去除未结合的标记物
- **注意事项**：
  - 探针序列需包含已报道的或预测的转录因子结合元件
  - 放射性操作需符合辐射安全规定
  - 标记效率影响后续检测灵敏度
  - 探针需保存于-20°C，避免反复冻融

### 步骤2：蛋白质与DNA结合反应

- **目的**：让蛋白质与DNA探针在体外适宜条件下充分结合
- **操作**：
  1. 准备结合反应体系（总体积20μL）：
     - 10× Binding Buffer：2μL
     - Poly(dI-dC)（1μg/μL）：1-2μL
     - 标记的DNA探针（~10fmol）：1-2μL
     - 蛋白质样品：适量（根据预实验确定）
     - 无核酸酶的水：补足至20μL
  2. 室温孵育20-30分钟，使蛋白质与DNA充分结合
  3. 如进行竞争实验，在加入探针前先加入100-200倍摩尔比的未标记竞争DNA
- **注意事项**：
  - 蛋白质浓度需优化，浓度过高可能导致假阳性
  - Poly(dI-dC)用于阻断非特异性结合，但用量过多会竞争特异性结合
  - 结合缓冲液中的Mg²⁺、Zn²⁺等金属离子对某些转录因子的结合活性至关重要
  - 某些转录因子需要添加ATP或其他辅因子才能有效结合DNA

### 步骤3：非变性聚丙烯酰胺凝胶制备

- **目的**：制备适合分离蛋白质-DNA复合物的非变性凝胶
- **操作**：
  1. 清洗玻璃板并组装凝胶模具
  2. 配制4-6%非变性聚丙烯酰胺凝胶：
     - 30%丙烯酰胺溶液：2mL（对于4%凝胶）
     - 10×TBE缓冲液：1mL
     - 50%甘油：1mL
     - 无菌水：补足至10mL
     - 10%过硫酸铵（APS）：100μL
     - TEMED：10μL
  3. 立即灌入模具，插入梳子
  4. 室温聚合30-60分钟
- **注意事项**：
  - 凝胶浓度影响分离效果：4%适用于大片段复合物（>100bp），6%适用于小片段探针
  - 丙烯酰胺有毒，操作时需佩戴手套
  - TEMED和APS需新鲜配制以确保凝胶聚合良好

### 步骤4：凝胶电泳

- **目的**：分离结合与未结合的DNA探针，观察蛋白质-DNA复合物的形成
- **操作**：
  1. 将凝胶板放入电泳槽中
  2. 加入0.5×TBE电泳缓冲液，确保缓冲液覆盖凝胶
  3. 预电泳30分钟（100V），去除凝胶中的杂质
  4. 向每个样品中加入1/6体积的6×上样缓冲液，轻轻混匀
  5. 上样（每个样品10-20μL）
  6. 在4°C或冰浴条件下电泳（100-150V，1-2小时）
  7. 电泳直至溴酚蓝迁移至凝胶的2/3处
- **注意事项**：
  - 必须使用非变性条件，保持蛋白质-DNA复合物的完整性
  - 低温电泳防止复合物在电场作用下解离
  - 电压过高会导致凝胶发热，使复合物变性
  - 上样量不宜过大，以免条带变形

### 步骤5：检测与分析

- **目的**：观察蛋白质与DNA的结合情况，鉴定特异性结合
- **操作**：
  1. **放射性标记探针检测**：
     - 将凝胶干燥后放入暗盒
     - 在暗室中放置X光片或磷屏
     - 曝光后显影和定影
  2. **生物素标记探针检测**：
     - 将凝胶转移至尼龙膜或Whatman纸
     - 使用紫外交联仪（254nm）交联DNA至膜上
     - 加入HRP标记的链霉亲和素
     - 使用化学发光底物进行曝光检测
- **注意事项**：
  - 设置正确的对照：游离探针对照（无蛋白）、竞争实验（加入未标记探针）
  - 特异性结合会被未标记探针竞争掉，形成超迁移（supershift）则说明是目标蛋白

## 5. 结果分析

### 正常EMSA结果模式

| 条带类型 | 位置 | 含义 |
|---------|------|------|
| 游离探针 | 凝胶前沿 | 无蛋白质结合的探针 |
| 滞后条带（Shift） | 远离前沿 | 蛋白质-探针复合物 |
| 超滞后条带（Supershift） | 更远离前沿 | 抗体-蛋白质-探针三元复合物 |

### 竞争实验分析

- **特异性结合**：加入100倍未标记探针后，滞后条带消失或显著减弱
- **非特异性结合**：即使加入大量未标记探针，滞后条带仍然存在

## 6. 应用场景

- **转录因子DNA结合活性检测**：快速验证OsWRKY、OsNAC、OsbZIP等转录因子是否与目标DNA元件结合
- **DNA结合蛋白的分离与鉴定**：检测水稻蛋白粗提液中的DNA结合活性
- **蛋白质-DNA结合亲和力分析**：通过竞争实验测定结合常数
- **化合物筛选**：筛选影响转录因子-DNA结合的小分子化合物
- **突变分析**：研究转录因子结合位点突变对蛋白质结合的影响

## 7. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **Supershift EMSA** | 加入特异性抗体形成三元复合物 | 验证结合蛋白的身份 |
| **竞争EMSA** | 使用未标记探针竞争 | 验证结合特异性 |
| **荧光EMSA** | 使用荧光素标记探针 | 无放射性污染 |
| **毛细管电泳EMSA** | 使用毛细管电泳分离 | 高通量筛选 |
| **蛋白质检测EMSA** | 检测蛋白质而非DNA | 研究蛋白-蛋白互作对DNA结合的影响 |

## 8. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 无滞后条带 | 蛋白失活或探针序列错误 | 验证蛋白活性，更换探针 |
| 条带弥散 | 蛋白或探针降解 | 使用新鲜试剂，加入蛋白酶抑制剂 |
| 背景高 | 非特异性结合过强 | 增加Poly(dI-dC)用量，加强洗涤 |
| 游离探针条带太弱 | 标记效率低或探针降解 | 优化标记反应，检测探针完整性 |
| 竞争实验失败 | 竞争探针序列不合适 | 设计更长的竞争探针 |

## 9. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 操作相对简单，耗时短（1-2天） | 只能在体外研究，不能反映体内真实情况 |
| 灵敏度高，可检测低亲和力结合 | 无法确定蛋白质在DNA上的精确结合位点 |
| 可定量分析结合亲和力 | 可能存在非特异性结合导致的假阳性 |
| 可用于筛选影响结合的化合物 | 对于复杂的蛋白质复合物分析能力有限 |
| 成本相对较低 | 放射性标记存在安全隐患（非放射性标记可避免） |
| 所需蛋白量少 | 需要纯化的蛋白或高质量的核提取物 |
