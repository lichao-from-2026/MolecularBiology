# 一、DNA亲和层析技术

## 1. 定义与核心概念

**DNA亲和层析技术**（DNA Affinity Chromatography）是一种利用DNA与蛋白质的特异性结合来分离和纯化DNA结合蛋白的实验方法。该技术将特定的DNA序列（通常包含转录因子结合位点）固定在层析介质上，当蛋白质样品通过层析柱时，与DNA序列特异性结合的蛋白质会被保留，而其他蛋白质则流出，从而实现目标蛋白质的富集和纯化。

## 2. 分子机制/原理

DNA亲和层析技术的基本原理是：
1. **DNA探针固定**：将含有目标结合位点的双链DNA序列共价连接到活化的层析介质（如Sepharose、磁珠）上
2. **样品加载**：将蛋白质样品缓慢通过DNA亲和层析柱
3. **特异性结合**：与固定化DNA序列具有亲和力的蛋白质被保留在柱子上
4. **洗涤**：用缓冲液洗涤层析柱，去除非特异性结合的蛋白质
5. **竞争洗脱**：加入高浓度游离DNA或改变缓冲条件，将特异性结合的蛋白质洗脱下来
6. **收集与分析**：收集洗脱液，进行SDS-PAGE和质谱分析鉴定

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| DNA亲和配体（含有转录因子结合位点） | 固定在层析介质上的DNA |
| 活化的Sepharose 4B或磁珠 | DNA固定的载体基质 |
| CNBr（氰溴化物）或 Streptavidin | 活化介质或生物素-链霉亲和素连接 |
| 蛋白质样品（细胞核提取物） | 待分离的DNA结合蛋白 |
| 结合缓冲液（Low Salt Buffer） | 提供适宜的结合条件 |
| 洗涤缓冲液（Medium Salt Buffer） | 去除非特异性结合的蛋白 |
| 洗脱缓冲液（High Salt Buffer或竞争洗脱液） | 洗脱特异性结合的蛋白 |
| 层析柱（空柱或预装柱） | 进行亲和层析的装置 |
| 核酸酶抑制剂 | 防止DNA降解 |
| 蛋白酶抑制剂 cocktail | 防止蛋白降解 |

## 4. 实验步骤

### 步骤1：DNA亲和层析柱的制备

- **目的**：将DNA探针固定在层析介质上，制备具有DNA结合活性的亲和层析柱
- **操作**：
  1. **方法一：CNBr活化法**
     - 将Sepharose 4B在水中溶胀
     - 用冰冷的水洗涤Sepharose
     - 溶于CNBr溶液（1g CNBr溶于20mL水中），在4°C、pH11-12条件下活化30分钟
     - 迅速用冰冷的水洗涤活化的Sepharose
     - 将活化的Sepharose与氨基修饰的DNA探针（0.5-1mg DNA/mL介质）在pH8-9条件下4°C孵育12-16小时
     - 用乙醇胺封闭未反应的活性基团
  2. **方法二：生物素-链霉亲和素法**
     - 将生物素修饰的DNA探针与链霉亲和素修饰的磁珠或介质混合
     - 室温孵育30-60分钟
     - 洗涤去除未结合的DNA
  3. 用结合缓冲液平衡DNA-介质复合物
  4. 将DNA-介质装入空柱或直接使用磁珠
- **注意事项**：
  - DNA探针长度通常为30-100bp，包含2-3个串联的转录因子结合位点
  - CNBr有毒且不稳定，需现用现配
  - 固定化效率影响最终纯化效果
  - 封闭步骤必须充分以减少非特异性结合

### 步骤2：蛋白质样品准备

- **目的**：制备含有目标DNA结合蛋白的样品，并去除干扰物质
- **操作**：
  1. 培养水稻细胞或组织，收获后用预冷的PBS洗涤
  2. 加入细胞裂解缓冲液（含核酸酶抑制剂和蛋白酶抑制剂）
  3. 在冰上裂解30分钟，期间 vortex 3-4次
  4. 超声波破碎（功率150W，工作3秒，间隔5秒，重复10次）
  5. 4°C，12000rpm离心20分钟
  6. 收集上清即为总蛋白提取物
  7. 如需核蛋白，采用差速离心分离细胞核后再进行超声破碎
  8. 用结合缓冲液平衡蛋白样品（降低盐浓度至~100mM NaCl）
- **注意事项**：
  - 所有操作必须在4°C进行，防止蛋白降解
  - 蛋白酶抑制剂必须新鲜配制
  - 蛋白样品需充分去除核酸（用核酸酶处理），否则会干扰结合
  - 样品盐浓度过高会降低DNA-蛋白结合效率

### 步骤3：样品加载与结合

- **目的**：让蛋白质样品与固定在柱上的DNA探针充分接触并结合
- **操作**：
  1. 将DNA亲和层析柱安装在 AKTA 或类似层析系统上
  2. 用10-20倍柱体积的结合缓冲液平衡柱子
  3. 将蛋白样品缓慢加载到柱子上（流速0.2-0.5mL/min）
  4. 收集流出液（Flow-through），可再次上样以提高结合效率
  5. 用10倍柱体积的结合缓冲液洗涤柱子，去除未结合的蛋白
  6. 监测UV280nm吸收，当吸收值回到基线后停止洗涤
- **注意事项**：
  - 上样流速应缓慢，确保蛋白有足够时间与DNA结合
  - 可将样品循环通过柱子2-3次以提高结合效率
  - 洗涤时流速可以适当加快
  - 必须记录所有收集的组分，便于后续分析

### 步骤4：洗涤与洗脱

- **目的**：去除非特异性结合的蛋白质，保留并收集特异性结合的蛋白质
- **操作**：
  1. **洗涤步骤**：
     - 用10倍柱体积的Low Salt Buffer（100mM NaCl）洗涤
     - 用5倍柱体积的Medium Salt Buffer（250mM NaCl）洗涤
     - 用5倍柱体积的High Salt Buffer（500mM NaCl）洗涤
     - 每次换缓冲液时收集洗脱液作为对照
  2. **洗脱步骤（选择一种）**：
     - **高盐洗脱**：用含有1M NaCl的缓冲液洗脱
     - **竞争洗脱**：加入含有过量游离目标DNA的缓冲液进行竞争洗脱
     - **梯度洗脱**：从100mM到1M NaCl线性梯度洗脱
  3. 分部收集洗脱液（每管0.5-1mL）
  4. 监测UV280nm吸收峰，收集目的蛋白峰
- **注意事项**：
  - 竞争洗脱可以获得更纯净的特异性结合蛋白
  - 高盐洗脱可能使某些蛋白变性
  - 分部收集便于确定目的蛋白所在管数

### 步骤5：蛋白质分析与鉴定

- **目的**：鉴定洗脱液中的蛋白质成分，确认目标蛋白的存在
- **操作**：
  1. 取10-20μL各组分加入SDS-PAGE上样缓冲液，95°C加热5分钟
  2. 进行SDS-PAGE电泳（12%凝胶）
  3. 用银染或考马斯亮蓝染色观察蛋白条带
  4. 对感兴趣的条带进行质谱鉴定
  5. 或进行Western blot验证目标蛋白
  6. 合并含有目标蛋白的组分
  7. 如需浓缩，可用超滤离心管进行浓缩
- **注意事项**：
  - 银染比考马斯亮蓝更灵敏，可检测低丰度蛋白
  - 质谱鉴定需要足够的蛋白量
  - 蛋白浓缩时注意防止沉淀

## 5. 应用场景

- **DNA结合蛋白的分离与纯化**：从水稻细胞总蛋白中富集特定的DNA结合蛋白
- **转录因子筛选**：利用含有特定结合位点的DNA探针筛选未知转录因子
- **转录因子辅因子鉴定**：研究与特定转录因子共同结合的辅因子蛋白
- **DNA-蛋白相互作用网络分析**：构建水稻转录因子调控网络
- **化合物筛选**：筛选能够影响特定DNA-蛋白结合的小分子化合物

## 6. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **单链DNA亲和层析** | 使用单链DNA | 分离单链DNA结合蛋白 |
| **核酸酶保护亲和层析** | 结合核酸酶切割 | 分离具有特殊结合模式的蛋白 |
| **诱饵-猎物系统** | 设计诱饵DNA捕获猎物蛋白 | 筛选蛋白复合物 |
| **磁珠-based 亲和纯化** | 使用磁珠作为载体 | 快速分离，无需层析柱 |
| **泛素化介导的亲和纯化** | 利用泛素化系统 | 研究泛素化修饰的DNA结合蛋白 |

## 7. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 结合效率低 | DNA固定量不足或蛋白失活 | 增加DNA固定量，验证蛋白活性 |
| 非特异性结合严重 | 柱子未充分封闭或洗涤不充分 | 加强封闭步骤，增加洗涤次数 |
| 蛋白洗脱不下来 | 洗脱条件太温和 | 增加盐浓度或使用竞争洗脱 |
| 蛋白条带复杂 | 存在多个非特异性结合蛋白 | 优化洗涤条件或使用更特异的DNA探针 |
| DNA探针降解 | 核酸酶污染 | 加入新鲜核酸酶抑制剂 |

## 8. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 可直接从复杂蛋白混合物中纯化DNA结合蛋白 | 实验步骤较为复杂，耗时较长 |
| 可获得接近天然状态的蛋白 | 需要制备高质量的DNA亲和柱 |
| 可富集低丰度的DNA结合蛋白 | DNA-蛋白相互作用可能受固定化影响 |
| 可同时捕获多个相关蛋白（蛋白复合物） | 存在非特异性结合，需要优化条件 |
| 可与质谱联用进行蛋白组学分析 | 某些蛋白可能与固定化DNA亲和力较弱 |
| 可重复使用（取决于配体稳定性） | 每次实验需要新鲜蛋白样品 |
