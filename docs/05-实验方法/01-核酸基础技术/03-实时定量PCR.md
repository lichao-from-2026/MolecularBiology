# 02-实时定量PCR

## 1. 定义与核心概念

**实时定量PCR**（Real-time Quantitative PCR，qPCR）是一种在PCR反应过程中实时监测DNA扩增情况的技术。它通过荧光信号的积累实时监测PCR产物的生成，从而实现对初始模板的定量分析。qPCR是基因表达定量、病原体检测和基因突变分析的重要工具。

## 2. 分子机制/原理

实时定量PCR的基本原理是：
1. **荧光标记**：使用荧光染料或荧光探针标记PCR产物
2. **实时监测**：在每个PCR循环结束后检测荧光信号强度
3. **数据分析**：根据荧光信号的增长曲线计算初始模板量
4. **定量方法**：
   - **绝对定量**：通过标准曲线计算模板的绝对量
   - **相对定量**：通过比较Ct值计算基因表达的相对变化

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| DNA/RNA模板 | 待检测的核酸样品 |
| 引物（正向和反向） | 与目标序列互补的寡核苷酸 |
| 荧光染料（如SYBR Green I） | 与双链DNA结合产生荧光 |
| 荧光探针（如TaqMan探针） | 与目标序列特异性结合 |
| 反转录酶（用于RNA模板） | 将RNA反转录为cDNA |
| qPCR专用酶（如Taq DNA聚合酶） | 催化DNA合成 |
| dNTP混合物 | 合成DNA的原料 |
| 10× qPCR缓冲液 | 提供适宜的反应环境 |
| MgCl2溶液 | 影响酶活性和引物退火 |
| 无菌水 | 配制反应体系 |
| 实时定量PCR仪 | 实时监测荧光信号 |
| 96孔或384孔PCR板 | 反应容器 |
| 封板膜 | 防止蒸发和污染 |

## 4. 实验步骤

### 步骤1：RNA提取与反转录（用于基因表达分析）

- **目的**：从样品中提取RNA并反转录为cDNA
- **操作**：
  1. 使用RNA提取试剂盒提取总RNA
  2. 测定RNA浓度和纯度（OD260/280=1.8-2.0）
  3. 使用反转录试剂盒将RNA反转录为cDNA
  4. 稀释cDNA至合适浓度
- **注意事项**：
  - 操作过程应避免RNA酶污染
  - 反转录反应应包含RNA酶抑制剂
  - 应设置无反转录对照（NRT）

### 步骤2：qPCR反应体系的配制

- **目的**：准备含有所有必要成分的qPCR反应混合液
- **操作**：
  1. 在冰上配制反应体系（总体积20μL）：
     - 2× qPCR Master Mix：10μL
     - 正向引物（10μM）：0.8μL
     - 反向引物（10μM）：0.8μL
     - 模板（cDNA或DNA）：2μL
     - 无菌水：6.4μL
  2. 轻轻混匀，短暂离心
  3. 将反应液加入96孔板中
  4. 封板膜密封
- **注意事项**：
  - 所有操作应在冰上进行
  - 应设置3-4个技术重复
  - 应设置阳性对照和阴性对照

### 步骤3：qPCR循环参数设置

- **目的**：设置适合目标片段扩增的温度循环参数
- **操作**：
  1. 预变性：95°C，30秒
  2. 循环（40次）：
     - 变性：95°C，5秒
     - 退火/延伸：60°C，30秒（同时采集荧光信号）
  3. 熔解曲线分析：95°C，15秒；60°C，1分钟；95°C，15秒
- **注意事项**：
  - 退火/延伸温度应根据引物的Tm值设置
  - 熔解曲线分析用于验证产物的特异性
  - 循环次数一般为40次，避免过度扩增

### 步骤4：数据分析

- **目的**：分析qPCR结果，计算基因表达水平或模板浓度
- **操作**：
  1. **基线设置**：选择合适的循环范围作为基线
  2. **阈值设置**：将阈值设置在扩增曲线的指数增长期
  3. **Ct值读取**：记录每个样品的Ct值
  4. **数据分析**：
     - **绝对定量**：根据标准曲线计算模板浓度
     - **相对定量**：使用2^(-ΔΔCt)法计算相对表达量
  5. **熔解曲线分析**：验证产物的特异性
- **注意事项**：
  - 阈值应设置在所有样品的指数增长期
  - 应使用看家基因作为内参
  - 熔解曲线应显示单一峰值，无杂峰

## 5. 应用场景

- **基因表达定量**：分析不同条件下基因表达的变化
- **病原体检测**：定量检测病毒、细菌等病原体的载量
- **基因突变分析**：检测基因的突变频率
- **拷贝数变异分析**：检测基因的拷贝数变化
- **miRNA表达分析**：分析小RNA的表达水平
- **药物疗效监测**：监测治疗过程中病原体的清除情况

## 6. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **SYBR Green I qPCR** | 使用荧光染料，成本低 | 一般基因表达分析 |
| **TaqMan探针 qPCR** | 使用荧光探针，特异性高 | 病原体检测，低丰度模板 |
| **分子信标 qPCR** | 使用发夹结构探针 | 基因突变检测 |
| **LNA探针 qPCR** | 使用锁核酸探针，特异性高 | 单核苷酸多态性检测 |
| **数字PCR** | 单分子水平定量，无需标准曲线 | 稀有突变检测，低丰度模板 |

## 7. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 无扩增信号 | 模板浓度低或质量差 | 增加模板量，提高模板质量 |
| 荧光信号弱 | 荧光染料浓度低或探针设计不当 | 优化染料浓度，重新设计探针 |
| 熔解曲线多峰 | 非特异性扩增或引物二聚体 | 优化引物设计，提高退火温度 |
| 标准曲线线性差 | 标准品稀释不准确 | 确保标准品稀释准确，使用新鲜标准品 |
| 内参基因不稳定 | 内参基因选择不当 | 选择稳定表达的内参基因，使用多个内参 |

## 8. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 实时监测，结果直观 | 设备成本较高 |
| 定量准确，灵敏度高 | 对模板质量要求高 |
| 操作简单，自动化程度高 | 荧光染料或探针成本较高 |
| 可同时分析多个样品 | 数据分析较为复杂 |
| 可检测低丰度模板 | 需要优化反应条件 |
