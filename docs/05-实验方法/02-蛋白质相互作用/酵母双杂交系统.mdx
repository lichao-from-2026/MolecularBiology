# 一、酵母双杂交系统

## 1. 定义与核心概念

**酵母双杂交系统**（Yeast Two-Hybrid System）是一种强大的遗传学方法，用于筛选和验证蛋白质-蛋白质相互作用。该技术基于转录激活原理，将待研究的目标蛋白（诱饵，Bait）与转录激活域（如GAL4 BD或LexA）融合表达，潜在的作用蛋白（猎物，Prey）与转录激活域（如GAL4 AD或B42）融合表达。如果两个蛋白发生相互作用，诱饵和猎物的接近会导致转录激活域形成功能性转录因子，激活报告基因的表达。

## 2. 分子机制/原理

酵母双杂交系统的基本原理是：
1. **构建诱饵载体**：将目标蛋白基因克隆到含有DNA结合域（BD）的载体中
2. **构建猎物文库**：将水稻cDNA文库克隆到含有转录激活域（AD）的载体中
3. **共转化酵母**：将诱饵和猎物载体共转化入特定酵母菌株
4. **报告基因筛选**：通过营养缺陷标记（HIS3、LEU2）和色素形成（LacZ）筛选阳性互作克隆
5. **互作验证**：通过β-半乳糖苷酶活性检测和回转实验验证互作的真实性
6. **序列鉴定**：回收阳性克隆中的猎物质粒并进行测序鉴定

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| 酵母菌株（AH109、Y2HGold、Mating型菌株） | 报告菌株 |
| 诱饵载体（pGBKT7等） | 表达GAL4 BD-诱饵蛋白融合 |
| 猎物载体（pGADT7等） | 表达GAL4 AD-猎物蛋白融合 |
| 水稻cDNA文库 | 提供猎物蛋白来源 |
| 限制性内切酶、T4 DNA连接酶 | 载体构建 |
| 酵母转化试剂（PEG/LiAc） | 化学法转化酵母 |
| SD/-Trp/-Leu缺陷培养基 | 一级筛选 |
| SD/-Trp/-Leu/-His缺陷培养基（含3-AT） | 二级筛选 |
| SD/-Trp/-Leu/-His/-Ade缺陷培养基 | 严格筛选 |
| X-α-Gal | 检测α-半乳糖苷酶活性 |
| Aureobasidin A（AbA） | 抗性筛选 |
| β-半乳糖苷酶检测试剂 | 定量验证 |
| 酵母质粒提取试剂盒 | 回收猎物质粒 |
| 大肠杆菌感受态细胞 | 扩增回收的质粒 |

## 4. 实验步骤

### 步骤1：诱饵载体构建与验证

- **目的**：将目标蛋白基因克隆到诱饵载体中，并验证其转录激活和自激活活性
- **操作**：
  1. 设计引物，PCR扩增目标蛋白编码序列（不含终止密码子）
  2. 使用限制性内切酶酶切诱饵载体（pGBKT7）
  3. 将目标基因片段插入到BD下游的多克隆位点
  4. 连接反应，转化大肠杆菌，筛选阳性克隆
  5. 测序验证阅读框正确
  6. **毒性检测**：转化酵母Y2HGold，检测诱饵蛋白是否影响酵母生长
  7. **自激活检测**：在SD/-Trp/-His平板上检测是否无需猎物即可生长
- **注意事项**：
  - 诱饵蛋白不能有转录激活活性，否则导致假阳性
  - 如有明显自激活，需截短诱饵蛋白去除激活域
  - 诱饵蛋白也不能对酵母有毒性

### 步骤2：酵母感受态制备

- **目的**：制备高质量的酵母感受态细胞用于转化
- **操作**：
  1. 从YPD平板挑取新鲜酵母单克隆，接种于5mL YPD液体培养基
  2. 30°C，200rpm振荡培养过夜（OD600约0.8-1.0）
  3. 取2-3mL培养物转接于50mL YPD中
  4. 30°C，200rpm振荡培养4-6小时（OD600约0.4-0.6）
  5. 室温，1000×g离心5分钟收集细胞
  6. 弃上清，用10mL无菌水重悬
  7. 室温，1000×g离心5分钟
  8. 弃上清，用1mL 1×TE/LiAc缓冲液重悬
  9. 立即使用或分装后-80°C保存
- **注意事项**：
  - 细胞密度OD600约0.4-0.6时状态最佳
  - 所有操作需无菌，避免污染
  - 感受态细胞最好现做现用

### 步骤3：文库筛选（共转化法）

- **目的**：将诱饵载体和猎物文库共转化入酵母，筛选蛋白互作克隆
- **操作**：
  1. 取50μL感受态细胞，加入：
     - 诱饵质粒（100-500ng）
     - 猎物文库质粒（500ng-1μg）
     - 鲑鱼精DNA（5μL，10mg/mL）
  2. 加入500μL PEG/LiAc，轻轻混匀
  3. 30°C水浴30分钟（每10分钟轻轻混匀一次）
  4. 加入20μL DMSO，轻轻混匀
  5. 42°C热激20分钟
  6. 冰浴2分钟
  7. 室温，1000×g离心5分钟，弃上清
  8. 用1mL YPD液体培养基重悬
  9. 30°C，100rpm振荡培养1小时
  10. 取少量稀释液涂板检测转化效率
  11. 其余培养物涂布于SD/-Trp/-Leu/-His（含3-AT）平板
  12. 30°C培养3-5天，观察菌落生长
- **注意事项**：
  - 文库转化效率应达到10⁵-10⁶转化子/μg DNA
  - 3-AT浓度需要优化（通常25-50mM）
  - 确保诱饵和猎物载体有不同的营养标记（Trp和Leu）

### 步骤4：阳性克隆筛选与纯化

- **目的**：从筛选平板上分离和纯化阳性互作克隆
- **操作**：
  1. 培养3-5天后，观察SD/-Trp/-Leu/-His平板上的菌落
  2. 挑选生长良好的单克隆
  3. 在同样的选择平板上划线纯化2-3次
  4. 同时在SD/-Trp/-Leu平板上划线确认转化子存在
  5. 挑选纯化后的克隆进行进一步验证
  6. 将阳性克隆接种于SD/-Trp/-Leu液体培养基中培养
  7. 保存甘油菌（-80°C，25%甘油）
- **注意事项**：
  - 每次划线都应包含阳性对照（已知互作对）和阴性对照
  - 应挑选多个克隆进行分析
  - 纯化过程要彻底，避免混杂

### 步骤5：β-半乳糖苷酶活性验证

- **目的**：定量验证诱饵和猎物之间的蛋白互作
- **操作**：
  1. **滤纸 assays**：
     - 在无菌滤纸上培养酵母菌落
     - 将滤纸置于液氮中冷冻10秒
     - 取出放在含有X-Gal的Z-buffer饱和的滤纸上
     - 观察蓝色出现时间和强度
  2. **液体培养 assays****：
     - 培养阳性克隆至对数期
     - 测定OD600
     - 收集细胞，制备裂解物
     - 加入ONPG底物，37°C孵育
     - 测定420nm吸光度
     - 计算Miller单位
- **注意事项**：
  - β-半乳糖苷酶活性定量有助于区分强互作和弱互作
  - 应设置阳性对照和空白对照

### 步骤6：回转验证

- **目的**：排除假阳性，确认互作的真实性
- **操作**：
  1. 从阳性克隆中提取猎物质粒
  2. 转化入大肠杆菌，扩增
  3. 将回收的猎物质粒单独与诱饵质粒共转化入新鲜酵母
  4. 在选择培养基上筛选
  5. 再次进行β-半乳糖苷酶检测
  6. 测序鉴定猎物插入片段的序列
  7. 进行生物信息学分析，确认是水稻蛋白
- **注意事项**：
  - 回转实验是验证互作真实性的金标准
  - 测序结果需要与水稻基因组数据库比对
  - 最好进行一对一互作验证

## 5. 应用场景

- **蛋白互作筛选**：从水稻cDNA文库中筛选与目标蛋白互作的蛋白
- **蛋白互作网络构建**：系统性地构建水稻蛋白互作网络
- **信号转导通路研究**：鉴定水稻信号转导通路中的蛋白复合物成员
- **蛋白结构域分析**：确定蛋白互作所需的结构域
- **激酶-底物关系验证**：验证水稻蛋白激酶与底物的相互作用
- **转录因子辅因子鉴定**：鉴定与转录因子互作的辅因子蛋白

## 6. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **酵母三杂交** | 增加第三个蛋白或RNA | 研究三元复合物或RNA-蛋白互作 |
| **反向双杂交** | 根据互作被破坏筛选 | 筛选破坏互作的突变或多肽 |
| **膜蛋白双杂交** | 使用Cre-loxP系统 | 研究膜蛋白互作 |
| **细菌双杂交** | 在细菌中筛选 | 大规模筛选 |
| **核酵母双杂交** | 经典系统 | 一般蛋白互作研究 |

## 7. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 无菌落生长 | 转化失败或选择条件不对 | 检查试剂和操作，提高转化效率 |
| 假阳性多 | 自激活或报告基因泄漏 | 提高3-AT浓度，使用更严格的报告基因 |
| 转化效率低 | 感受态细胞质量差 | 现做现用，优化制备条件 |
| 猎物自激活 | 猎物融合蛋白有激活能力 | 截短猎物蛋白或更换猎物载体 |
| 互作弱或不稳定 | 蛋白表达水平低或互作条件不适 | 检查表达，优化培养条件 |

## 8. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 可从文库中直接筛选蛋白互作 | 存在假阳性和假阴性 |
| 可筛选未知互作蛋白 | 某些蛋白在酵母中不能正确折叠 |
| 可定量分析互作强度 | 膜蛋白和分泌蛋白难以研究 |
| 可研究蛋白复合物的多个成员 | 需要构建和验证诱饵载体 |
| 实验周期相对较短（2-3周） | 受限于转录激活域融合系统 |
