# 一、GST pull-down实验

## 1. 定义与核心概念

**GST pull-down实验**（GST Pull-down）是一种用于研究蛋白质-蛋白质相互作用的体外实验方法。该技术利用谷胱甘肽S-转移酶（Glutathione S-transferase, GST）标签融合蛋白作为诱饵，通过 glutathione-agarose 亲和纯化介质将其固定，然后捕获与其相互作用的猎物蛋白，从而检测和纯化蛋白质间的相互作用。

## 2. 分子机制/原理

GST pull-down技术的基本原理是：
1. **融合蛋白表达**：将诱饵蛋白与GST标签融合，在原核或真核系统中表达
2. **GST-融合蛋白固定**：利用glutathione-agarose介质与GST的高亲和力，将融合蛋白固定
3. **猎物蛋白孵育**：将含有潜在猎物蛋白的溶液通过固定化的GST-融合蛋白柱
4. **洗涤**：用缓冲液洗涤，去除非特异性结合的蛋白
5. **洗脱**：使用还原型谷胱甘肽洗脱复合物
6. **检测分析**：通过SDS-PAGE和Western blot或质谱检测猎物蛋白

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| GST-诱饵融合表达载体 | 表达GST-诱饵蛋白 |
| E. coli表达菌株（BL21/ Rosetta） | 原核表达系统 |
| Glutathione-agarose/Sepharose | 亲和纯化介质 |
| IPTG | 诱导蛋白表达 |
| 蛋白纯化缓冲液 | 平衡和洗涤 |
| 还原型谷胱甘肽（GSH） | 洗脱蛋白 |
| 猎物蛋白溶液 | 待检测的互作蛋白 |
| GST-标签蛋白（单独） | 阴性对照 |
| 蛋白酶抑制剂cocktail | 防止蛋白降解 |
| SDS-PAGE试剂 | 蛋白电泳分析 |
| Western blot试剂 | 蛋白检测 |

## 4. 实验步骤

### 步骤1：GST-融合诱饵蛋白的表达与制备

- **目的**：在大肠杆菌中大量表达GST-诱饵融合蛋白
- **操作**：
  1. 将GST-诱饵融合表达载体转化入E. coli表达菌株（BL21或Rosetta）
  2. 挑取单个克隆，接种于5mL含相应抗生素的LB培养基
  3. 37°C，200rpm振荡培养过夜
  4. 取1-2mL过夜培养物转接于100mL新鲜LB培养基
  5. 37°C，200rpm振荡培养至OD600约0.4-0.6
  6. 加入IPTG至终浓度0.1-0.5mM（根据蛋白特性优化）
  7. 16°C或25°C诱导表达6-16小时
  8. 4°C，5000rpm离心10分钟收集菌体
  9. 弃上清，菌体用预冷的PBS重悬洗涤
  10. 再次离心，收集菌体，-80°C保存或直接进行下一步
- **注意事项**：
  - IPTG浓度和诱导温度、时间需要优化
  - 某些蛋白需要低温诱导才能正确折叠
  - 菌体可在-80°C保存数周

### 步骤2：GST-融合蛋白的纯化与固定

- **目的**：裂解菌体，释放GST-融合蛋白，并将其固定在glutathione-agarose介质上
- **操作**：
  1. 将菌体用10mL GST lysis buffer重悬
  2. 加入溶菌酶（0.5mg/mL）和蛋白酶抑制剂cocktail
  3. 冰上孵育30分钟
  4. 超声波破碎（功率150W，工作3秒，间隔5秒，重复10次）
  5. 4°C，15000rpm离心30分钟
  6. 收集上清（含有可溶性GST-融合蛋白）
  7. 取100μL glutathione-agarose介质
  8. 用GST lysis buffer洗涤3次
  9. 将上清与洗涤后的介质混合
  10. 4°C慢摇孵育2-4小时或过夜
  11. 离心或磁性分离，收集介质
  12. 用GST lysis buffer洗涤5次
  13. 重悬于适量GST lysis buffer中
- **注意事项**：
  - 如表达 inclusion bodies，可使用变性缓冲液溶解
  - 介质与蛋白孵育时间充分可提高结合效率
  - 洗涤要彻底去除杂质蛋白

### 步骤3：猎物蛋白溶液制备

- **目的**：制备含有潜在猎物蛋白的溶液，用于与固定化的GST-诱饵蛋白孵育
- **操作**：
  1. **如猎物蛋白为原核表达**：
     - 将猎物蛋白表达载体转化入E. coli
     - 诱导表达（可使用不同的标签如His、GST等）
     - 破碎菌体，收集上清
  2. **如猎物蛋白为水稻细胞提取物**：
     - 收取水稻组织或悬浮细胞
     - 用IP裂解缓冲液裂解
     - 4°C，12000rpm离心15分钟
     - 收集上清
  3. 测定蛋白浓度
  4. 用GST binding buffer调节至适当浓度
- **注意事项**：
  - 猎物蛋白应尽量新鲜制备
  - 可预先用glutathione-agarose进行预清除
  - 设置适当的阳性和阴性对照

### 步骤4：Pull-down反应

- **目的**：让固定化的GST-诱饵蛋白与猎物蛋白充分接触并结合
- **操作**：
  1. 将固定化的GST-诱饵蛋白介质分成两份：
     - 实验组：GST-诱饵 + 猎物蛋白
     - 对照组：单独GST介质（或GST-无关蛋白）+ 猎物蛋白
  2. 加入猎物蛋白溶液
  3. 4°C慢摇孵育2-4小时
  4. 离心或磁性分离，收集介质
  5. 用GST wash buffer洗涤5次
  6. 每次洗涤在4°C慢摇5分钟
- **注意事项**：
  - 猎物蛋白用量根据预实验确定
  - 洗涤条件要平衡，既要去除非特异性，又不能破坏真实互作
  - 必须同时设置阴性对照组

### 步骤5：洗脱与检测

- **目的**：将结合的蛋白复合物从介质上洗脱下来，并进行检测分析
- **操作**：
  1. **洗脱方法**：
     - **还原型谷胱甘肽洗脱**：加入50μL 10mM还原型GSH（现配现用）
     - 4°C孵育10分钟
     - 离心收集洗脱液
     - 重复洗脱2-3次
  2. **SDS-PAGE分析**：
     - 合并洗脱液
     - 加入SDS-PAGE上样缓冲液
     - 95°C加热5分钟
     - 进行SDS-PAGE电泳
  3. **Western blot检测**：
     - 电转移到PVDF膜
     - 用猎物蛋白的特异性抗体进行检测
     - 如猎物带有标签，用相应抗体检测
  4. **质谱鉴定（如需鉴定未知蛋白）**：
     - SDS-PAGE后银染或考马斯亮蓝染色
     - 切下感兴趣的条带
     - 质谱分析
- **注意事项**：
  - 还原型GSH必须新鲜配制
  - 洗脱液可能需要浓缩以提高检测灵敏度
  - 阳性对照组应该出现预期大小的条带

### 步骤6：结果分析

- **目的**：判断诱饵蛋白与猎物蛋白之间是否存在特异性相互作用
- **分析要点**：
  1. **阴性对照组**：单独GST介质不应捕获猎物蛋白
  2. **实验组vs阴性组**：实验组出现而阴性组不出现的条带为特异性结合
  3. **分子量验证**：条带大小应与预期猎物蛋白大小一致
  4. **Input对照**：确认猎物蛋白在原始溶液中存在
- **注意事项**：
  - 阴性对照非常重要，用于排除非特异性结合
  - 某些弱相互作用可能在洗涤过程中丢失
  - 需要排除GST标签本身的相互作用

## 5. 应用场景

- **验证水稻蛋白-蛋白直接相互作用**：验证两个水稻蛋白是否直接结合
- **筛选互作蛋白**：用水稻cDNA文库筛选与诱饵蛋白相互作用的蛋白
- **确定相互作用结构域**：使用诱饵蛋白的不同截短体确定互作区域
- **研究竞争性结合**：检测多个蛋白与同一诱饵的竞争性结合
- **体外互作研究**：在严格控制的体外条件下研究蛋白互作

## 6. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 猎物蛋白不被捕获 | 蛋白失活或没有互作 | 检查蛋白表达和活性，优化条件 |
| 背景高 | 非特异性结合强 | 加强洗涤，更换缓冲液 |
| GST-融合蛋白表达量低 | 表达条件不佳 | 优化IPTG浓度、温度、时间 |
| 蛋白形成包涵体 | 表达条件不适合 | 降低诱导温度，使用助溶剂 |
| 洗脱不下来 | 结合太紧密 | 使用酸性或变性条件洗脱 |

## 7. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 可以在体外条件下研究蛋白互作 | 需要纯化GST-融合诱饵蛋白 |
| 操作相对简单 | 可能存在GST标签的干扰 |
| 可用于筛选文库 | 不能反映细胞内蛋白互作的真实状态 |
| 可研究直接互作（无第三者介导） | 对于弱相互作用检测灵敏度有限 |
| 结果明确，易于解释 | 需要高质量的抗体进行检测 |
