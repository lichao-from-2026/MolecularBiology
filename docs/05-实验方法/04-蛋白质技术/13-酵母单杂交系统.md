# 一、酵母单杂交系统

## 1. 定义与核心概念

**酵母单杂交系统**（Yeast One-Hybrid System）是一种用于鉴定和研究DNA序列与蛋白质相互作用的技术。该技术基于转录激活原理，将目标DNA序列（诱饵）插入到报告基因的上游激活序列位置，如果存在能够与该DNA序列结合的蛋白质（猎物），它会激活报告基因的表达，从而通过报告基因的筛选或检测来识别DNA-蛋白质的相互作用。

## 2. 分子机制/原理

酵母单杂交系统的基本原理是：
1. **构建诱饵载体**：将目标DNA序列（含有待测转录因子结合位点）克隆到报告基因的上游
2. **构建猎物表达文库**：将水稻cDNA文库片段与转录激活域（如GAL4 AD、VP16 AD）融合表达
3. **转化酵母菌株**：将诱饵载体和猎物文库共同转化入酵母报告菌株
4. **筛选与鉴定**：根据报告基因（营养缺陷、色素形成等）筛选阳性克隆
5. **验证**：通过β-半乳糖苷酶活性检测或PCR验证阳性克隆
6. **回收质粒**：从阳性克隆中回收猎物质粒并进行测序鉴定

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| 酵母菌株（Y187、EGY48等） | 报告菌株 |
| 诱饵载体（pAbAi等） | 含有目标DNA序列的报告载体 |
| 猎物载体（pGADT7等） | 表达GAL4 AD-猎物蛋白融合 |
| 水稻cDNA文库 | 提供猎物蛋白来源 |
| 限制性内切酶 | 载体构建 |
| T4 DNA连接酶 | DNA连接 |
| 酵母转化试剂（PEG/LiAc） | 化学法转化酵母 |
| SD/-Leu/-Ura缺陷培养基 | 筛选转化子 |
| X-α-Gal | 检测α-半乳糖苷酶活性 |
| Aureobasidin A（AbA） | 抗性筛选 |
| β-半乳糖苷酶检测试剂盒 | 验证相互作用 |
| 酵母质粒提取试剂盒 | 回收猎物质粒 |
| 大肠杆菌感受态细胞（DH5α） | 扩增回收的质粒 |

## 4. 实验步骤

### 步骤1：诱饵载体构建

- **目的**：将含有目标转录因子结合位点的DNA序列克隆到报告基因上游
- **操作**：
  1. 合成或PCR扩增含有目标结合位点的DNA片段（通常串联3个拷贝）
  2. 使用限制性内切酶酶切诱饵载体（如pAbAi）
  3. 将DNA片段插入到报告基因（如LacZ或HIS3）上游的多克隆位点
  4. 连接反应，转化大肠杆菌
  5. 筛选阳性克隆，测序验证
  6. 将诱饵载体线性化（用于整合入酵母基因组）
- **注意事项**：
  - DNA片段应包含完整的转录因子结合元件
  - 串联重复可以增强信号，但过多可能导致假阳性
  - 载体线性化位点选择很重要

### 步骤2：诱饵菌株制备与自激活检测

- **目的**：将诱饵载体整合入酵母基因组，并检测是否存在自激活
- **操作**：
  1. 用限制性内切酶将诱饵载体线性化
  2. 纯化线性化DNA
  3. 制备酵母感受态细胞（Y187或EGY48）
  4. 将线性化诱饵载体转化入酵母
  5. 在SD/-Ura缺陷培养基上筛选转化子
  6. 挑取单个菌落，检测报告基因 basal 表达水平
  7. 如存在自激活，加入3-AT抑制HIS3 basal表达
- **注意事项**：
  - 不同酵母菌株的基因型不同，应根据实验需求选择
  - 自激活会导致假阳性，需要用3-AT或增加AbA浓度抑制
  - 3-AT浓度需要优化，过高会抑制真实相互作用

### 步骤3：猎物表达文库转化

- **目的**：将水稻cDNA文库（与激活域融合）导入含有诱饵的酵母菌株中
- **操作**：
  1. 制备诱饵菌株的感受态细胞
  2. 将猎物文库质粒（pGADT7-cDNA文库）转化入诱饵菌株
  3. 在SD/-Leu/-Ura双缺陷培养基上筛选共转化子
  4. 培养2-3天后观察菌落生长
  5. 将所有转化子洗脱下来，接种于液体培养基中
- **注意事项**：
  - 文库转化效率影响筛选覆盖率
  - 需要确保诱饵菌株状态良好
  - 操作过程需无菌，避免污染

### 步骤4：阳性克隆筛选

- **目的**：通过报告基因筛选能够与诱饵DNA结合的猎物克隆
- **操作**：
  1. 将转化子涂布于含X-α-Gal和AbA的SD/-Leu/-Ura缺陷培养基
  2. 培养2-4天，观察蓝色菌落
  3. 挑选蓝色菌落（表示报告基因被激活）进行二次筛选
  4. 再次在选择培养基上划线纯化
  5. 去除假阳性（无诱饵也能激活报告基因的克隆）
- **注意事项**：
  - 蓝色菌落不一定是真正的阳性，需要进一步验证
  - 应设置阳性对照和阴性对照
  - 筛选过程需要耐心，避免遗漏

### 步骤5：阳性克隆验证

- **目的**：验证筛选到的克隆是否真正与诱饵DNA特异性结合
- **操作**：
  1. **β-半乳糖苷酶滤纸 assays**：验证LacZ报告基因激活
  2. **回转验证**：将回收的猎物质粒单独转化入诱饵菌株，确认功能
  3. **排除假阳性**：将猎物质粒转化入不含诱饵的酵母，确认无basal激活
  4. **PCR鉴定**：使用猎物载体上的通用引物扩增插入片段
  5. **测序鉴定**：对插入片段进行测序，确定编码的蛋白
- **注意事项**：
  - 必须排除假阳性克隆
  - 测序结果需要与水稻基因组比对
  - 最好进行co-IP或EMSA进一步验证

### 步骤6：猎物质粒回收与鉴定

- **目的**：从酵母阳性克隆中回收猎物质粒并进行序列分析
- **操作**：
  1. 使用酵母质粒提取试剂盒提取酵母中的质粒DNA
  2. 将提取的DNA转化入大肠杆菌感受态细胞（如DH5α）
  3. 在含Amp的LB平板上筛选
  4. 挑取单个菌落扩大培养
  5. 提取质粒DNA进行测序
  6. 分析测序结果，确定猎物质粒编码的蛋白
- **注意事项**：
  - 酵母质粒提取效率较低，需要使用专门的试剂盒
  - 从酵母回收的质粒通常混合有猎物质粒和线性化的诱饵载体
  - 需要通过酶切或PCR区分

## 5. 应用场景

- **转录因子筛选**：筛选能够与水稻特定基因启动子结合的转录因子
- **顺式作用元件结合蛋白鉴定**：鉴定与水稻基因上游调控元件结合的蛋白
- **DNA-蛋白相互作用结构域分析**：确定蛋白中负责DNA结合的结构域
- **转录因子家族成员功能分析**：筛选水稻转录因子家族成员的DNA结合特异性
- **非编码DNA结合蛋白鉴定**：鉴定与lncRNA、 enhancer等非编码序列结合的蛋白

## 6. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 无蓝色菌落 | 文库转化效率低或诱饵自激活 | 提高转化效率，去除自激活 |
| 蓝色很淡 | 结合弱或报告基因表达低 | 增加拷贝数或使用其他报告基因 |
| 假阳性多 | 诱饵序列特异性低 | 优化诱饵序列，增加筛选严格度 |
| 文库覆盖率不足 | 转化子数不够 | 增加转化量，使用高效转化方法 |
| 猎物质粒回收失败 | 酵母细胞破壁不充分 | 使用优化的酵母质粒提取方法 |

## 7. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 可从文库中直接筛选DNA结合蛋白 | 需要构建和验证诱饵载体 |
| 可筛选未知蛋白，发现新的转录因子 | 存在假阳性和假阴性 |
| 在酵母体内进行，蛋白经过适当修饰 | 某些水稻蛋白可能在酵母中不正确折叠 |
| 可同时筛选整个cDNA文库 | 膜蛋白等难以在酵母中表达 |
| 实验周期相对较短（2-3周） | 限于转录激活域融合表达 |
