# 一、免疫共沉淀技术（Co-IP）

## 1. 定义与核心概念

**免疫共沉淀技术**（Co-Immunoprecipitation, Co-IP）是一种用于研究蛋白质-蛋白质相互作用的实验方法。该技术利用抗体特异性识别并结合目标蛋白（诱饵），当目标蛋白与其相互作用的蛋白（猎物）在细胞内形成复合物时，抗体能够将整个蛋白复合物一起沉淀下来，从而分离和检测与目标蛋白相互作用的其他蛋白。

## 2. 分子机制/原理

Co-IP技术的基本原理是：
1. **细胞裂解**：在温和条件下裂解细胞，保持蛋白质复合物的天然状态
2. **免疫沉淀**：加入针对目标蛋白的特异性抗体，与蛋白复合物结合
3. **蛋白A/G亲和**：加入Protein A或Protein G琼脂糖/磁珠，结合抗体Fc段
4. **洗涤**：洗涤沉淀复合物，去除非特异性结合的蛋白
5. **变性处理**：加入SDS-PAGE上样缓冲液，煮沸变性
6. **检测分析**：通过SDS-PAGE和Western blot检测沉淀中的蛋白

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| 特异性一抗 | 识别目标蛋白 |
| Protein A/G 琼脂糖/磁珠 | 结合抗体-蛋白复合物 |
| IP裂解缓冲液（NP-40或Triton X-100） | 温和裂解细胞 |
| 蛋白酶抑制剂cocktail | 抑制蛋白降解 |
| 磷酸酶抑制剂 | 抑制蛋白去磷酸化 |
| Wash Buffer（低盐/高盐） | 洗涤沉淀 |
| SDS-PAGE上样缓冲液 | 变性蛋白 |
| 对应一抗的HRP标记二抗 | Western检测 |
| 预染蛋白分子量标准 | 判断蛋白大小 |
| ECL发光液 | 显影检测 |

## 4. 实验步骤

### 步骤1：水稻蛋白样品制备

- **目的**：在温和条件下裂解水稻组织或细胞，获得含有天然蛋白复合物的蛋白提取物
- **操作**：
  1. 收取水稻幼苗或悬浮细胞（约1-2g）
  2. 用预冷的PBS洗涤2次
  3. 加入500μL-1mL IP裂解缓冲液
  4. 在冰上用匀浆器彻底匀浆
  5. 冰上孵育30分钟，期间 vortex 3-4次
  6. 4°C，12000rpm离心15分钟
  7. 收集上清（总蛋白溶液）
  8. 取少量样品作为Input对照
  9. 测定蛋白浓度（BCA或Bradford法）
- **注意事项**：
  - 裂解缓冲液需预冷，所有操作在4°C进行
  - 裂解条件要温和，避免破坏蛋白互作
  - 蛋白酶和磷酸酶抑制剂必须新鲜配制
  - 如检测磷酸化修饰，需加入磷酸酶抑制剂

### 步骤2：预清除（可选但推荐）

- **目的**：去除蛋白提取物中可能与Protein A/G或抗体非特异性结合的蛋白，减少假阳性
- **操作**：
  1. 取100μL Protein A/G磁珠/琼脂糖
  2. 用IP裂解缓冲液洗涤3次
  3. 将蛋白提取物与预处理过的磁珠混合
  4. 4°C孵育30分钟-1小时（慢摇）
  5. 离心或磁性分离，收集上清
  6. 进行后续免疫沉淀步骤
- **注意事项**：
  - 预清除可以显著降低背景
  - 孵育时间不宜过长，避免丢失目标蛋白
  - 对照组和实验组必须同样处理

### 步骤3：免疫沉淀反应

- **目的**：利用特异性抗体捕获目标蛋白及其相互作用的蛋白
- **操作**：
  1. 将蛋白提取物分成两份：
     - 实验组：加入目标蛋白抗体
     - 对照组：加入相同量的IgG或无关抗体
  2. 4°C慢摇孵育过夜
  3. 取50μL Protein A/G磁珠/琼脂糖
  4. 用IP裂解缓冲液洗涤3次
  5. 将磁珠加入抗体-蛋白复合物中
  6. 4°C慢摇孵育2-4小时
  7. 磁性分离或离心收集磁珠
  8. 保留上清作为未结合部分
- **注意事项**：
  - 抗体用量需要优化，一般1-5μg
  - 抗体必须预先验证特异性
  - IgG对照必须与实验组抗体来源相同物种
  - 孵育时间充分但不过长

### 步骤4：洗涤

- **目的**：去除非特异性结合的蛋白，保留特异性互作的蛋白
- **操作**：
  1. 用Wash Buffer 1（含0.1% NP-40）洗涤5次
  2. 每次洗涤时在4°C慢摇5分钟
  3. 最后一次洗涤后，磁性分离或离心收集磁珠
  4. 尽量去除所有上清
  5. 加入30-50μL 1×SDS-PAGE上样缓冲液
- **注意事项**：
  - 洗涤力度需要平衡，太强会破坏真实互作，太弱会增加背景
  - 可设置不同盐浓度的洗涤缓冲液进行梯度洗涤
  - 洗涤过程要彻底，避免 buffer 残留

### 步骤5：样品处理与检测

- **目的**：将免疫沉淀的蛋白变性处理后进行SDS-PAGE和Western blot分析
- **操作**：
  1. 将样品在95°C加热5分钟使蛋白变性
  2. 磁性分离或离心，收集上清
  3. 进行SDS-PAGE电泳（10-12%凝胶）
  4. 电转移到PVDF膜或NC膜
  5. 用5%脱脂牛奶封闭1小时
  6. 孵育一抗（检测目标蛋白或潜在互作蛋白）4°C过夜
  7. TBST洗涤3次，每次10分钟
  8. 孵育HRP标记的二抗，室温1小时
  9. TBST洗涤3次，每次10分钟
  10. ECL发光液显影
- **注意事项**：
  - Input对照必须同样处理和检测
  - 二抗稀释倍数需要优化
  - 显影时间根据信号强度调整

### 步骤6：结果分析

- **目的**：判断目标蛋白与潜在互作蛋白之间是否存在特异性相互作用
- **分析要点**：
  1. **Input对照**：确认目标蛋白和潜在互作蛋白在总蛋白中有表达
  2. **IgG对照**：排除抗体和磁珠的非特异性结合
  3. **实验组vs IgG组**：在实验组中出现而IgG组中不出现的条带才是特异性互作
  4. **分子量判断**：根据条带大小确认是否为预期蛋白
- **注意事项**：
  - 只有在实验组中特异性出现，且分子量正确的条带才有意义
  - 阴性结果不能完全排除互作可能性，可能是互作条件不对

## 5. 应用场景

- **验证水稻蛋白-蛋白相互作用**：验证在体内条件下两个水稻蛋白是否直接互作
- **发现新的互作蛋白**：通过质谱分析发现与目标蛋白互作的新蛋白
- **研究蛋白复合物组成**：分析水稻蛋白复合物（如转录因子复合物、信号复合物）的成员
- **研究蛋白修饰对互作的影响**：检测磷酸化、泛素化等修饰对蛋白互作的影响
- **信号转导通路研究**：研究水稻激素信号、病害响应等通路中的蛋白互作关系

## 6. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 无目标条带 | 抗体效价低或蛋白表达量低 | 更换抗体，增加蛋白上样量 |
| IgG对照也有条带 | 抗体非特异性或磁珠非特异性 | 更换抗体，加强洗涤 |
| 背景高 | 洗涤不充分或抗体过量 | 增加洗涤次数，降低抗体用量 |
| 蛋白被降解 | 蛋白酶抑制不足 | 新鲜配制蛋白酶抑制剂 |
| 磷酸化修饰丢失 | 磷酸酶抑制不足 | 加入磷酸酶抑制剂 |
| 假阴性 | 互作被裂解条件破坏 | 优化裂解条件，使用更温和的去垢剂 |

## 7. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 在体内条件下研究蛋白互作，更接近真实状态 | 需要高质量的特异性抗体 |
| 可研究完整的蛋白复合物 | 抗体可能无法有效免疫沉淀 |
| 可发现未知的互作蛋白（结合质谱） | 裂解条件可能破坏弱互作或瞬时互作 |
| 可研究修饰依赖的互作 | 存在假阳性和假阴性 |
| 操作相对简单 | 难以区分直接互作和间接互作（通过第三蛋白） |
