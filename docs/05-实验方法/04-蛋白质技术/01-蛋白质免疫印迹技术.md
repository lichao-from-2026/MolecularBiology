# 01-蛋白质免疫印迹技术（Western Blot）

## 1. 定义与核心概念

**蛋白质免疫印迹技术**（Western Blot）是一种用于检测和定量特定蛋白质的分子生物学技术。它通过凝胶电泳分离蛋白质混合物，然后将蛋白质转移到膜上，最后使用特异性抗体检测目标蛋白质。Western Blot是研究蛋白质表达、修饰和相互作用的重要工具。

## 2. 分子机制/原理

Western Blot技术的基本原理是：
1. **蛋白质分离**：通过SDS-聚丙烯酰胺凝胶电泳（SDS-PAGE）根据分子量分离蛋白质
2. **蛋白质转移**：将分离的蛋白质从凝胶转移到固相支持膜（如PVDF或NC膜）
3. **封闭**：用非特异性蛋白质（如脱脂牛奶）封闭膜上的非特异性结合位点
4. **抗体孵育**：使用一抗特异性识别目标蛋白质，二抗识别一抗
5. **检测**：通过酶促反应或荧光检测显示目标蛋白质的位置和量

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| 蛋白质样品 | 待检测的蛋白质 |
| SDS-PAGE凝胶 | 分离蛋白质 |
| 转移缓冲液 | 蛋白质转移 |
| PVDF膜或NC膜 | 蛋白质载体 |
| 封闭液（5%脱脂牛奶） | 封闭非特异性结合位点 |
| 一抗 | 特异性识别目标蛋白质 |
| 二抗（HRP或荧光标记） | 识别一抗 |
| ECL发光液 | 化学发光检测 |
| 电泳仪 | 蛋白质分离 |
| 转膜装置 | 蛋白质转移 |
| 摇床 | 孵育抗体 |
| 凝胶成像系统 | 检测信号 |
| 蛋白质分子量标准 | 确定蛋白质大小 |

## 4. 实验步骤

### 步骤1：蛋白质样品制备

- **目的**：提取和处理蛋白质样品，为电泳做准备
- **操作**：
  1. **细胞裂解**：
     - 收集细胞或组织样品
     - 加入RIPA裂解缓冲液（含蛋白酶抑制剂）
     - 冰上孵育30分钟，期间涡旋数次
     - 12000rpm离心15分钟，收集上清
  2. **蛋白质定量**：
     - 使用BCA或Bradford法测定蛋白质浓度
     - 调整所有样品至相同浓度
  3. **样品处理**：
     - 加入5×SDS上样缓冲液
     - 95°C加热5分钟，使蛋白质变性
     - 短暂离心，冷却至室温
- **注意事项**：
  - 裂解缓冲液应新鲜配制，含蛋白酶抑制剂
  - 蛋白质定量应准确，确保上样量一致
  - 变性处理应充分，确保蛋白质完全解聚

### 步骤2：SDS-PAGE电泳

- **目的**：根据分子量分离蛋白质样品
- **操作**：
  1. **凝胶制备**：
     - 配制分离胶（根据目标蛋白质分子量选择浓度）
     - 灌注分离胶，加入水封
     - 凝固后，配制浓缩胶
     - 插入梳子，待凝固
  2. **上样**：
     - 组装电泳装置，加入电泳缓冲液
     - 上样蛋白质样品和分子量标准
  3. **电泳**：
     - 恒压80V，待样品进入分离胶后，调至120V
     - 电泳至溴酚蓝到达凝胶底部
- **注意事项**：
  - 凝胶浓度应根据目标蛋白质分子量选择
  - 上样量应适当，避免过载
  - 电泳过程中应注意冷却

### 步骤3：蛋白质转移

- **目的**：将分离的蛋白质从凝胶转移到膜上
- **操作**：
  1. **膜预处理**：
     - PVDF膜用甲醇浸泡1-2分钟激活
     - NC膜用转移缓冲液润湿
  2. **转膜装置组装**：
     - 按照海绵→滤纸→凝胶→膜→滤纸→海绵的顺序组装
     - 排除气泡
  3. **转移**：
     - 恒压100V转移1-2小时（根据蛋白质大小调整）
     - 或恒流300mA转移1小时
- **注意事项**：
  - 转移缓冲液应新鲜配制
  - 组装时应避免气泡
  - 转移时间应根据蛋白质大小调整

### 步骤4：封闭

- **目的**：封闭膜上的非特异性结合位点
- **操作**：
  1. 转移完成后，取出膜
  2. 用TBST洗膜1次，5分钟
  3. 加入5%脱脂牛奶（或BSA）封闭液
  4. 室温摇床孵育1-2小时，或4°C过夜
- **注意事项**：
  - 封闭液应覆盖整个膜
  - 封闭时间应足够，避免背景过高
  - 某些抗体可能需要使用BSA作为封闭剂

### 步骤5：抗体孵育

- **目的**：使用特异性抗体检测目标蛋白质
- **操作**：
  1. **一抗孵育**：
     - 用封闭液稀释一抗（根据说明书推荐浓度）
     - 加入膜，室温摇床孵育2小时，或4°C过夜
     - TBST洗膜3次，每次10分钟
  2. **二抗孵育**：
     - 用封闭液稀释二抗（1:5000-1:10000）
     - 加入膜，室温摇床孵育1小时
     - TBST洗膜3次，每次10分钟
- **注意事项**：
  - 抗体浓度应优化，避免背景过高或信号过弱
  - 孵育时间应足够，确保抗体充分结合
  - 洗膜应彻底，去除未结合的抗体

### 步骤6：检测与分析

- **目的**：检测目标蛋白质的存在和表达水平
- **操作**：
  1. **化学发光检测**：
     - 混合ECL发光液A和B
     - 均匀覆盖膜表面
     - 孵育1-2分钟
     - 使用凝胶成像系统曝光
  2. **荧光检测**：
     - 使用荧光标记的二抗
     - 使用荧光成像系统检测
  3. **结果分析**：
     - 观察条带位置和强度
     - 与分子量标准比较，确定蛋白质大小
     - 半定量分析蛋白质表达水平
- **注意事项**：
  - ECL发光液应现用现配
  - 曝光时间应根据信号强度调整
  - 应设置阳性对照和阴性对照

## 5. 应用场景

- **蛋白质表达分析**：检测不同条件下蛋白质的表达水平
- **蛋白质修饰分析**：检测蛋白质的磷酸化、泛素化等修饰
- **蛋白质相互作用研究**：检测蛋白质复合物的存在
- **抗体特异性验证**：验证抗体的特异性
- **疾病标志物检测**：检测疾病相关蛋白质的表达
- **药物作用机制研究**：研究药物对蛋白质表达的影响

## 6. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **常规Western Blot** | 基本技术，使用化学发光检测 | 一般蛋白质检测 |
| **半干转印** | 转移速度快，缓冲液使用少 | 快速检测 |
| **湿转印** | 转移效率高，适合大分子蛋白质 | 大分子蛋白质检测 |
| **荧光Western Blot** | 使用荧光标记二抗，多通道检测 | 同时检测多个蛋白质 |
| **定量Western Blot** | 使用内参蛋白，半定量分析 | 蛋白质表达定量 |
| **Far-Western Blot** | 使用标记的蛋白质作为探针 | 蛋白质-蛋白质相互作用研究 |

## 7. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 无条带 | 蛋白质未转移或抗体不识别 | 检查转膜效率，验证抗体特异性 |
| 背景过高 | 封闭不充分或抗体浓度过高 | 延长封闭时间，降低抗体浓度 |
| 条带弥散 | 蛋白质降解或电泳条件不当 | 加入蛋白酶抑制剂，优化电泳条件 |
| 非特异性条带 | 抗体交叉反应 | 使用特异性更高的抗体，优化抗体浓度 |
| 信号弱 | 蛋白质表达量低或抗体效价低 | 增加上样量，使用效价更高的抗体 |

## 8. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 特异性强，可检测特定蛋白质 | 操作步骤多，耗时较长 |
| 可同时分析多个样品 | 需要高质量的抗体 |
| 可检测蛋白质修饰 | 半定量分析，准确性有限 |
| 可用于蛋白质相互作用研究 | 对样品量有一定要求 |
| 技术成熟，应用广泛 | 可能存在假阳性或假阴性 |
