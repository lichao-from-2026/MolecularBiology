# 02-真核蛋白表达与纯化

## 1. 定义与核心概念

**真核蛋白表达与纯化**是指在真核细胞（如酵母、昆虫细胞、哺乳动物细胞）中表达重组蛋白并通过各种方法纯化的技术。真核表达系统能够提供正确的蛋白质折叠、翻译后修饰（如糖基化、磷酸化等），因此适用于表达需要复杂修饰的真核蛋白。

## 2. 分子机制/原理

真核蛋白表达与纯化的基本原理是：
1. **表达载体构建**：将目标基因克隆到真核表达载体中，在启动子的控制下表达
2. **转染或转化**：将表达载体导入真核细胞
3. **蛋白表达**：在真核细胞中表达目标蛋白，进行正确的折叠和修饰
4. **细胞破碎或分泌**：通过破碎细胞或收集培养基获取蛋白
5. **蛋白纯化**：通过亲和层析、离子交换层析等方法纯化目标蛋白
6. **蛋白鉴定**：通过SDS-PAGE、Western blot等方法鉴定蛋白

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| 真核表达载体（如pcDNA系列） | 用于表达重组蛋白 |
| 真核细胞系（如HEK293、CHO） | 表达宿主 |
| 细胞培养基 | 培养细胞 |
| 转染试剂 | 将载体导入细胞 |
| 抗生素 | 筛选阳性细胞 |
| 超声破碎仪 | 破碎细胞 |
| 离心机 | 分离细胞和蛋白 |
| 亲和层析柱（如Ni-NTA） | 纯化带标签的蛋白 |
| 凝胶过滤柱 | 进一步纯化蛋白 |
| SDS-PAGE凝胶 | 分析蛋白表达和纯度 |
| 蛋白质分子量标准 | 确定蛋白大小 |
| Bradford试剂 | 测定蛋白浓度 |
| 移液器 | 移取液体 |
| 离心管 | 反应容器 |

## 4. 实验步骤

### 步骤1：表达载体构建

- **目的**：构建含有目标基因的真核表达载体
- **操作**：
  1. **基因克隆**：
     - PCR扩增目标基因
     - 限制性酶切目标基因和表达载体
     - 连接目标基因和表达载体
     - 转化大肠杆菌，筛选阳性克隆
     - 测序验证
  2. **载体选择**：
     - 根据目标蛋白的特性选择合适的真核表达载体
     - 选择合适的标签（如His、Flag、Myc等）
     - 选择合适的启动子（如CMV、EF1α等）
- **注意事项**：
  - 应选择适合真核表达的密码子优化的基因序列
  - 应考虑目标蛋白的溶解性和毒性
  - 应选择合适的启动子和标签

### 步骤2：细胞培养与转染

- **目的**：培养真核细胞并将表达载体导入细胞
- **操作**：
  1. **细胞培养**：
     - 培养真核细胞至70-80%汇合度
     - 更换新鲜培养基
  2. **转染**：
     - 选择合适的转染方法（脂质体转染、电转等）
     - 按照转染试剂说明书操作
     - 将表达载体转染到细胞中
  3. **培养**：
     - 转染后培养48-72小时
     - 更换新鲜培养基
- **注意事项**：
  - 转染条件应优化
  - 应设置阴性对照（仅转染空载体）
  - 转染效率应通过荧光标记等方法评估

### 步骤3：蛋白表达分析

- **目的**：分析目标蛋白的表达情况
- **操作**：
  1. **细胞收集**：
     - 转染后72小时，收集细胞或培养基
  2. **蛋白提取**：
     - 如果是胞内表达，破碎细胞提取蛋白
     - 如果是分泌表达，收集培养基
  3. **SDS-PAGE分析**：
     - 取细胞裂解液或培养基进行SDS-PAGE
     - 考马斯亮蓝染色或Western blot分析
- **注意事项**：
  - 应设置未转染对照
  - 应分析蛋白表达水平和溶解性
  - 应检测蛋白的翻译后修饰情况

### 步骤4：大量表达与蛋白收集

- **目的**：大量表达目标蛋白并收集
- **操作**：
  1. **扩大培养**：
     - 在较大的培养瓶或生物反应器中培养细胞
     - 转染表达载体
  2. **蛋白表达**：
     - 培养48-72小时，让蛋白充分表达
  3. **蛋白收集**：
     - 如果是胞内表达，收集细胞并破碎
     - 如果是分泌表达，收集培养基
- **注意事项**：
  - 应根据小量测试的结果优化表达条件
  - 细胞应尽快处理，避免蛋白降解
  - 可加入蛋白酶抑制剂防止蛋白降解

### 步骤5：蛋白纯化

- **目的**：纯化目标蛋白
- **操作**：
  1. **亲和层析**：
     - 平衡亲和层析柱（如Ni-NTA柱）
     - 上样细胞裂解上清或培养基
     - 用洗涤缓冲液洗涤杂蛋白
     - 用洗脱缓冲液洗脱目标蛋白
  2. **离子交换层析**：
     - 平衡离子交换柱
     - 上样亲和纯化的蛋白
     - 洗脱并收集蛋白峰
  3. **凝胶过滤层析**：
     - 平衡凝胶过滤柱
     - 上样离子交换纯化的蛋白
     - 洗脱并收集蛋白峰
  4. **蛋白浓缩**：
     - 使用超滤管浓缩蛋白
     - 更换缓冲液至合适的储存缓冲液
- **注意事项**：
  - 应根据蛋白标签选择合适的亲和层析柱
  - 应优化洗涤和洗脱条件
  - 应检测每一步的蛋白纯度

### 步骤6：蛋白鉴定与定量

- **目的**：鉴定和定量纯化的蛋白
- **操作**：
  1. **SDS-PAGE分析**：
     - 取纯化的蛋白进行SDS-PAGE
     - 考马斯亮蓝染色，观察蛋白纯度
  2. **Western blot验证**：
     - 使用特异性抗体检测目标蛋白
  3. **蛋白定量**：
     - 使用Bradford法或BCA法测定蛋白浓度
  4. **蛋白储存**：
     - 将蛋白分装，保存于-80°C
- **注意事项**：
  - 应设置蛋白分子量标准
  - 应使用合适的方法定量蛋白
  - 蛋白应在适当的条件下储存，避免降解

## 5. 应用场景

- **蛋白结构研究**：表达和纯化需要正确折叠和修饰的蛋白
- **功能研究**：研究蛋白的生物学功能，尤其是需要翻译后修饰的蛋白
- **抗体制备**：表达抗原蛋白用于抗体生产
- **药物开发**：筛选药物靶点或开发蛋白药物
- **疫苗开发**：表达病毒或细菌抗原蛋白
- **工业应用**：生产需要复杂修饰的蛋白

## 6. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **瞬时表达** | 操作简单，表达速度快 | 小量蛋白制备，快速筛选 |
| **稳定表达** | 表达稳定，可长期生产 | 大量蛋白制备，持续生产 |
| **分泌表达** | 蛋白分泌到培养基中，易于收集 | 毒性蛋白或需要正确折叠的蛋白 |
| **酵母表达** | 培养成本低，可进行简单的翻译后修饰 | 中等规模蛋白表达 |
| **昆虫细胞表达** | 可进行较复杂的翻译后修饰 | 需要复杂修饰的蛋白 |
| **哺乳动物细胞表达** | 可进行最复杂的翻译后修饰 | 需要高度修饰的蛋白 |

## 7. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 转染效率低 | 转染试剂选择不当或细胞状态差 | 优化转染条件，使用新鲜的细胞 |
| 蛋白表达量低 | 启动子强度不够或基因序列问题 | 更换启动子，优化基因序列 |
| 蛋白不溶性 | 表达温度过高或蛋白本身特性 | 降低表达温度，使用融合标签 |
| 蛋白降解 | 蛋白酶降解 | 加入蛋白酶抑制剂，缩短操作时间 |
| 纯化效率低 | 亲和标签结合效率低 | 优化洗涤和洗脱条件，使用更特异性的标签 |
| 翻译后修饰异常 | 表达系统不合适 | 选择合适的真核表达系统 |

## 8. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 可进行正确的蛋白折叠 | 培养成本高，生长缓慢 |
| 可进行复杂的翻译后修饰 | 表达水平相对较低 |
| 适合表达真核蛋白 | 操作步骤复杂，技术要求高 |
| 可分泌表达，易于收集 | 大规模培养困难 |
| 蛋白活性更接近天然状态 | 可能存在蛋白降解问题 |
