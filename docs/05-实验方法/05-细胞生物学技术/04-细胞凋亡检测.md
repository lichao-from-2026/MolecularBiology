# 04-细胞凋亡检测

## 1. 定义与核心概念

**细胞凋亡**（Apoptosis）是一种程序性细胞死亡过程，是细胞在生理或病理条件下的一种主动、有序的死亡方式。细胞凋亡检测技术用于识别和定量处于凋亡过程中的细胞，是研究细胞死亡机制、药物筛选和疾病诊断的重要工具。

## 2. 分子机制/原理

细胞凋亡的基本过程包括：
1. **起始阶段**：细胞接收到凋亡信号
2. **执行阶段**： caspase 蛋白酶级联激活
3. **降解阶段**：DNA断裂、细胞结构破坏
4. **清除阶段**：凋亡细胞被吞噬细胞清除

细胞凋亡检测技术基于凋亡过程中的各种生化和形态学变化，如细胞膜磷脂酰丝氨酸外翻、线粒体膜电位变化、DNA断裂、caspase激活等。

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| 细胞样本 | 待检测的细胞 |
| Annexin V-荧光染料 | 检测细胞膜磷脂酰丝氨酸外翻 |
| PI（碘化丙啶） | 检测细胞膜完整性 |
| 线粒体膜电位检测试剂盒 | 检测线粒体膜电位变化 |
| TUNEL检测试剂盒 | 检测DNA断裂 |
| caspase活性检测试剂盒 | 检测caspase激活 |
| 流式细胞仪 | 分析细胞凋亡 |
| 荧光显微镜 | 观察凋亡细胞形态 |
| 酶标仪 | 检测荧光或吸光度 |
| 移液器 | 移取液体 |
| 离心管 | 反应容器 |

## 4. 实验步骤

### 步骤1：Annexin V/PI双染法

- **目的**：检测早期和晚期凋亡细胞
- **操作**：
  1. **细胞收集**：
     - 收集细胞悬液，离心收集细胞
     - 用PBS洗涤细胞2次
  2. **染色**：
     - 重悬细胞于Binding Buffer中
     - 加入Annexin V-荧光染料和PI
     - 室温孵育15分钟，避光
  3. **流式细胞分析**：
     - 加入Binding Buffer终止反应
     - 流式细胞仪分析
     - 凋亡细胞分为早期凋亡（Annexin V+PI-）和晚期凋亡（Annexin V+PI+）
- **注意事项**：
  - 染色后应尽快分析，避免细胞死亡
  - 应设置阴性对照和阳性对照

### 步骤2：线粒体膜电位检测

- **目的**：检测线粒体膜电位变化，评估凋亡早期事件
- **操作**：
  1. **细胞处理**：
     - 培养细胞至适当密度
     - 加入凋亡诱导剂处理
  2. **染色**：
     - 加入线粒体膜电位染料（如JC-1）
     - 37°C孵育15-30分钟
  3. **分析**：
     - 流式细胞仪分析或荧光显微镜观察
     - 正常细胞：红色荧光（聚合态JC-1）
     - 凋亡细胞：绿色荧光（单体JC-1）
- **注意事项**：
  - 染料浓度应优化
  - 孵育时间应适当

### 步骤3：TUNEL检测

- **目的**：检测DNA断裂，评估凋亡晚期事件
- **操作**：
  1. **细胞固定**：
     - 固定细胞或组织切片
     - 通透处理
  2. **TUNEL反应**：
     - 加入TdT酶和荧光标记的dUTP
     - 37°C孵育60分钟
  3. **分析**：
     - 荧光显微镜观察或流式细胞仪分析
     - 凋亡细胞细胞核呈绿色荧光
- **注意事项**：
  - 固定和通透条件应优化
  - 应设置阳性和阴性对照

### 步骤4：Caspase活性检测

- **目的**：检测caspase激活，评估凋亡执行阶段
- **操作**：
  1. **细胞裂解**：
     - 收集细胞，加入裂解缓冲液
     - 冰上孵育30分钟
     - 离心收集上清
  2. **酶活性检测**：
     - 加入caspase底物（如Ac-DEVD-pNA）
     - 37°C孵育2-4小时
  3. **分析**：
     - 酶标仪检测吸光度（405nm）
     - 计算caspase活性
- **注意事项**：
  - 裂解缓冲液应新鲜配制
  - 应设置阳性和阴性对照

### 步骤5：形态学检测

- **目的**：通过形态学变化检测凋亡细胞
- **操作**：
  1. **细胞固定**：
     - 固定细胞或组织切片
  2. **染色**：
     - 加入Hoechst或DAPI染色细胞核
  3. **观察**：
     - 荧光显微镜观察
     - 凋亡细胞特征：核固缩、核碎裂、凋亡小体形成
- **注意事项**：
  - 染色时间应适当
  - 应使用合适的荧光滤镜

## 5. 应用场景

- **药物筛选**：筛选诱导或抑制凋亡的药物
- **疾病研究**：研究疾病发生和发展中的细胞凋亡
- **毒理学研究**：评估化合物的细胞毒性
- **肿瘤研究**：研究肿瘤细胞的凋亡机制
- **神经科学**：研究神经细胞的凋亡
- **免疫学**：研究免疫细胞的凋亡

## 6. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **Annexin V/PI双染** | 检测早期和晚期凋亡 | 快速定量分析 |
| **线粒体膜电位检测** | 检测凋亡早期事件 | 研究凋亡机制 |
| **TUNEL检测** | 检测DNA断裂 | 组织切片凋亡检测 |
| **Caspase活性检测** | 检测caspase激活 | 研究凋亡信号通路 |
| **形态学检测** | 通过形态学变化检测凋亡 | 直观观察凋亡过程 |
| **DNA ladder检测** | 检测DNA片段化 | 经典凋亡检测方法 |

## 7. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 非特异性染色 | 染料浓度过高或孵育时间过长 | 优化染色条件 |
| 假阳性结果 | 细胞坏死或操作不当 | 设置合适的对照，优化实验条件 |
| 信号弱 | 染料浓度过低或凋亡率低 | 增加染料浓度，延长孵育时间 |
| 实验重复性差 | 实验条件不一致 | 标准化实验流程 |
| 细胞聚集 | 细胞悬液制备不当 | 确保制备单细胞悬液 |

## 8. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 多种检测方法，适应不同需求 | 不同方法检测凋亡的不同阶段，需结合使用 |
| 灵敏度高，可检测早期凋亡 | 部分方法操作复杂，耗时较长 |
| 可定量分析凋亡率 | 某些方法需要专业设备 |
| 可在细胞和组织水平检测 | 结果可能受到多种因素影响 |
| 技术相对成熟，应用广泛 | 不同细胞类型的凋亡特征可能不同 |
