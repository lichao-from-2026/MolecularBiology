# 02-流式细胞术

## 1. 定义与核心概念

**流式细胞术**（Flow Cytometry）是一种用于分析和分选细胞的技术，通过流式细胞仪对单个细胞进行快速、多参数的定量分析。它可以同时分析细胞的大小、粒度、荧光强度等多种参数，是细胞生物学、免疫学、肿瘤学等领域的重要研究工具。

## 2. 分子机制/原理

流式细胞术的基本原理是：
1. **细胞悬液制备**：将细胞制备成单细胞悬液
2. **荧光标记**：使用荧光染料或荧光抗体标记细胞
3. **细胞流动**：细胞悬液通过流式细胞仪的流动室，形成单细胞流
4. **激光照射**：激光照射细胞，激发荧光染料发出荧光
5. **信号检测**：检测细胞的散射光和荧光信号
6. **数据分析**：分析检测到的信号，获得细胞的各种参数
7. **细胞分选**：根据分析结果，分选特定的细胞群体

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| 细胞样本 | 待分析的细胞 |
| 荧光抗体 | 标记细胞表面或内部抗原 |
| 荧光染料 | 标记细胞DNA、RNA等 |
| 缓冲液（如PBS） | 洗涤和悬浮细胞 |
| 固定液（如多聚甲醛） | 固定细胞 |
| 通透液（如Triton X-100） | 通透细胞膜 |
| 流式细胞仪 | 分析和分选细胞 |
| 流式管 | 盛装细胞悬液 |
| 移液器 | 移取液体 |
| 离心机 | 离心细胞 |

## 4. 实验步骤

### 步骤1：细胞样本制备

- **目的**：制备单细胞悬液，用于流式细胞分析
- **操作**：
  1. **贴壁细胞**：
     - 用胰蛋白酶消化细胞
     - 加入培养基终止消化
     - 离心收集细胞
     - 用PBS洗涤细胞2次
     - 调整细胞浓度至1×10^6 cells/mL
  2. **悬浮细胞**：
     - 直接离心收集细胞
     - 用PBS洗涤细胞2次
     - 调整细胞浓度至1×10^6 cells/mL
- **注意事项**：
  - 应确保细胞为单细胞悬液，避免细胞聚集
  - 应保持细胞活力
  - 应调整合适的细胞浓度

### 步骤2：荧光标记

- **目的**：使用荧光抗体或染料标记细胞
- **操作**：
  1. **表面抗原标记**：
     - 将细胞悬液加入流式管
     - 加入适量荧光抗体
     - 4°C孵育30分钟，避光
     - 用PBS洗涤细胞2次
  2. **胞内抗原标记**：
     - 固定细胞：加入固定液，室温孵育15-30分钟
     - 通透细胞：加入通透液，室温孵育10-15分钟
     - 加入适量荧光抗体
     - 4°C孵育30分钟，避光
     - 用PBS洗涤细胞2次
  3. **DNA含量分析**：
     - 固定细胞
     - 加入DNA染料（如PI）
     - 室温孵育30分钟，避光
- **注意事项**：
  - 应使用合适的抗体浓度
  - 应设置阴性对照和阳性对照
  - 应避光操作，避免荧光淬灭

### 步骤3：流式细胞仪分析

- **目的**：使用流式细胞仪分析细胞
- **操作**：
  1. **仪器准备**：
     - 启动流式细胞仪，进行校准
     - 设置荧光通道和电压
  2. **样本分析**：
     - 将标记好的细胞悬液放入流式细胞仪
     - 运行分析程序
     - 收集数据
  3. **数据分析**：
     - 使用流式分析软件（如FlowJo）分析数据
     - 绘制散点图、直方图等
     - 计算细胞群体的比例和荧光强度
- **注意事项**：
  - 应定期校准流式细胞仪
  - 应设置合适的阈值，排除碎片和死细胞
  - 应收集足够的细胞数量以确保统计准确性

### 步骤4：细胞分选

- **目的**：根据分析结果，分选特定的细胞群体
- **操作**：
  1. **仪器准备**：
     - 启动流式细胞分选仪
     - 进行仪器校准
  2. **分选设置**：
     - 设置分选参数和目标细胞群体
     - 选择分选模式（如纯度优先或产量优先）
  3. **细胞分选**：
     - 将标记好的细胞悬液放入分选仪
     - 运行分选程序
     - 收集分选后的细胞
  4. **分选后处理**：
     - 培养分选后的细胞
     - 或进行其他实验分析
- **注意事项**：
  - 分选过程应保持无菌操作
  - 应根据实验需求选择合适的分选模式
  - 应验证分选效率和细胞活力

## 5. 应用场景

- **细胞表型分析**：分析细胞表面标志物的表达
- **细胞周期分析**：分析细胞在不同周期阶段的分布
- **细胞凋亡检测**：检测凋亡细胞
- **细胞增殖分析**：分析细胞的增殖能力
- **免疫细胞亚群分析**：分析不同免疫细胞亚群的比例
- **干细胞分析**：分析干细胞的特性
- **药物筛选**：筛选影响细胞功能的药物

## 6. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **传统流式细胞术** | 分析细胞表面和胞内标志物 | 一般细胞分析 |
| **分选流式细胞术** | 分选特定细胞群体 | 细胞分离和纯化 |
| **多色流式细胞术** | 同时检测多种标志物 | 复杂细胞亚群分析 |
| **成像流式细胞术** | 结合流式和显微镜技术 | 细胞形态和功能分析 |
| **质谱流式细胞术** | 使用金属标记，检测更多参数 | 超高参数细胞分析 |

## 7. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 细胞聚集 | 细胞悬液制备不当 | 确保制备单细胞悬液，可使用DNA酶处理 |
| 非特异性染色 | 抗体浓度过高或交叉反应 | 优化抗体浓度，使用合适的同型对照 |
| 荧光淬灭 | 光照或保存时间过长 | 避光操作，及时分析样本 |
| 仪器堵塞 | 细胞悬液中有碎片或聚集 | 过滤细胞悬液，去除碎片 |
| 数据重复性差 | 实验条件不一致 | 标准化实验流程，使用对照样本 |

## 8. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 快速分析大量细胞 | 设备昂贵，需要专业操作 |
| 多参数同时分析 | 样本制备较为复杂 |
| 可定量分析细胞特性 | 对样本质量要求高 |
| 可分选特定细胞群体 | 某些细胞类型可能难以分析 |
| 应用范围广泛 | 数据分析需要专业知识 |
