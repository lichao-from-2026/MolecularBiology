# 01-质粒提取技术

## 1. 定义与核心概念

**质粒提取技术**是一种从细菌中分离和纯化质粒DNA的实验方法。质粒是细菌染色体外的环状双链DNA分子，通常携带抗生素抗性基因等非必需基因。质粒提取技术是分子克隆、基因工程和DNA测序等实验的基础。

## 2. 分子机制/原理

质粒提取技术的基本原理是：
1. **细菌培养**：大量培养含有目标质粒的细菌
2. **细胞裂解**：使用碱或去污剂破坏细菌细胞壁和细胞膜
3. **蛋白质变性**：使用碱性溶液使蛋白质和染色体DNA变性
4. **质粒DNA复性**：通过酸性中和使质粒DNA复性，而染色体DNA保持变性状态
5. **离心分离**：通过离心去除细胞碎片和变性蛋白质
6. **质粒DNA沉淀**：使用乙醇或异丙醇沉淀质粒DNA
7. **质粒DNA洗涤与溶解**：去除杂质并溶解质粒DNA

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| 含有质粒的细菌菌株 | 质粒DNA来源 |
| LB液体培养基 | 培养细菌 |
| 抗生素 | 筛选含有质粒的细菌 |
| 溶液I（50mM葡萄糖，25mM Tris-HCl pH 8.0，10mM EDTA） | 悬浮细菌细胞 |
| 溶液II（0.2M NaOH，1% SDS） | 裂解细胞，变性蛋白质和DNA |
| 溶液III（3M NaAc pH 4.8） | 中和碱性溶液，使质粒DNA复性 |
| 苯酚/氯仿/异戊醇（25:24:1） | 去除蛋白质 |
| 无水乙醇 | 沉淀DNA |
| 70%乙醇 | 洗涤DNA |
| TE缓冲液（pH 8.0） | 溶解DNA |
| RNase A | 去除RNA |
| 离心机 | 分离细胞成分 |
| 移液器 | 移取液体 |
| 离心管 | 反应容器 |

## 4. 实验步骤

### 步骤1：细菌培养

- **目的**：大量培养含有目标质粒的细菌
- **操作**：
  1. 从平板上挑取含有目标质粒的单菌落
  2. 接种于5-10mL LB液体培养基（含相应抗生素）
  3. 37°C，200rpm振荡培养过夜（12-16小时）
- **注意事项**：
  - 抗生素浓度应根据质粒的抗性标记选择
  - 培养时间不宜过长，避免细菌进入稳定期

### 步骤2：细胞收集与裂解

- **目的**：收集细菌细胞并裂解，释放质粒DNA
- **操作**：
  1. 取1.5mL培养物，12000rpm离心1分钟
  2. 弃上清，倒置离心管在滤纸上去除残留液体
  3. 加入250μL溶液I，涡旋混匀，使细菌充分悬浮
  4. 加入250μL溶液II，轻轻颠倒混匀5-10次
  5. 冰上孵育2-3分钟
  6. 加入350μL溶液III，轻轻颠倒混匀5-10次
  7. 冰上孵育5分钟
- **注意事项**：
  - 溶液II应新鲜配制
  - 操作应轻柔，避免剧烈振荡导致染色体DNA断裂
  - 裂解时间不宜过长，避免质粒DNA被降解

### 步骤3：离心与上清收集

- **目的**：去除细胞碎片和变性蛋白质，收集含有质粒DNA的上清
- **操作**：
  1. 12000rpm离心10分钟
  2. 小心吸取上清至新的离心管中，避免吸入沉淀
- **注意事项**：
  - 离心时间应足够，确保沉淀完全
  - 吸取上清时应小心，避免吸入沉淀

### 步骤4：苯酚/氯仿抽提

- **目的**：进一步去除蛋白质杂质
- **操作**：
  1. 加入等体积苯酚/氯仿/异戊醇
  2. 轻轻颠倒混匀10分钟
  3. 12000rpm离心10分钟
  4. 吸取上层水相至新的离心管中
- **注意事项**：
  - 操作应轻柔，避免剧烈振荡导致DNA断裂
  - 吸取水相时应避免吸入中间的蛋白质层

### 步骤5：DNA沉淀

- **目的**：沉淀质粒DNA
- **操作**：
  1. 加入2倍体积无水乙醇
  2. 轻轻颠倒混匀
  3. -20°C沉淀30分钟或过夜
  4. 12000rpm离心15分钟
  5. 弃上清
- **注意事项**：
  - 乙醇应预冷，提高沉淀效率
  - 沉淀时间应足够，确保DNA完全沉淀

### 步骤6：DNA洗涤与溶解

- **目的**：去除杂质和盐分，溶解质粒DNA
- **操作**：
  1. 加入1mL 70%乙醇，轻轻颠倒洗涤
  2. 12000rpm离心5分钟
  3. 弃上清，倒置离心管在滤纸上去除残留液体
  4. 室温晾干DNA沉淀（避免过度干燥）
  5. 加入50-100μL TE缓冲液
  6. 37°C溶解1小时
  7. 加入RNase A（终浓度10μg/mL），37°C孵育30分钟
- **注意事项**：
  - 70%乙醇应预冷
  - DNA沉淀不宜过度干燥，否则难以溶解
  - RNase A应不含DNase

### 步骤7：质粒DNA质量与浓度检测

- **目的**：评估提取质粒DNA的质量和浓度
- **操作**：
  1. **琼脂糖凝胶电泳**：
     - 取2μL质粒DNA进行1%琼脂糖凝胶电泳
     - 观察DNA条带的完整性和纯度
  2. **分光光度计检测**：
     - 测定OD260/280比值（1.8-2.0为纯DNA）
     - 测定OD260值计算DNA浓度（1 OD260 = 50μg/mL双链DNA）
- **注意事项**：
  - 琼脂糖凝胶电泳可检测质粒DNA的完整性和纯度
  - 分光光度计可检测质粒DNA的浓度和纯度

## 5. 应用场景

- **基因克隆**：提供克隆所需的质粒载体
- **基因表达**：提供表达载体用于蛋白质表达
- **基因测序**：提供测序模板
- **基因工程**：构建重组DNA分子
- **转基因技术**：提供转基因载体
- **基因编辑**：提供CRISPR-Cas9等基因编辑载体

## 6. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **碱性裂解法** | 经典方法，成本低 | 常规质粒提取 |
| **柱层析法** | 操作简单，纯度高 | 快速质粒提取 |
| **磁珠法** | 自动化程度高，提取速度快 | 高通量质粒提取 |
| **大提质粒** | 提取量多，适用于大量质粒制备 | 大规模质粒制备 |
| **小提质粒** | 操作简单，适用于快速筛选 | 质粒快速筛选 |

## 7. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 质粒DNA产量低 | 细菌培养时间不足或质粒拷贝数低 | 延长培养时间，使用高拷贝数质粒 |
| 质粒DNA纯度低 | 蛋白质或RNA污染 | 增加苯酚/氯仿抽提次数，加入RNase A |
| 质粒DNA降解 | 操作过程中DNA酶污染 | 加入EDTA抑制DNA酶，使用无菌试剂 |
| 质粒DNA不溶解 | 沉淀过度干燥 | 避免过度干燥，延长溶解时间 |
| 质粒DNA有RNA污染 | RNase A处理不充分 | 增加RNase A用量，延长处理时间 |

## 8. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 操作相对简单，成本低 | 耗时较长，需要多个步骤 |
| 可获得高质量的质粒DNA | 需要专业设备（离心机等） |
| 可根据需要调整提取规模 | 提取过程中可能损失DNA |
| 适用于多种后续实验 | 某些质粒提取困难（如大质粒） |
| 技术成熟，应用广泛 | 可能存在染色体DNA污染 |
