# 03-感受态细胞制备与转化

## 1. 定义与核心概念

**感受态细胞**是指能够摄取外源DNA的细菌细胞。**转化**是指将外源DNA导入感受态细胞的过程。感受态细胞制备与转化是分子克隆中的基本技术，用于将重组DNA分子导入细菌细胞，以实现基因克隆、表达和功能研究。

## 2. 分子机制/原理

### 感受态细胞的制备原理
1. **氯化钙处理**：使用低渗的氯化钙溶液处理细菌，使细胞膜通透性增加
2. **冷休克**：通过低温处理使细胞膜形成孔隙
3. **热激**：通过短暂的高温处理促进DNA进入细胞
4. **恢复培养**：在富含营养的培养基中培养，使细胞恢复正常生理状态

### 转化的机制
1. **DNA吸附**：DNA分子吸附到细胞表面
2. **DNA进入**：通过细胞膜上的孔隙进入细胞
3. **DNA整合**：质粒DNA在细胞内自主复制，或整合到染色体中
4. **表达筛选**：通过抗生素抗性等标记筛选成功转化的细胞

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| 大肠杆菌菌株（如DH5α、TOP10） | 感受态细胞的来源 |
| LB液体培养基 | 培养细菌 |
| LB固体培养基 | 筛选转化子 |
| 0.1M CaCl2溶液（预冷） | 制备感受态细胞 |

| 离心机 | 收集细胞 |
| 移液器 | 移取液体 |
| 离心管 | 反应容器 |
| 水浴锅 | 控制温度 |
| -80°C冰箱 | 保存感受态细胞 |

## 4. 实验步骤

### 步骤1：感受态细胞的制备

- **目的**：制备能够摄取外源DNA的大肠杆菌感受态细胞
- **操作**：
  1. **细菌培养**：
     - 从-80°C冰箱取出大肠杆菌菌株，在LB平板上划线活化
     - 37°C培养过夜
     - 挑取单菌落接种于5mL LB液体培养基
     - 37°C，200rpm振荡培养过夜
  2. **扩大培养**：
     - 取1mL过夜培养物转接于100mL LB液体培养基
     - 37°C，200rpm振荡培养至OD600约0.4-0.6
  3. **感受态细胞制备**：
     - 将培养物冰浴10分钟
     - 4°C，4000rpm离心10分钟，收集细胞
     - 弃上清，用预冷的0.1M CaCl2溶液50mL重悬细胞
     - 冰浴30分钟
     - 4°C，4000rpm离心10分钟，收集细胞
     - 弃上清，用预冷的0.1M CaCl2溶液10mL重悬细胞
     - 加入50%甘油至终浓度15%
     - 分装到1.5mL离心管中，每管100μL
     - -80°C保存
- **注意事项**：
  - 所有操作应在冰上进行
  - 细胞密度应控制在OD600 0.4-0.6
  - CaCl2溶液应预冷
  - 感受态细胞应立即使用或-80°C保存

### 步骤2：转化反应

- **目的**：将外源DNA导入感受态细胞
- **操作**：
  1. **感受态细胞 thawing**：
     - 从-80°C冰箱取出感受态细胞，冰上 thawing
  2. **DNA添加**：
     - 向感受态细胞中加入1-10μL质粒DNA或连接产物（约10-100ng）
     - 轻轻混匀，冰浴30分钟
  3. **热激**：
     - 42°C热激90秒
     - 立即冰浴2分钟
  4. **恢复培养**：
     - 加入900μL LB液体培养基
     - 37°C，200rpm振荡培养1小时
  5. **涂布筛选**：
     - 取100-200μL培养物涂布于含抗生素的LB固体培养基
     - 37°C倒置培养过夜
- **注意事项**：
  - DNA用量应适当，过多会抑制转化
  - 热激时间和温度应严格控制
  - 恢复培养时间应足够，确保抗生素抗性基因表达

### 步骤3：转化子的验证

- **目的**：验证转化是否成功，筛选阳性克隆
- **操作**：
  1. **菌落PCR**：
     - 挑取单菌落，用无菌牙签或枪头蘸取少量菌苔
     - 作为模板进行PCR扩增
     - 琼脂糖凝胶电泳验证
  2. **质粒提取与酶切**：
     - 挑取单菌落接种于LB液体培养基（含抗生素）
     - 培养过夜后提取质粒
     - 限制性酶切验证
  3. **测序验证**：
     - 对阳性克隆进行DNA测序
     - 确认插入片段的正确性
- **注意事项**：
  - 应设置阴性对照（未转化的感受态细胞）
  - 应设置阳性对照（已知质粒的转化）
  - 测序验证是最可靠的方法

## 5. 应用场景

- **基因克隆**：将重组DNA分子导入细菌细胞
- **基因表达**：在细菌中表达重组蛋白
- **质粒扩增**：大量扩增质粒DNA
- **突变体筛选**：筛选含有特定突变的克隆
- **文库构建**：构建cDNA文库或基因组文库
- **基因功能研究**：在细菌中研究基因功能

## 6. 技术变体

| 变体 | 特点 | 适用场景 |
|-----|------|---------|
| **CaCl2法** | 经典方法，操作简单 | 常规转化 |
| **电转化法** | 转化效率高，适用于大片段DNA | 大片段DNA转化 |
| **PEG介导转化** | 适用于某些难转化的菌株 | 特殊菌株转化 |
| **化学转化法** | 商业化感受态细胞，使用方便 | 快速转化 |
| **热激转化** | 适用于常规质粒转化 | 一般转化实验 |

## 7. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 转化效率低 | 感受态细胞质量差或DNA量不当 | 优化感受态细胞制备条件，调整DNA用量 |
| 无转化子 | 抗生素浓度过高或DNA未正确连接 | 检查抗生素浓度，验证DNA连接产物 |
| 假阳性克隆 | 载体自连或污染 | 使用双酶切，严格无菌操作 |
| 感受态细胞保存后活性下降 | 保存条件不当 | 确保-80°C保存，避免反复冻融 |
| 菌落生长缓慢 | 抗生素浓度过高或细胞状态差 | 调整抗生素浓度，使用新鲜的感受态细胞 |

## 8. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 操作相对简单，技术成熟 | 转化效率受多种因素影响 |
| 可用于多种大肠杆菌菌株 | 对某些大片段DNA转化效率低 |
| 成本相对较低 | 感受态细胞保存时间有限 |
| 可实现高通量转化 | 某些菌株难以制备感受态 |
| 应用广泛 | 需要专门的设备（如-80°C冰箱） |
