# 01-农杆菌介导转化

## 1. 定义与核心概念

**农杆菌介导转化**（Agrobacterium-mediated transformation）是一种利用农杆菌（Agrobacterium tumefaciens）将外源基因导入植物细胞的分子生物学技术。农杆菌是一种天然的植物病原菌，能够通过其 Ti 质粒（Tumor-inducing plasmid）将 T-DNA（Transfer DNA）转移到植物基因组中，从而实现外源基因的稳定整合和表达。

## 2. 分子机制/原理

农杆菌介导转化的基本原理是：
1. **农杆菌识别**：农杆菌通过其表面的受体识别植物伤口释放的信号分子（如乙酰丁香酮）
2. **Vir 基因激活**：信号分子诱导农杆菌 Ti 质粒上的 Vir 基因表达
3. **T-DNA 加工**：Vir 基因产物将 Ti 质粒上的 T-DNA 切割并加工成单链 T-DNA
4. **T-DNA 转移**：单链 T-DNA 与 Vir 蛋白形成复合物，通过农杆菌的 Type IV 分泌系统转移到植物细胞
5. **T-DNA 整合**：T-DNA 进入植物细胞核后，通过非同源重组或同源重组整合到植物基因组中
6. **基因表达**：整合的 T-DNA 上的基因在植物细胞中表达，产生新的表型

## 3. 实验材料

| 试剂/材料 | 用途 |
|---------|------|
| 农杆菌菌株（如 EHA105、GV3101） | 用于转化的农杆菌 |
| 植物材料（水稻愈伤组织、烟草叶片等） | 转化受体 |
| 表达载体（如 pCAMBIA 系列） | 携带外源基因的载体 |
| YEP 培养基 | 培养农杆菌 |
| 抗生素（如 Kanamycin、Rifampicin） | 筛选农杆菌转化子 |
| 乙酰丁香酮（AS） | 诱导 Vir 基因表达 |
| MS 培养基 | 植物组织培养 |
| 卡那霉素或潮霉素 | 筛选植物转化子 |
| 头孢霉素或羧苄青霉素 | 抑制农杆菌生长 |
| 酶解液（纤维素酶、果胶酶） | 制备原生质体（可选） |

## 4. 实验步骤

### 步骤1：农杆菌的培养与转化

- **目的**：将含有外源基因的表达载体导入农杆菌，获得可用于转化的工程菌
- **操作**：
  1. 从-80°C 冰箱取出农杆菌菌株，在 YEP 平板（含相应抗生素）上划线活化
  2. 挑取单菌落接种于 5mL YEP 液体培养基（含相应抗生素）
  3. 28°C，200rpm 振荡培养过夜
  4. 取 1mL 培养物转接于 50mL YEP 液体培养基（含相应抗生素）
  5. 28°C，200rpm 振荡培养至 OD600 约 0.5-0.8
  6. 4°C，5000rpm 离心 10 分钟收集菌体
  7. 用 10mM CaCl2 重悬菌体，制备感受态细胞
  8. 加入表达载体 DNA，冰浴 30 分钟
  9. 液氮速冻 5 分钟，37°C 热激 5 分钟
  10. 加入 1mL YEP 液体培养基，28°C 培养 2-4 小时
  11. 涂布于 YEP 平板（含相应抗生素），28°C 培养 2-3 天
  12. 挑取单克隆验证，确认载体已成功导入
- **注意事项**：
  - 农杆菌感受态细胞应新鲜制备，避免反复冻融
  - 热激时间和温度应严格控制
  - 抗生素浓度应根据农杆菌菌株和载体进行调整

### 步骤2：植物材料的准备

- **目的**：制备适合农杆菌感染的植物材料
- **操作**：
  1. **水稻愈伤组织**：
     - 取成熟水稻种子，去壳
     - 75% 乙醇消毒 1 分钟，0.1% HgCl2 消毒 15 分钟
     - 无菌水冲洗 5-6 次
     - 接种于愈伤组织诱导培养基
     - 25°C 暗培养 2-3 周，诱导愈伤组织
     - 挑选生长良好的愈伤组织进行转化
  2. **烟草叶片**：
     - 取 4-6 周龄烟草无菌苗的叶片
     - 切成 0.5-1cm² 的小块
     - 预培养 24 小时
- **注意事项**：
  - 植物材料的消毒应彻底，避免污染
  - 愈伤组织应选择质地松散、呈淡黄色的胚性愈伤
  - 叶片材料应选择健康、无病虫害的叶片

### 步骤3：农杆菌感染与共培养

- **目的**：让农杆菌与植物材料充分接触，实现 T-DNA 的转移
- **操作**：
  1. 将验证阳性的农杆菌接种于 YEP 液体培养基（含相应抗生素和 100μM AS）
  2. 28°C，200rpm 振荡培养至 OD600 约 0.5-0.8
  3. 4°C，5000rpm 离心 10 分钟收集菌体
  4. 用含有 100μM AS 的 MS 液体培养基重悬菌体，调整 OD600 至 0.5
  5. 将植物材料浸泡在农杆菌悬浮液中 10-30 分钟，期间轻轻摇晃
  6. 取出植物材料，用无菌滤纸吸干多余的菌液
  7. 将植物材料接种于共培养基（含 100μM AS）
  8. 25°C 暗培养 2-3 天
- **注意事项**：
  - 农杆菌浓度应适当，过高会导致过度生长，过低会影响转化效率
  - 共培养时间应根据植物材料进行调整
  - 共培养基的 pH 应控制在 5.2-5.8

### 步骤4：抗性筛选与再生

- **目的**：筛选成功转化的植物细胞，并再生为完整植株
- **操作**：
  1. 共培养结束后，将植物材料转移到筛选培养基（含抗生素和抑菌剂）
  2. 25°C 光照培养，每 2-3 周继代一次
  3. 待抗性愈伤组织形成后，转移到分化培养基
  4. 待芽分化后，转移到生根培养基
  5. 待根系发达后，炼苗并移栽到土壤中
- **注意事项**：
  - 筛选抗生素浓度应根据植物材料进行调整
  - 抑菌剂应选择对植物细胞毒性小的种类
  - 再生过程中应注意光照和温度的控制

### 步骤5：转化植株的验证

- **目的**：验证外源基因是否成功整合到植物基因组中并表达
- **操作**：
  1. **PCR 验证**：提取转化植株的基因组 DNA，用特异性引物进行 PCR 扩增
  2. **Southern blot 验证**：检测外源基因的整合拷贝数
  3. **RT-PCR 或 qRT-PCR**：检测外源基因的转录水平
  4. **Western blot**：检测外源蛋白的表达
  5. **表型观察**：观察转化植株的表型变化
- **注意事项**：
  - 应设置阳性对照和阴性对照
  - 验证方法应根据实验目的选择
  - 应选择多个转化事件进行验证

## 5. 应用场景

- **基因功能研究**：通过过表达或沉默特定基因，研究其在植物生长发育中的功能
- **作物改良**：导入抗虫、抗病、抗逆等基因，提高作物的产量和品质
- **植物生物反应器**：利用植物表达药用蛋白、工业酶等
- **分子标记辅助育种**：导入易于检测的标记基因，辅助传统育种
- **基础研究**：研究植物基因表达调控、信号转导等分子机制

## 6. 注意事项与常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 转化效率低 | 农杆菌活力不足或植物材料状态不佳 | 优化农杆菌培养条件，选择合适的植物材料 |
| 农杆菌过度生长 | 共培养时间过长或抑菌剂浓度不足 | 缩短共培养时间，增加抑菌剂浓度 |
| 抗性愈伤不分化 | 筛选压力过大或分化条件不适 | 调整抗生素浓度，优化分化培养基 |
| 假阳性 | 农杆菌污染或瞬时表达 | 严格无菌操作，进行分子验证 |
| 基因沉默 | 拷贝数过高或位置效应 | 选择单拷贝插入的转化事件 |

## 7. 优缺点分析

| 优点 | 缺点 |
|-----|------|
| 可以实现外源基因的稳定整合 | 转化效率因植物种类而异 |
| 可以转移较大的 DNA 片段 | 操作步骤较为繁琐，耗时较长 |
| 整合拷贝数相对较低，基因沉默现象较少 | 对某些植物种类转化效率较低 |
| 技术相对成熟，应用广泛 | 可能存在位置效应，影响基因表达 |
| 可以同时转移多个基因 | 需要组织培养技术支持 |
