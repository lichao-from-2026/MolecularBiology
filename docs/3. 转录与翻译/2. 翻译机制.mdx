# 二、翻译机制

## 1. 定义与核心概念

### 1.1 翻译 (Translation)

**定义**：翻译是指以 mRNA 为模板合成蛋白质的过程，是基因表达的第二步，将 mRNA 中的遗传信息转化为蛋白质的氨基酸序列。

**核心概念**：
- **核糖体**：蛋白质合成的场所
- **tRNA**：携带氨基酸并识别密码子
- **密码子**：mRNA 上编码氨基酸的三个碱基
- **起始因子**：参与翻译起始的蛋白质
- **延伸因子**：参与翻译延伸的蛋白质
- **释放因子**：参与翻译终止的蛋白质

## 2. 核糖体结构

### 2.1 原核生物核糖体

- **组成**：30S 小亚基 + 50S 大亚基 = 70S 核糖体
- **rRNA**：16S rRNA (小亚基)、23S rRNA 和 5S rRNA (大亚基)
- **蛋白质**：约 50 种蛋白质

### 2.2 真核生物核糖体

- **组成**：40S 小亚基 + 60S 大亚基 = 80S 核糖体
- **rRNA**：18S rRNA (小亚基)、28S rRNA、5.8S rRNA 和 5S rRNA (大亚基)
- **蛋白质**：约 80 种蛋白质

### 2.3 核糖体活性位点

- **A 位**：氨基酰-tRNA 结合位点
- **P 位**：肽酰-tRNA 结合位点
- **E 位**：退出位点
- **肽酰转移酶中心**：催化肽键形成

## 3. 遗传密码

### 3.1 密码子表

- **起始密码子**：AUG (编码甲硫氨酸)
- **终止密码子**：UAA、UAG、UGA
- **密码子数量**：64 个密码子，其中 61 个编码氨基酸

### 3.2 密码子特性

- **简并性**：多个密码子编码同一个氨基酸
- **通用性**：几乎所有生物使用相同的密码子表
- **摆动性**：tRNA 反密码子与 mRNA 密码子的配对存在摆动
- **方向性**：从 5' 到 3' 阅读

## 4. 翻译过程

### 4.1 起始

1. **原核生物起始**：
   - 30S 小亚基结合到 mRNA 的 SD 序列
   - 起始 tRNA (fMet-tRNAfMet) 结合到 P 位
   - 50S 大亚基结合，形成 70S 起始复合物

2. **真核生物起始**：
   - 40S 小亚基结合到 mRNA 的 5' 帽子
   - 扫描到起始密码子 AUG
   - 起始 tRNA (Met-tRNAMet) 结合到 P 位
   - 60S 大亚基结合，形成 80S 起始复合物

### 4.2 延伸

1. **进位**：
   - 氨基酰-tRNA 进入 A 位
   - 由延伸因子 EF-Tu (原核) 或 eEF1 (真核) 协助

2. **肽键形成**：
   - 肽酰转移酶催化肽键形成
   - P 位上的氨基酸转移到 A 位的氨基酸上

3. **移位**：
   - 核糖体沿 mRNA 移动一个密码子
   - 由延伸因子 EF-G (原核) 或 eEF2 (真核) 协助
   - 肽酰-tRNA 从 A 位移动到 P 位
   - 空载 tRNA 从 P 位移动到 E 位并释放

### 4.3 终止

1. **识别终止密码子**：
   - 释放因子识别终止密码子
   - 原核生物：RF1 识别 UAA、UAG；RF2 识别 UAA、UGA
   - 真核生物：eRF1 识别所有终止密码子

2. **肽链释放**：
   - 肽酰转移酶活性转变为水解酶活性
   - 肽链从 tRNA 上释放

3. **核糖体解离**：
   - 核糖体亚基解离
   - mRNA 和 tRNA 释放

## 5. 翻译后加工

### 5.1 蛋白质折叠

- **分子伴侣**：协助蛋白质正确折叠
- **内质网**：真核生物中蛋白质折叠的场所
- **二硫键形成**：稳定蛋白质结构

### 5.2 蛋白质修饰

| 修饰类型 | 功能 |
|----------|------|
| 磷酸化 | 调控酶活性、信号传导 |
| 糖基化 | 蛋白质定位、稳定性 |
| 甲基化 | 调控蛋白质功能 |
| 乙酰化 | 调控蛋白质活性 |
| 泛素化 | 标记蛋白质降解 |
| 脂酰化 | 蛋白质膜定位 |

### 5.3 蛋白质定位

- **信号肽**：引导蛋白质到特定位置
- **核定位信号**：引导蛋白质进入细胞核
- **线粒体靶向序列**：引导蛋白质进入线粒体
- **内质网信号肽**：引导蛋白质进入内质网

## 6. 翻译调控

### 6.1 翻译起始调控

- **mRNA 结构**：5' UTR 长度和二级结构
- **起始密码子上下文**：Kozak 序列
- **起始因子活性**：磷酸化状态
- **mRNA 稳定性**：3' UTR 中的调控元件

### 6.2 翻译延伸调控

- **延伸因子活性**：磷酸化状态
- **tRNA 可用性**：氨基酸供应
- **核糖体停滞**：mRNA 二级结构

### 6.3 翻译终止调控

- **释放因子活性**：调控终止效率
- **通读**：抑制终止，产生 longer 蛋白质

## 7. 应用

### 7.1 基因表达研究

- **Western blot**：检测蛋白质表达
- **ELISA**：定量蛋白质表达
- **免疫荧光**：蛋白质定位
- **蛋白质组学**：全蛋白质组分析

### 7.2 基因工程

- **重组蛋白表达**：在宿主细胞中表达蛋白质
- **融合蛋白**：标签蛋白便于纯化
- **蛋白质工程**：改造蛋白质功能

### 7.3 药物研发

- **抗生素**：抑制细菌翻译
- **蛋白质抑制剂**：调控特定蛋白质功能
- **mRNA 疫苗**：利用翻译表达抗原
