# 一、CRISPR-Cas9 基因编辑技术

## 1. 定义与核心概念

### 1.1 CRISPR-Cas9

**定义**：CRISPR-Cas9 是一种革命性的基因编辑技术，利用细菌的适应性免疫系统来精确修改基因组。

**核心概念**：
- **CRISPR**：成簇的规律间隔短回文重复序列
- **Cas9**：CRISPR 相关蛋白 9，一种核酸酶
- **sgRNA**：单指导 RNA，引导 Cas9 到目标位点
- **PAM**：原间隔基序相邻基序，Cas9 识别的序列
- **双链断裂**：Cas9 切割 DNA 产生的断裂
- **修复机制**：非同源末端连接 (NHEJ) 和同源定向修复 (HDR)

## 2. 基本原理

### 2.1 CRISPR-Cas9 系统组成

- **Cas9 蛋白**：
  - 核酸酶活性：切割 DNA 双链
  - 结构：含有 RuvC 和 HNH 两个核酸酶结构域

- **sgRNA**：
  - 引导序列：与目标 DNA 互补
  - 支架序列：与 Cas9 结合

- **PAM 序列**：
  - 通常为 NGG
  - 位于目标序列下游
  - 是 Cas9 识别的必要序列

### 2.2 工作机制

1. **识别**：
   - sgRNA 引导 Cas9 到目标 DNA 序列
   - Cas9 识别 PAM 序列

2. **切割**：
   - Cas9 的两个核酸酶结构域切割 DNA 双链
   - 在目标位点产生双链断裂 (DSB)

3. **修复**：
   - **非同源末端连接 (NHEJ)**：容易产生插入/缺失突变
   - **同源定向修复 (HDR)**：利用模板进行精确修复

## 3. 构建 CRISPR-Cas9 系统

### 3.1 sgRNA 设计

- **目标序列选择**：
  - 长度：18-20 bp
  - 下游有 PAM 序列 (NGG)
  - 避免重复序列
  - 避免脱靶效应

- **设计工具**：
  - CRISPR Design (MIT)
  - sgRNA Scorer
  - Benchling

### 3.2 载体构建

- **表达载体**：
  - 含有 Cas9 编码序列
  - 含有 sgRNA 表达 cassette
  - 含有选择标记

- **常用载体**：
  - pSpCas9(BB)-2A-Puro
  - pX330
  - pLentiCRISPRv2

### 3.3 递送方法

- **质粒转染**：
  - 适用于细胞系
  - 使用脂质体或电转

- **病毒递送**：
  - 慢病毒：适用于难转染细胞
  - 腺病毒：瞬时表达
  - AAV：适用于体内递送

- **RNP 递送**：
  - 直接递送 Cas9 蛋白和 sgRNA 复合物
  - 减少脱靶效应
  - 适用于体内编辑

## 4. 应用

### 4.1 基因敲除

- **原理**：通过 NHEJ 产生移码突变
- **应用**：
  - 研究基因功能
  - 疾病模型构建
  - 作物改良

### 4.2 基因敲入

- **原理**：通过 HDR 插入外源基因
- **应用**：
  - 基因标记
  - 基因修复
  - 转基因动物

### 4.3 基因编辑

- **原理**：通过 HDR 精确修改基因序列
- **应用**：
  - 修复基因突变
  - 引入特定突变
  - 疾病治疗

### 4.4 转录调控

- **dCas9**：失活的 Cas9，无核酸酶活性
- **应用**：
  - CRISPRi：抑制基因表达
  - CRISPRa：激活基因表达
  - 表观遗传修饰

### 4.5 基因筛选

- **原理**：通过 CRISPR 文库进行全基因组筛选
- **应用**：
  - 识别癌症驱动基因
  - 发现药物靶点
  - 研究基因功能网络

## 5. 脱靶效应

### 5.1 定义
- Cas9 切割非目标位点的现象

### 5.2 检测方法

- **GUIDE-seq**：
  - 标记 DSB 位点
  - 测序检测脱靶位点

- **CIRCLE-seq**：
  - 捕获 DSB 位点
  - 高灵敏度检测

- **DISCOVER-seq**：
  - 基于修复蛋白招募
  - 检测体内脱靶

### 5.3 减少脱靶效应的方法

- **优化 sgRNA**：
  - 选择特异性高的 sgRNA
  - 使用 sgRNA 设计工具

- **改进 Cas9**：
  - HiFi Cas9：提高特异性
  - Cas9 nickase：产生单链断裂
  - Base editors：无需 DSB

- **递送优化**：
  - 使用 RNP 复合物
  - 控制 Cas9 表达时间

## 6. 安全性考虑

### 6.1 脱靶效应
- 可能导致非预期的基因突变

### 6.2 免疫原性
- Cas9 蛋白可能引起免疫反应

### 6.3 染色体异常
- 可能导致染色体易位或缺失

### 6.4 伦理问题
- 生殖细胞编辑的伦理争议
- 基因增强的伦理考量

## 7. 技术改进

### 7.1 碱基编辑器

- **原理**：在不产生 DSB 的情况下修改单个碱基
- **类型**：
  - CBE：C→T 转换
  - ABE：A→G 转换
- **应用**：修复点突变疾病

### 7.2 Prime 编辑

- **原理**：结合 Cas9 nickase 和逆转录酶
- **特点**：
  - 无需 DSB
  - 无需 donor DNA
  - 精确编辑
- **应用**：修复多种类型的突变

### 7.3 靶向 RNA

- **Cas13**：靶向 RNA 的 CRISPR 系统
- **应用**：
  - RNA 编辑
  - 基因沉默
  - 病毒检测
