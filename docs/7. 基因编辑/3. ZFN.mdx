# 三、ZFN 基因编辑技术

## 1. 定义与核心概念

### 1.1 ZFN (Zinc Finger Nuclease)

**定义**：ZFN 是一种基因编辑技术，由锌指蛋白 (ZFP) 和 FokI 核酸酶组成，能够精确识别和切割特定 DNA 序列。

**核心概念**：
- **锌指蛋白 (ZFP)**：由多个锌指结构域组成，每个锌指识别 3 个碱基对
- **FokI**：一种限制性内切酶，需要二聚化才具有切割活性
- **DNA 识别**：锌指蛋白通过锌指结构域识别特定 DNA 序列
- **双链断裂**：FokI 切割 DNA 产生的断裂
- **修复机制**：非同源末端连接 (NHEJ) 和同源定向修复 (HDR)

## 2. 基本原理

### 2.1 ZFN 结构

- **锌指结构域**：
  - 每个锌指由约 30 个氨基酸组成
  - 包含 α-螺旋和 β-折叠
  - 通过锌离子稳定结构

- **FokI 核酸酶**：
  - 切割 DNA 双链
  - 需要二聚化才具有活性
  - 产生粘性末端

### 2.2 锌指识别密码

- **每个锌指识别 3 个碱基对**
- **锌指的 α-螺旋区域与 DNA 大沟相互作用**
- **特定氨基酸残基决定识别的碱基**

### 2.3 工作机制

1. **识别**：
   - 两个 ZFN 分别识别目标 DNA 的两条链
   - 识别序列通常为 9-18 bp

2. **二聚化**：
   - 两个 FokI 结构域相互作用形成二聚体

3. **切割**：
   - FokI 二聚体切割 DNA 双链
   - 在目标位点产生双链断裂 (DSB)

4. **修复**：
   - **非同源末端连接 (NHEJ)**：容易产生插入/缺失突变
   - **同源定向修复 (HDR)**：利用模板进行精确修复

## 3. ZFN 构建

### 3.1 锌指设计

- **模块组装法**：
  - 基于已知的锌指-DNA 相互作用
  - 组装多个锌指结构域

- **选择法**：
  - 噬菌体展示
  - 酵母单杂交
  - 细菌单杂交

### 3.2 常用锌指库

- **Sangamo**：商业化锌指库
- **Wright Lab**：开源锌指库
- **Barbas Lab**：锌指资源

### 3.3 表达载体

- **组成**：
  - 锌指蛋白编码序列
  - FokI 编码序列
  - 启动子
  - 选择标记

- **常用载体**：
  - pZFN
  - pcDNA-ZFN

## 4. 递送方法

- **质粒转染**：
  - 适用于细胞系
  - 使用脂质体或电转

- **病毒递送**：
  - 慢病毒：适用于难转染细胞
  - 腺病毒：瞬时表达

- **mRNA 递送**：
  - 体外转录 ZFN mRNA
  - 直接转染 mRNA
  - 减少基因组整合

## 5. 应用

### 5.1 基因敲除

- **原理**：通过 NHEJ 产生移码突变
- **应用**：
  - 研究基因功能
  - 疾病模型构建
  - 作物改良

### 5.2 基因敲入

- **原理**：通过 HDR 插入外源基因
- **应用**：
  - 基因标记
  - 基因修复
  - 转基因动物

### 5.3 基因编辑

- **原理**：通过 HDR 精确修改基因序列
- **应用**：
  - 修复基因突变
  - 引入特定突变
  - 疾病治疗

### 5.4 作物改良

- **应用**：
  - 抗虫作物
  - 抗病作物
  - 品质改良

## 6. 优缺点

### 6.1 优点

- **特异性高**：识别序列较长，脱靶效应低
- **灵活性**：可设计识别特定 DNA 序列
- **效率较高**：在多种细胞类型中有效

### 6.2 缺点

- **设计复杂**：锌指设计需要专业知识
- **构建困难**：组装过程复杂
- **成本较高**：设计和构建成本高
- **耗时**：开发周期长

### 8.1 FokI 变体

- **E490K/K538R**：提高切割效率
- **Sharkey**：提高特异性
- **Goldy**：提高活性和特异性

### 8.2 锌指设计优化
