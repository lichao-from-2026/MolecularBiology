# 二、RNA 结构与功能

## 1. 定义与核心概念

### 1.1 RNA (Ribonucleic Acid)

**定义**：RNA 是一种单链核酸分子，在基因表达过程中起着重要的中介作用，包括传递遗传信息、参与蛋白质合成等多种功能。

**核心概念**：
- **核苷酸**：RNA 的基本组成单位，由磷酸、核糖和碱基组成
- **mRNA**：信使 RNA，携带遗传信息从 DNA 到核糖体
- **tRNA**：转运 RNA，携带氨基酸到核糖体
- **rRNA**：核糖体 RNA，组成核糖体的结构成分
- **非编码 RNA**：不编码蛋白质的 RNA 分子

## 2. 分子结构

### 2.1 一级结构
- **组成**：由核糖核苷酸通过磷酸二酯键连接而成的线性多聚体
- **碱基**：腺嘌呤 (A)、尿嘧啶 (U)、鸟嘌呤 (G)、胞嘧啶 (C)
- **方向**：5' → 3' 方向

### 2.2 二级结构
- **发夹结构**：RNA 单链自身折叠形成的双链区域
- **茎环结构**：由双链茎和单链环组成的结构
- **假结**：RNA 分子中的复杂折叠结构

### 2.3 三级结构
- **tRNA 的三叶草结构**：由氨基酸臂、反密码子环、D 环、TΨC 环组成
- **rRNA 的复杂折叠**：形成核糖体的功能区域

## 3. RNA 的类型与功能

### 3.1 mRNA (Messenger RNA)

**结构**：
- 5' 端帽子结构：m⁷GpppN
- 3' 端 poly(A) 尾
- 编码区：包含开放阅读框 (ORF)

**功能**：
- 携带遗传信息从 DNA 到核糖体
- 作为蛋白质合成的模板

### 3.2 tRNA (Transfer RNA)

**结构**：
- 三叶草结构
- 3' 端 CCA 序列：结合氨基酸
- 反密码子环：识别 mRNA 上的密码子

**功能**：
- 携带氨基酸到核糖体
- 识别 mRNA 上的密码子

### 3.3 rRNA (Ribosomal RNA)

**结构**：
- 原核生物：16S、23S、5S rRNA
- 真核生物：18S、28S、5.8S、5S rRNA

**功能**：
- 组成核糖体的结构成分
- 催化肽键形成

### 3.4 非编码 RNA

| 类型 | 长度 | 功能 |
|------|------|------|
| miRNA | 20-24 nt | 基因表达调控 |
| siRNA | 20-25 nt | 基因沉默 |
| lncRNA | >200 nt | 基因表达调控、染色质重塑 |
| circRNA | 环状 |  miRNA 海绵、基因调控 |
| snRNA | 100-300 nt | 剪接体组成成分 |
| snoRNA | 60-300 nt | rRNA 修饰 |

## 4. RNA 合成（转录）

### 4.1 转录过程
1. **起始**：RNA 聚合酶结合到启动子
2. **延伸**：RNA 聚合酶合成 RNA 链
3. **终止**：RNA 聚合酶脱离 DNA 模板

### 4.2 转录后加工

**mRNA 加工**：
- 5' 端加帽
- 3' 端加尾
- 剪接：去除内含子，连接外显子

**tRNA 加工**：
- 5' 端切割
- 3' 端添加 CCA
- 碱基修饰

**rRNA 加工**：
- 剪切前体 rRNA
- 碱基修饰

## 5. RNA 的降解

### 5.1 mRNA 降解
- **脱腺苷酸化**：去除 poly(A) 尾
- **脱帽**：去除 5' 端帽子
- **核酸酶降解**：5' → 3' 或 3' → 5' 方向降解

### 5.2 非编码 RNA 降解
- **miRNA 降解**：被核酸酶降解
- **siRNA 降解**：在 RNAi 过程中降解

## 6. 应用

### 6.1 基因表达分析
- **RT-PCR**：检测基因表达水平
- **RNA-seq**：全转录组分析
- **Northern blot**：检测特定 RNA 分子

### 6.2 基因沉默
- **RNAi**：利用 siRNA 沉默基因表达
- **antisense RNA**：通过互补序列抑制基因表达

### 6.3 疫苗开发
- **mRNA 疫苗**：利用 mRNA 表达抗原
- **病毒载体疫苗**：利用 RNA 病毒作为载体

### 6.4 诊断技术
- **核酸检测**：检测病原体 RNA
- **基因芯片**：分析基因表达谱
