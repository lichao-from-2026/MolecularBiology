# 二、qPCR 技术

## 1. 定义与核心概念

### 1.1 qPCR (Quantitative Real-time PCR)

**定义**：qPCR 是一种实时监测 PCR 扩增过程的技术，能够定量测定样品中特定 DNA 或 RNA 的含量。

**核心概念**：
- **实时监测**：每轮循环后检测荧光信号
- **Ct 值**：达到阈值时的循环数
- **标准曲线**：用于定量分析
- **相对定量**：比较不同样品的基因表达水平
- **绝对定量**：测定样品中靶基因的绝对拷贝数

## 2. 基本原理

### 2.1 荧光检测原理

1. **SYBR Green 法**：
   - 荧光染料与双链 DNA 结合
   - 结合后荧光强度增强
   - 优点：操作简单，成本低
   - 缺点：非特异性结合

2. **TaqMan 探针法**：
   - 荧光标记的探针与模板结合
   - Taq 酶的 5'-3' 外切酶活性切割探针
   - 荧光基团与淬灭基团分离，产生荧光
   - 优点：特异性高
   - 缺点：成本高，设计复杂

3. **Molecular Beacon 法**：
   - 发夹结构的探针
   - 与模板结合后发夹打开，产生荧光
   - 优点：特异性高
   - 缺点：设计复杂

### 2.2 qPCR 反应成分

| 成分 | 作用 | 浓度 |
|------|------|------|
| 模板 DNA/RNA | 提供扩增模板 | 1-100 ng |
| 引物 | 与模板结合 | 0.1-0.5 μM |
| 荧光染料/探针 | 检测扩增产物 | 按说明书 |
| dNTPs | 提供合成原料 | 0.2-0.5 mM |
| DNA 聚合酶 | 催化 DNA 合成 | 1-2 U |
| 缓冲液 | 提供反应环境 | 1× |
| Mg²⁺ | 酶活性必需 | 1.5-2.5 mM |

### 2.3 qPCR 反应程序

1. **预变性**：
   - 温度：94-98°C
   - 时间：30 秒-2 分钟

2. **循环反应**：
   - 变性：94-98°C，15-30 秒
   - 退火：50-65°C，30-60 秒
   - 延伸：72°C，30-60 秒（同时检测荧光）

3. **熔解曲线分析**：
   - 温度：从 60°C 缓慢升至 95°C
   - 目的：检测产物特异性

## 3. 定量分析方法

### 3.1 相对定量

- **ΔΔCt 法**：
  1. 计算目标基因和内参基因的 Ct 值
  2. 计算 ΔCt = Ct(目标基因) - Ct(内参基因)
  3. 计算 ΔΔCt = ΔCt(样品) - ΔΔCt(对照)
  4. 相对表达量 = 2^(-ΔΔCt)

- **内参基因**：
  - 选择稳定表达的基因
  - 如 GAPDH、β-actin、18S rRNA

### 3.2 绝对定量

- **标准曲线法**：
  1. 制备已知浓度的标准品
  2. 绘制 Ct 值与浓度的标准曲线
  3. 根据样品的 Ct 值计算浓度

- **标准品制备**：
  - 质粒 DNA
  - 体外转录的 RNA
  - 线性化的 DNA

## 4. qPCR 实验设计

### 4.1 实验对照

- **阳性对照**：确认反应体系正常
- **阴性对照**：检测污染
- **空白对照**：检测试剂污染
- **内参对照**：校正样品量差异

### 4.2 引物设计

- **长度**：18-25 bp
- **GC 含量**：40-60%
- **Tm 值**：58-62°C
- **产物长度**：70-200 bp
- **避免二级结构**：发夹、二聚体

### 4.3 RNA 样品处理

- **RNA 提取**：使用 TRIzol 或试剂盒
- **DNase 处理**：去除基因组 DNA 污染
- **反转录**：合成 cDNA
- **cDNA 质量**：检测浓度和纯度

## 5. qPCR 数据处理

### 5.1 质量控制

- **熔解曲线**：单峰表示特异性扩增
- **扩增效率**：90-110%
- **R² 值**：>0.99
- **重复性**：同一样品重复间 Ct 值差异 < 0.5

### 5.2 数据分析软件

- **仪器自带软件**：如 StepOne Software、LightCycler Software
- **第三方软件**：如 qBase、GenEx
- **统计分析**：使用 Excel、GraphPad Prism

## 6. 应用

### 6.1 基因表达分析

- **差异表达基因检测**：比较不同条件下的基因表达
- **时间 course 分析**：监测基因表达随时间的变化
- **组织特异性表达**：分析基因在不同组织中的表达

### 6.2 病原体检测

- **病毒定量**：如 HIV、HBV、HCV
- **细菌检测**：如结核杆菌、肺炎链球菌
- **病原体分型**：检测病原体的基因型

### 6.3 基因拷贝数变异

- **基因扩增检测**：如 HER2 扩增
- **基因缺失检测**：如 BRCA1/2 缺失
- **染色体异常**：如三体综合征

### 6.4 药物研发

- **药物靶点验证**：检测药物对基因表达的影响
- **药效评估**：监测治疗过程中的基因表达变化
- **耐药性检测**：检测耐药相关基因的突变

## 7. 注意事项

### 7.1 避免污染

- **实验环境**：使用专用 PCR 实验室
- **移液器**：使用带滤芯的枪头
- **操作顺序**：先准备试剂，后处理样品
- **紫外线照射**：定期照射工作台

### 7.2 优化反应条件

- **引物浓度**：优化至最佳浓度
- **退火温度**：使用梯度 PCR 优化
- **模板量**：避免过多或过少

### 7.3 数据可靠性

- **重复实验**：至少 3 次生物学重复
- **技术重复**：每个样品至少 2 次技术重复
- **数据标准化**：使用合适的内参基因
