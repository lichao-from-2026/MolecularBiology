# 三、测序数据分析

## 1. 定义与核心概念

### 1.1 测序数据分析 (Sequencing Data Analysis)

**定义**：测序数据分析是指对测序产生的原始数据进行处理、分析和解读的过程，以获取有生物学意义的信息。

**核心概念**：
- **原始数据**：测序仪产生的原始测序 reads
- **质量控制**：评估测序数据质量
- **比对**：将 reads 映射到参考基因组
- **变异检测**：识别基因组变异
- **表达分析**：定量基因表达水平
- **功能注释**：注释基因功能

## 2. 数据类型

### 2.1 DNA 测序数据

- **全基因组测序 (WGS)**：覆盖整个基因组
- **全外显子组测序 (WES)**：覆盖编码区
- **靶向测序**：覆盖特定区域
- **ChIP-seq**：检测蛋白质结合位点
- **ATAC-seq**：检测染色质可及性

### 2.2 RNA 测序数据

- **mRNA-seq**：检测编码 RNA
- **小 RNA-seq**：检测 miRNA、siRNA 等
- **lncRNA-seq**：检测长非编码 RNA
- **单细胞 RNA-seq**：检测单个细胞的转录组

### 2.3 其他测序数据

- **甲基化测序**：检测 DNA 甲基化
- **宏基因组测序**：检测环境微生物
- **16S rRNA 测序**：分析微生物群落

## 3. 分析流程

### 3.1 1. 原始数据质量控制

- **FastQC**：
  - 评估测序质量
  - 检测接头污染
  - 分析碱基分布

- **数据过滤**：
  - 去除低质量 reads
  - 去除接头序列
  - 去除重复 reads

### 3.2 2. 序列比对/组装

- **参考基因组比对**：
  - BWA：用于 Illumina 数据
  - Bowtie2：快速比对
  - HISAT2：适用于 RNA-seq

- **从头组装**：
  - SPAdes：适用于细菌基因组
  - Canu：适用于长读长数据
  - Trinity：适用于转录组组装

### 3.3 3. 变异检测

- **SNP 检测**：
  - GATK：金标准工具
  - SAMtools：轻量级工具
  - FreeBayes：适用于复杂变异

- **Indel 检测**：
  - GATK Indel  caller
  - Pindel：专门检测 Indels

- **结构变异检测**：
  - DELLY：检测各种结构变异
  - LUMPY：基于拆分 reads

### 3.4 4. 表达分析

- **基因表达定量**：
  - HTSeq：基于比对结果计数
  - featureCounts：快速计数
  - Kallisto：无需比对的定量

- **差异表达分析**：
  - DESeq2：适用于 RNA-seq
  - edgeR：适用于转录组数据
  - limma：适用于微阵列数据

### 3.5 5. 功能注释与富集分析

- **基因注释**：
  - Ensembl：综合注释数据库
  - RefSeq：参考序列数据库
  - UniProt：蛋白质数据库

- **富集分析**：
  - DAVID：功能注释工具
  - GO 分析：基因本体论分析
  - KEGG 分析：通路分析

## 4. 常用分析工具

| HISAT2 | RNA-seq 比对 | 考虑剪接位点 |

### 4.3 变异检测工具

### 4.4 表达分析工具

| 工具 | 用途 | 特点 |
|------|------|------|
| HTSeq | 基因表达计数 | 基于比对结果 |
| featureCounts | 基因表达计数 | 速度快 |

| DESeq2 | 差异表达分析 | 适用于 RNA-seq |

### 4.5 组装工具

| 工具 | 用途 | 特点 |
|------|------|------|
| SPAdes | 基因组组装 | 适用于细菌基因组 |
| Canu | 基因组组装 | 适用于长读长数据 |
| Trinity | 转录组组装 | 适用于 RNA-seq 数据 |
| Flye | 基因组组装 | 适用于 Nanopore 数据 |

## 5. 数据分析实例

### 5.1 WGS 数据分析流程

1. **原始数据质量控制**：
   - FastQC 评估质量
   - Trimmomatic 过滤数据

2. **参考基因组比对**：
   - BWA-MEM 比对
   - SAMtools 排序和索引

3. **变异检测**：
   - GATK HaplotypeCaller 检测变异
   - GATK VariantFiltration 过滤变异

4. **注释与分析**：
   - ANNOVAR 注释变异
   - 功能预测和分析

### 5.2 RNA-seq 数据分析流程

1. **原始数据质量控制**：
   - FastQC 评估质量
   - Trimmomatic 过滤数据

2. **参考基因组比对**：
   - HISAT2 比对
   - SAMtools 排序和索引

3. **基因表达定量**：
   - featureCounts 计数
   - 生成表达矩阵

4. **差异表达分析**：
   - DESeq2 分析差异表达基因
   - 筛选显著差异基因

5. **功能富集分析**：
   - GO 分析
   - KEGG 通路分析

## 7. 注意事项

### 7.1 数据质量

- **测序深度**：确保足够的测序深度
- **覆盖均匀性**：避免覆盖偏差
- **错误率**：控制测序错误率

### 7.2 分析参数

- **比对参数**：选择合适的比对参数
- **变异过滤**：设置合理的过滤阈值
- **统计方法**：选择合适的统计方法

### 7.3 结果验证

- **Sanger 测序**：验证重要变异
- **qPCR**：验证基因表达变化
- **功能实验**：验证生物学意义

### 7.4 计算资源

- **内存**：确保足够的内存
- **存储**：准备充足的存储空间
- **计算时间**：合理安排计算时间
