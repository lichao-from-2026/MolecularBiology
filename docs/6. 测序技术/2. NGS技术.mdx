# 二、下一代测序 (NGS)

## 1. 定义与核心概念

### 1.1 下一代测序 (Next-Generation Sequencing, NGS)

**定义**：NGS 是一种高通量 DNA 测序技术，能够同时对大量 DNA 片段进行测序，大大提高了测序速度和效率。

**核心概念**：
- **高通量**：一次测序 millions 到 billions 个 DNA 片段
- **边合成边测序**：在 DNA 合成过程中检测序列
- **桥式 PCR**：扩增 DNA 片段形成簇
- **荧光标记**：标记核苷酸以便检测
- **数据量**：产生大量测序数据

## 2. NGS 技术平台

### 2.1 Illumina 测序平台

- **原理**：边合成边测序
- **特点**：
  - 高准确性
  - 高 throughput
  - 较短读长 (50-300 bp)
- **应用**：全基因组测序、转录组测序、外显子组测序

### 2.2 Ion Torrent 测序平台

- **原理**：离子半导体测序
- **特点**：
  - 快速测序
  - 中等读长 (200-400 bp)
  - 实时检测
- **应用**：靶向测序、病原体检测

### 2.3 PacBio 测序平台

- **原理**：单分子实时测序
- **特点**：
  - 长读长 (10-25 kb)
  - 高错误率 (需校正)
  - 实时测序
- **应用**：基因组组装、复杂区域测序

### 2.4 Oxford Nanopore 测序平台

- **原理**：纳米孔测序
- **特点**：
  - 超长读长 (可达 Mb 级)
  - 实时测序
  - 便携式设备
- **应用**：现场测序、宏基因组学

## 3. NGS 测序流程

### 3.1 1. 文库制备

- **DNA 提取**：
  - 从样品中提取 DNA
  - 检测质量和浓度

- **DNA 片段化**：
  - 物理打断（超声、雾化）
  - 酶切打断

- **末端修复**：
  - 修复粘性末端
  - 添加磷酸基团

- **接头连接**：
  - 连接测序接头
  - 接头包含索引序列

- **片段选择**：
  - 凝胶电泳或磁珠选择
  - 选择合适大小的片段

- **PCR 扩增**：
  - 扩增文库
  - 富集含接头的片段

### 3.2 2. 测序反应

- **簇生成**：
  - Illumina：桥式 PCR
  - Ion Torrent：乳液 PCR
  - PacBio/Oxford Nanopore：单分子

- **测序**：
  - Illumina：边合成边测序
  - Ion Torrent：离子检测
  - PacBio：实时检测
  - Oxford Nanopore：纳米孔检测

### 3.3 3. 数据处理

- **原始数据过滤**：
  - 去除低质量 reads
  - 去除接头序列

- **比对**：
  - 将 reads 比对到参考基因组
  - 使用比对软件（如 BWA、Bowtie）

- **变异检测**：
  - 检测 SNPs、Indels
  - 使用变异检测软件（如 GATK）

- **组装**：
  - 从头组装（de novo assembly）
  - 使用组装软件（如 SPAdes、Canu）

## 4. NGS 应用

### 4.1 全基因组测序 (WGS)

- **定义**：测序整个基因组
- **应用**：
  - 疾病基因发现
  - 群体遗传学研究
  - 癌症基因组分析

### 4.2 全外显子组测序 (WES)

- **定义**：测序编码区（外显子）
- **应用**：
  - 孟德尔遗传病研究
  - 癌症驱动基因发现
  - 成本效益高

### 4.3 转录组测序 (RNA-seq)

- **定义**：测序转录组（mRNA）
- **应用**：
  - 基因表达分析
  - 差异表达基因检测
  - 可变剪接分析
  - 新基因发现

### 4.4 小 RNA 测序

- **定义**：测序小 RNA（miRNA、siRNA）
- **应用**：
  - miRNA 表达分析
  - 小 RNA 功能研究

### 4.5 靶向测序

- **定义**：测序特定区域
- **应用**：
  - 癌症相关基因检测
  - 药物靶点分析
  - 遗传病基因检测

### 4.6 宏基因组测序

- **定义**：测序环境样品中的所有 DNA
- **应用**：
  - 微生物多样性分析
  - 新物种发现
  - 环境监测

## 5. NGS 数据分析

### 5.1 数据分析流程

1. **原始数据处理**：
   - 质量控制
   - 接头去除
   -  reads 过滤

2. **比对/组装**：
   - 参考基因组比对
   - 从头组装

3. **变异检测**：
   - SNP 检测
   - Indel 检测
   - 结构变异检测

4. **注释与分析**：
   - 基因注释
   - 功能分析
   - 通路分析

### 5.2 常用软件

| 功能 | 软件 | 用途 |
|------|------|------|
| 质量控制 | FastQC | 评估测序质量 |
| 接头去除 | Trimmomatic | 去除接头和低质量序列 |
| 比对 | BWA, Bowtie2 | 将 reads 比对到参考基因组 |
| 变异检测 | GATK, SAMtools | 检测 SNPs 和 Indels |
| 组装 | SPAdes, Canu | 从头组装基因组 |
| 表达分析 | DESeq2, edgeR | 差异表达分析 |
| 功能注释 | DAVID, KEGG | 基因功能注释 |

## 6. 优缺点

### 6.1 优点

- **高通量**：一次测序大量 DNA 片段
- **低成本**：单位碱基测序成本低
- **速度快**：短时间内完成测序
- **应用广**：适用于多种测序需求

### 6.2 缺点

- **读长较短**：Illumina 平台读长有限
- **错误率**：不同平台错误率不同
- **数据量大**：需要大量存储空间和计算资源
- **分析复杂**：需要专业的生物信息学知识

## 7. 注意事项

### 7.1 样品准备

- **DNA 质量**：高质量 DNA 是成功的关键
- **DNA 浓度**：符合测序平台要求
- **样品污染**：避免交叉污染

### 7.2 文库制备

- **片段大小**：根据测序目的选择合适的片段大小
- **接头质量**：确保接头连接效率
- **PCR 循环数**：避免过度扩增

### 7.3 测序参数

- **测序深度**：根据研究目的选择合适的深度
- **读长**：根据应用选择合适的读长
- **重复序列**：注意重复序列的影响

### 7.4 数据分析

- **质量控制**：严格的质量控制
- **参考基因组**：选择合适的参考基因组
- **变异过滤**：设置合理的过滤参数
- **结果验证**：使用 Sanger 测序验证重要变异
