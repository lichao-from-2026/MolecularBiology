# 七、基因工程应用

## 1. 医学应用

### 1.1 重组蛋白质药物

**重组药物**是利用基因工程技术将外源基因导入宿主细胞，表达生产的新型药物。

**主要产品类型**：

| 类别 | 代表药物 | 适应症 |
|------|----------|--------|
| 激素类 | 胰岛素 | 糖尿病 |
| 干扰素 | IFN-α, IFN-β | 肝炎、多发性硬化症 |
| 白介素 | IL-2 | 肾癌、黑色素瘤 |
| 生长因子 | G-CSF, EPO | 贫血、化疗后中性粒细胞减少 |
| 凝血因子 | Factor VIII, VIIa | 血友病 |

**胰岛素发展历程**：

| 年代 | 技术 | 说明 |
|------|------|------|
| 1921 | 动物胰岛素 | 首次提取 |
| 1978 | 重组人胰岛素 | 基因工程生产 |
| 1982 | 商业化 | 第一代基因工程药物 |
| 2000s | 胰岛素类似物 | 更优的药代动力学 |

### 1.2 基因治疗

**基因治疗 (Gene Therapy)** 是指向患者细胞中引入正常基因，以纠正或补偿缺陷基因，从而治疗疾病的技术。

**治疗策略**：

| 策略 | 原理 | 应用 |
|------|------|------|
| 基因替代 | 引入正常基因替代缺陷基因 | 单基因遗传病 |
| 基因沉默 | 抑制异常基因表达 | 显性负性遗传病 |
| 基因编辑 | 直接修复突变基因 | 精准治疗 |
| 基因增补 | 引入新基因赋予新功能 | 复杂疾病 |

**已批准的基因治疗产品**：

| 产品 | 疾病 | 技术 |
|------|------|------|
| Luxturna | 遗传性视网膜疾病 | AAV 递送 |
| Zolgensma | 脊髓性肌萎缩 | AAV 递送 |
| Yescarta | 淋巴瘤 | CAR-T (体外) |
| Strimvelis | ADA-SCID | 干细胞 (体外) |
| Glybera | 脂蛋白脂肪酶缺乏 | AAV 递送 |

**在研基因治疗**：

| 疾病领域 | 代表疾病 | 进展 |
|----------|----------|------|
| 血液病 | 地中海贫血、血友病 | 临床试验 |
| 神经系统 | 帕金森病、阿尔茨海默病 | 临床试验 |
| 眼科 | 黄斑变性、色素性视网膜炎 | 临床试验 |
| 心血管 | 心力衰竭、血管生成 | 临床试验 |

### 1.3 细胞治疗

**CAR-T 细胞治疗**：

| 组成 | 功能 |
|------|------|
| 抗原受体 (CAR) | 识别肿瘤抗原 |
| 共刺激信号 | 激活 T 细胞 |
| 细胞内信号域 | 杀伤信号 |

**已批准 CAR-T 产品**：

| 产品 | 靶点 | 适应症 |
|------|------|--------|
| Kymriah | CD19 | ALL, DLBCL |
| Yescarta | CD19 | DLBCL, FL |
| Tecartus | CD19 | MCL |

### 1.4 疫苗

**重组疫苗**：

| 类型 | 原理 | 实例 |
|------|------|------|
| 重组亚单位疫苗 | 表达病原体蛋白 | 乙肝疫苗 |
| 重组载体疫苗 | 病毒载体递送抗原 | 埃博拉疫苗 |
| mRNA 疫苗 | mRNA 体内表达抗原 | 新冠疫苗 |
| DNA 疫苗 | DNA 体内表达抗原 | 多种在研 |

**mRNA 疫苗优势**：

- 研发速度快
- 安全性高（无感染风险）
- 可快速更新迭代
- 适用于多种疾病

## 2. 农业应用

### 2.1 转基因作物

**转基因作物 (Genetically Modified Crops)** 是指通过基因工程技术导入了外源基因的作物。

**主要性状**：

| 性状类型 | 代表作物 | 基因 |
|----------|----------|------|
| 抗虫 | Bt 玉米、棉花 | 苏云金杆菌 Cry 基因 |
| 抗除草剂 | 抗草甘膦大豆 | EPSPS 基因 |
| 抗病 | 抗病毒木瓜 | 外壳蛋白基因 |
| 抗逆 | 抗寒番茄 | 抗冻蛋白基因 |
| 品质改良 | 金大米 | 类胡萝卜素合成基因 |

**Bt 抗虫作物**：

- 原理：表达苏云金杆菌 (Bt) 产生的杀虫蛋白
- 目标害虫：鳞翅目、鞘翅目昆虫
- 优势：减少化学农药使用

**抗除草剂作物**：

- 原理：导入抗除草剂基因
- 实例：抗草甘膦大豆 (Roundup Ready)
- 优势：便于田间管理

### 2.2 基因编辑作物

**基因编辑作物**是通过基因编辑技术（如 CRISPR）直接改造作物基因组获得的作物。

| 作物 | 改良性状 | 技术 |
|------|----------|------|
| 大豆 | 油脂组成 | CRISPR |
| 水稻 | 抗病、产量 | CRISPR |
| 小麦 | 矮秆、抗白粉病 | CRISPR |
| 玉米 | 抗除草剂 | TALEN |
| 马铃薯 | 抗褐变 | CRISPR |

### 2.3 分子标记辅助育种

**分子标记 (Molecular Marker)** 是指与性状紧密关联的 DNA 序列变异，可用于辅助选择育种。

| 标记类型 | 特点 | 应用 |
|----------|------|------|
| SSR | 重复序列，多态性高 | 品种鉴定 |
| SNP | 单核苷酸变异，高通量 | 基因分型、全基因组选择 |
| AFLP | 限制性片段多态性 | 遗传多样性 |
| RAPD | 随机引物扩增 | 初步定位 |

**全基因组选择 (Genomic Selection)**：

- 利用全基因组标记预测育种值
- 加速育种进程
- 降低成本

### 2.4 动物基因工程

**转基因动物**：

| 动物 | 应用 | 技术 |
|------|------|------|
| 转基因小鼠 | 疾病模型 | 原核注射、胚胎干细胞 |
| 转基因大鼠 | 疾病模型 | 原核注射 |
| 转基因兔 | 药用蛋白生产 | 显微注射 |
| 转基因猪 | 器官移植、育种 | 体细胞核移植 |
| 转基因牛 | 乳腺生物反应器 | 显微注射 |

**体细胞核移植 (Clone)**：

1. 体细胞供核
2. 卵母细胞去核
3. 细胞融合
4. 胚胎发育
5. 移植代孕母体

### 2.5 合成生物学在农业中的应用

- **生物固氮**：改造非固氮菌固氮能力
- **生物农药**：工程菌生产天然农药
- **土壤修复**：工程菌降解污染物

## 3. 工业应用

### 3.1 工业酶生产

**工业酶 (Industrial Enzymes)** 广泛应用于食品、纺织、洗涤、造纸等行业。

| 行业 | 酶类 | 应用 |
|------|------|------|
| 食品 | 淀粉酶、蛋白酶 | 淀粉加工、奶酪成熟 |
| 纺织 | 淀粉酶、纤维素酶 | 退浆、牛仔布处理 |
| 洗涤 | 蛋白酶、淀粉酶、纤维素酶 | 清洁去污 |
| 造纸 | 木聚糖酶、纤维素酶 | 纸浆漂白 |
| 饲料 | 植酸酶、蛋白酶 | 提高饲料利用率 |

**酶的定向进化改造**：

- 提高热稳定性
- 增强有机溶剂耐受性
- 改变底物特异性
- 提高催化效率

### 3.2 生物能源

| 领域 | 技术 | 说明 |
|------|------|------|
| 乙醇 | 纤维素乙醇 | 降解木质纤维素生产乙醇 |
| 丁醇 | 丙酮丁醇发酵 | 工程菌改造 |
| 沼气 | 甲烷发酵 | 厌氧消化 |
| 生物柴油 | 油脂酯交换 | 工程微藻 |

**纤维素乙醇**：

```
纤维素 → 葡萄糖 → 乙醇
    ↑
木质纤维素酶 (工程菌生产)
```

## 9. 展望

### 9.1 机遇与挑战

**机遇**：

- 技术不断进步，成本持续降低
- 临床转化加速
- 多学科交叉融合

**挑战**：

- 伦理法规完善
- 生物安全问题
- 公众接受度

### 9.2 社会影响

- 推动医学治疗变革
- 促进农业可持续发展
- 带动生物经济发展
- 引发伦理思考
