# 四、DNA 测序技术

## 1. 定义与核心概念

### 1.1 DNA 测序的定义

**DNA 测序 (DNA Sequencing)** 是指确定 DNA 分子中核苷酸排列顺序的技术，是基因组学研究和分子生物学研究的基础。

### 1.2 核心概念

| 术语 | 英文 | 定义 |
|------|------|------|
| 读长 | Read Length | 单次测序获得的 DNA 片段长度 |
| 覆盖度 | Coverage | 每个碱基被测序的平均次数 |
| 质量值 | Q Score | 碱基识别的可信度指标 |
| 测序深度 | Sequencing Depth | 测序数据对基因组的覆盖程度 |
| reads | Reads | 测序产生的短 DNA 序列片段 |

## 2. Sanger 测序（第一代测序）

### 2.1 Sanger 测序原理

**Sanger 测序** 又称双脱氧链终止法，由 Frederick Sanger 于 1977 年发明。

**核心原理**：

1. DNA 模板与引物退火
2. DNA 聚合酶催化链延伸
3. 双脱氧核苷酸 (ddNTP) 随机掺入导致链终止
4. ddNTP 带有荧光标记
5. 毛细管电泳分离不同长度片段
6. 检测荧光信号确定碱基序列

**ddNTP 与 dNTP 的区别**：

| 特性 | dNTP | ddNTP |
|------|------|-------|
| 3' 羟基 | 有 | **无** |
| 掺入后 | 继续延伸 | **终止延伸** |
| 荧光标记 | 不标记 | 标记 |

### 2.2 Sanger 测序反应

**反应体系**：

| 成分 | 用量 |
|------|------|
| 模板 DNA | 100-500 ng |
| 引物 | 3.2 pmol |
| DNA 聚合酶 | 0.5 μL |
| dNTPs | 适当浓度 |
| ddNTPs | 荧光标记 |
| 缓冲液 | 1× |

**荧光标记**：

| ddNTP | 荧光颜色 |
|--------|----------|
| ddATP | 绿色 |
| ddTTP | 红色 |
| ddCTP | 蓝色 |
| ddGTP | 黄色 |

### 2.3 Sanger 测序流程

1. **模板制备**
   - 质粒 DNA：纯度 OD260/280 = 1.8-2.0
   - PCR 产物：纯化去除杂质

2. **测序反应**
   - 热循环：96°C 10 秒 → 50°C 5 秒 → 60°C 4 分钟，循环 25 次

3. **产物纯化**
   - 乙醇沉淀或磁珠纯化

4. **毛细管电泳**
   - 遗传分析仪自动检测

5. **数据分析**
   - 序列读取和质量评估

### 2.4 测序质量评估

**质量值 (Q Score)**：

```
Q = -10 × log10(P)
```

其中 P 是碱基识别错误概率

| Q 值 | 错误率 | 准确性 |
|------|--------|--------|
| Q10 | 10% | 90% |
| Q20 | 1% | 99% |
| Q30 | 0.1% | 99.9% |
| Q40 | 0.01% | 99.99% |

### 2.5 Sanger 测序的应用

| 应用领域 | 具体内容 |
|----------|----------|
| 基因克隆验证 | 确认插入片段序列 |
| 突变检测 | 点突变、小片段缺失/插入 |
| SNP 分型 | 单核苷酸多态性分析 |
| 法医学 | DNA 指纹分析 |
| 病原体鉴定 | 病毒/细菌分型 |

### 2.6 Sanger 测序的优缺点

| 优点 | 缺点 |
|------|------|
| 准确性极高 (>99.99%) | 通量低 |
| 操作简单 | 测序长度有限 (500-1000 bp) |
| 成本低（小规模） | 大规模测序成本高 |
| 读长适中 | 速度慢 |

## 3. 下一代测序（NGS）

### 3.1 NGS 定义

**下一代测序 (Next-Generation Sequencing, NGS)** 又称高通量测序，能够同时对 millions 到 billions 个 DNA 片段进行测序。

**技术特点**：

- 高通量：一次运行产生 G-T 级数据
- 低成本：大幅降低单个碱基测序成本
- 自动化：减少人工操作

### 3.2 NGS 技术平台

**Illumina 测序平台**

| 参数 | 说明 |
|------|------|
| 原理 | 边合成边测序 (SBS) |
| 读长 | 50-300 bp (双端) |
| 通量 | G-T 级/运行 |
| 准确性 | > 99.9% |
| 特点 | 高通量、高准确性 |

**Ion Torrent 测序平台**

| 参数 | 说明 |
|------|------|
| 原理 | 离子半导体测序 |
| 读长 | 200-400 bp |
| 通量 | M-G 级/运行 |
| 准确性 | > 99% |
| 特点 | 快速测序 |

**PacBio 测序平台**

| 参数 | 说明 |
|------|------|
| 原理 | 单分子实时测序 (SMRT) |
| 读长 | 10-25 kb |
| 准确性 | > 99.9% (HiFi reads) |
| 特点 | 长读长，直接检测修饰 |

**Oxford Nanopore 测序平台**

| 参数 | 说明 |
|------|------|
| 原理 | 纳米孔测序 |
| 读长 |可达 Mb 级 |
| 准确性 | > 99.9% (Q20+) |
| 特点 | 超长读长，实时测序，便携 |

### 3.3 NGS 测序平台对比

| 平台 | 读长 | 通量 | 准确性 | 主要应用 |
|------|------|------|--------|----------|
| Illumina | 50-300 bp | 极高 | 极高 | WGS、WES、RNA-seq |
| Ion Torrent | 200-400 bp | 中 | 高 | 靶向测序 |
| PacBio | 10-25 kb | 高 | 高 | 基因组组装 |
| Nanopore | kb-Mb 级 | 高 | 高 | 超长读长应用 |

### 3.4 Illumina 测序原理

**文库制备**：

1. DNA 片段化（超声或酶切）
2. 末端修复，添加 A 尾
3. 连接测序接头
4. PCR 扩增富集
5. 片段大小选择

**流动池 (Flow Cell) 反应**：

1. 文库变性，单链 DNA 结合到 flow cell
2. 桥式 PCR 扩增，形成簇 (Cluster)
3. 簇线性化，准备测序

**边合成边测序 (SBS)**：

1. 加入荧光标记的 dNTP
2. DNA 聚合酶催化合成
3. 激光激发，检测荧光
4. 化学切割荧光基团，恢复 3'-OH
5. 重复循环

### 3.5 NGS 测序流程

**整体流程**：

```
样品采集
    ↓
DNA/RNA 提取
    ↓
文库制备
    ↓
上机测序
    ↓
原始数据生成
    ↓
数据分析
    ↓
结果解读
```

**文库制备步骤**：

1. **DNA 片段化**
   - 物理方法：超声、雾化
   - 酶切法：核酸酶

2. **末端修复**
   - 修复粘性末端
   - 添加磷酸基团

3. **接头连接**
   - 连接测序接头
   - 包含 index 序列

4. **片段选择**
   - 凝胶电泳或磁珠选择
   - 选择合适大小片段

5. **PCR 扩增**
   - 富集含接头的片段

### 3.6 NGS 应用领域

**全基因组测序 (WGS)**：

- 覆盖整个基因组
- 发现 SNP、Indel、SV
- 全基因组关联研究

**全外显子组测序 (WES)**：

- 覆盖编码区（约 1-2% 基因组）
- 高深度测序
- 孟德尔遗传病研究

**转录组测序 (RNA-seq)**：

- mRNA 定量分析
- 差异表达基因检测
- 可变剪接分析
- 新转录本发现

**小 RNA 测序**：

- miRNA 表达分析
- 发现新小 RNA

**靶向测序**：

- 疾病相关基因 panel
- 癌症基因检测
- 遗传病诊断

**宏基因组测序**：

- 环境微生物多样性
- 新物种发现
- 微生物群落分析

**ChIP-seq**：

- 转录因子结合位点
- 组蛋白修饰分析

**ATAC-seq**：

- 染色质可及性
- 开放染色质区域

## 4. 测序数据分析

### 4.1 数据类型

**DNA 测序数据**：

| 数据类型 | 描述 | 应用 |
|----------|------|------|
| WGS | 全基因组测序 | 变异检测 |
| WES | 全外显子组测序 | 编码区变异 |
| 靶向测序 | 特定区域 | 疾病基因检测 |
| ChIP-seq | 蛋白-DNA 结合 | 转录因子研究 |
| ATAC-seq | 染色质可及性 | 调控区域研究 |

**RNA 测序数据**：

| 数据类型 | 描述 | 应用 |
|----------|------|------|
| mRNA-seq | mRNA 测序 | 基因表达分析 |
| smallRNA-seq | 小 RNA 测序 | miRNA 分析 |
| scRNA-seq | 单细胞 RNA 测序 | 细胞异质性 |

### 4.2 数据分析流程

**DNA-seq 分析流程**：

```
原始数据 (FASTQ)
    ↓
质量控制 (FastQC)
    ↓
过滤去噪 (Trimmomatic)
    ↓
比对到参考基因组 (BWA/Bowtie2)
    ↓
处理比对结果 (SAMtools/Picard)
    ↓
变异检测 (GATK/SAMtools)
    ↓
变异注释 (ANNOVAR/VEP)
    ↓
下游分析
```

**RNA-seq 分析流程**：

```
原始数据 (FASTQ)
    ↓
质量控制 (FastQC)
    ↓
过滤去噪 (Trimmomatic)
    ↓
比对到参考基因组 (HISAT2/STAR)
    ↓
表达定量 (featureCounts/HTSeq)
    ↓
差异表达分析 (DESeq2/edgeR)
    ↓
功能富集分析 (GO/KEGG)
    ↓
下游分析
```

### 4.3 质量控制工具

**FastQC**：

- 评估测序质量
- 生成质量报告
- 检查项目：
  - 碱基质量分布
  - GC 含量分布
  - 接头污染
  - 重复序列

**Trimmomatic**：

- 过滤和修剪 reads
- 去除接头序列
- 去除低质量 reads

### 4.4 序列比对工具

| 工具 | 适用数据 | 特点 |
|------|----------|------|
| BWA | DNA-seq | 高精度 |
| Bowtie2 | DNA-seq | 快速，内存低 |
| HISAT2 | RNA-seq | 考虑剪接 |
| STAR | RNA-seq | 高精度，高速度 |

### 4.5 变异检测工具

**SNP/Indel 检测**：

| 工具 | 特点 |
|------|------|
| GATK | 金标准，功能全面 |
| SAMtools | 轻量级，快速 |
| FreeBayes | 适合复杂变异 |

**结构变异检测**：

| 工具 | 检测类型 |
|------|----------|
| DELLY | 缺失、倒位、易位 |
| LUMPY | 多种结构变异 |
| CNVnator | 拷贝数变异 |

### 4.6 表达分析工具

**表达定量**：

| 工具 | 原理 | 特点 |
|------|------|------|
| featureCounts | 比对计数 | 快速准确 |
| HTSeq | 比对计数 | 成熟稳定 |
| Kallisto | 无需比对 | 速度极快 |
| Salmon | 无需比对 | 准确性高 |

**差异表达分析**：

| 工具 | 统计模型 | 适用场景 |
|------|----------|----------|
| DESeq2 | 负二项分布 | RNA-seq |
| edgeR | 负二项分布 | 转录组数据 |
| limma | 线性模型 | 微阵列 |

### 4.7 功能注释与富集分析

**基因注释数据库**：

| 数据库 | 内容 |
|--------|------|
| Ensembl | 综合基因组注释 |
| RefSeq | 参考序列注释 |
| UniProt | 蛋白质功能 |
| KEGG | 代谢通路 |

**富集分析**：

| 分析类型 | 工具 |
|----------|------|
| GO 分析 | DAVID, clusterProfiler |
| KEGG 分析 | KEGG database, clusterProfiler |
| GSEA | GSEA software |

## 5. 三代测序技术对比

| 特性 | 第一代 (Sanger) | 第二代 (Illumina) | 第三代 (PacBio/Nanopore) |
|------|-----------------|-------------------|------------------------|
| 读长 | 500-1000 bp | 50-300 bp | **kb-Mb 级** |
| 通量 | 低 | 极高 | 高 |
| 准确性 | > 99.99% | > 99.9% | > 99% |
| 成本/样品 | 高 | **低** | 中 |
| 测序时间 | 小时-天 | 天 | 小时-天 |
| 修饰检测 | 不能 | 不能 | **可以直接检测** |

## 6. 测序技术发展前沿

### 6.1 单细胞测序

| 技术 | 应用 |
|------|------|
| scRNA-seq | 单细胞转录组 |
| scATAC-seq | 单细胞染色质可及性 |
| 空间转录组 | 空间位置信息 |

### 6.2 临床应用

- **遗传病诊断**：全外显子组测序
- **肿瘤精准医疗**：液体活检
- **感染性疾病**：宏基因组测序
- **药物基因组学**：个体化用药

### 6.3 发展趋势

1. **长读长测序普及**：成本持续降低
2. **单细胞多组学**：多组学联合分析
3. **空间组学**：保留空间信息
4. **临床转化加速**：精准医疗应用

## 7. 常见问题与解决方案

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| Sanger 峰图异常 | 模板质量差 | 重新制备模板 |
| Sanger 无信号 | 引物问题 | 更换引物 |
| NGS 数据质量差 | 样品降解 | 优化样品制备 |
| 比对率低 | 参考基因组不匹配 | 选择正确参考 |
| 变异假阳性高 | 分析参数不当 | 调整过滤参数 |

> :::tip 学习提示
> DNA 测序技术从 Sanger 测序到 NGS 再到第三代测序，经历了快速发展。理解各代技术的原理和特点，有助于根据研究目的选择合适的测序方案。测序数据分析能力同样重要，需要系统学习生物信息学知识。
> :::

> :::warning 注意
> 测序数据的质量控制是保证分析结果可靠性的关键。每一步分析都需要进行质量检查，避免引入误差或假阳性结果。
> :::
