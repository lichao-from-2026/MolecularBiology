# 二、基因克隆技术

## 1. 定义与核心概念

### 1.1 基因克隆的定义

**基因克隆 (Gene Cloning)** 是指在体外将特定 DNA 片段插入到载体 DNA 分子中，形成重组 DNA 分子，然后导入宿主细胞进行复制和扩增的技术。

**核心目标**：

- 获得大量纯净的特定 DNA 片段
- 稳定保存遗传材料
- 改造生物遗传特性
- 大量表达特定蛋白质

### 1.2 核心概念

| 术语 | 英文 | 定义 |
|------|------|------|
| 克隆载体 | Cloning Vector | 用于携带和扩增外源 DNA 的载体分子 |
| 表达载体 | Expression Vector | 用于在宿主细胞中表达外源基因的载体 |
| 限制性酶切 | Restriction Digestion | 使用限制性内切酶切割 DNA |
| 连接反应 | Ligation | DNA 连接酶催化形成磷酸二酯键 |
| 转化 | Transformation | 外源 DNA 进入细菌细胞 |
| 转染 | Transfection | 外源 DNA 进入真核细胞 |

## 2. 克隆载体系统

### 2.1 质粒载体

**质粒 (Plasmid)** 是存在于细菌细胞质中的环状双链 DNA 分子，是最常用的克隆载体。

**质粒的基本结构**：

```
┌─────────────────────────────────────┐
│            复制起点 (ori)            │
│  ┌───────────────────────────────┐  │
│  │         选择标记基因            │  │
│  │    (抗生素抗性：AmpR/KanR)    │  │
│  └───────────────────────────────┘  │
│  ┌───────────────────────────────┐  │
│  │         多克隆位点 (MCS)       │  │
│  │    (多种限制酶切位点聚集)       │  │
│  └───────────────────────────────┘  │
│            启动子序列（如需要）       │
└─────────────────────────────────────┘
```

**常用质粒载体**：

| 载体 | 特点 | 应用 |
|------|------|------|
| pUC19 | 高拷贝，含 LacZ | 基础克隆 |
| pBR322 | 双抗性 | 传统克隆 |
| pGEM-T | TA 克隆专用 | PCR 产物克隆 |
| pET 系列 | T7 启动子 | 原核表达 |

### 2.2 噬菌体载体

**λ 噬菌体载体**：

- 基因组：约 48 kb
- 克隆容量：约 25 kb
- 特点：可包装成噬菌体颗粒
- 应用：基因组文库构建

**M13 噬菌体载体**：

- 基因组：单链环状 DNA，约 7 kb
- 特点：产生单链 DNA
- 应用：测序、定点突变

### 2.3 人工染色体载体

| 载体 | 宿主 | 克隆容量 | 应用 |
|------|------|----------|------|
| BAC | 大肠杆菌 | 100-300 kb | 基因组文库 |
| YAC | 酵母 | 100-2000 kb | 大片段克隆 |
| PAC | 大肠杆菌 | 100-300 kb | 基因组分析 |
| HAC | 哺乳动物细胞 | Mb 级 | 人造染色体 |

### 2.4 表达载体

**原核表达载体**：

| 元件 | 功能 |
|------|------|
| 启动子 | 启动转录 (lac, tac, T7) |
| RBS | 核糖体结合位点 |
| 终止子 | 终止转录 |
| 标签序列 | 蛋白纯化/检测 (His, GST) |

**真核表达载体**：

| 元件 | 功能 |
|------|------|
| 启动子 | 真核启动子 (CMV, SV40) |
| poly(A) 信号 | mRNA 加工 |
| 内含子 | 可选，促进表达 |

## 3. 限制性内切酶

### 3.1 限制性内切酶的发现

**限制性内切酶 (Restriction Endonuclease)** 是细菌产生的一类核酸酶，能够识别并切割特定的 DNA 序列，是细菌防御噬菌体感染的机制。

**命名规则**：属名首字母大写 + 种名前两个字母小写 + 菌株号 + 罗马数字

例如：*Escherichia coli* strain RY13 → EcoRI

### 3.2 限制性内切酶的分类

| 类型 | 识别与切割 | 辅因子 | 应用 |
|------|-----------|--------|------|
| I | 识别位点与切割位点分离 | ATP + SAM | 很少使用 |
| **II** | **同一位点** | **Mg²⁺** | **主要工具酶** |
| III | 识别位点与切割位点分离 | ATP | 很少使用 |

### 3.3 常用限制性内切酶

| 酶 | 识别序列 | 切割位点 | 末端类型 |
|----|----------|----------|----------|
| EcoRI | GAATTC | G↓AATTC | 5' 粘性末端 |
| HindIII | AAGCTT | A↓AGCTT | 5' 粘性末端 |
| BamHI | GGATCC | G↓GATCC | 5' 粘性末端 |
| XhoI | CTCGAG | C↓TCGAG | 5' 粘性末端 |
| PstI | CTGCAG | CTGCA↓G | 3' 粘性末端 |
| SmaI | CCCGGG | CCC↓GGG | 平末端 |
| NotI | GCGGCCGC | GC↓GGCCGC | 5' 粘性末端 |

### 3.4 酶切反应

**标准反应体系**：

| 成分 | 体积 | 终浓度 |
|------|------|--------|
| 10× 缓冲液 | 2 μL | 1× |
| DNA | 1-5 μg | - |
| 限制性内切酶 | 1-5 U | 1 U/μg DNA |
| ddH₂O | 补至 20 μL | - |

**反应条件**：

- 温度：37°C（大多数酶）
- 时间：1-3 小时
- 终止：65°C 20 分钟或加入 EDTA

## 4. DNA 连接反应

### 4.1 DNA 连接酶

**DNA 连接酶 (DNA Ligase)** 催化 DNA 片段之间形成磷酸二酯键。

**T4 DNA 连接酶**（最常用）：

- 来源：T4 噬菌体感染大肠杆菌
- 辅酶：ATP
- 功能：连接粘性末端和平末端

### 4.2 连接反应原理

1. **酶-AMP 中间体形成**
   ```
   Ligase + ATP → Ligase-AMP + PPi
   ```

2. **DNA 末端结合**
   ```
   Ligase-AMP + DNA → Ligase-AMP-DNA
   ```

3. **磷酸二酯键形成**
   ```
   3'-OH + 5'-P → 磷酸二酯键 + AMP
   ```

### 4.3 连接反应体系

| 成分 | 体积 | 说明 |
|------|------|------|
| 10× 连接缓冲液 | 2 μL | 含 ATP |
| 载体 DNA | 50-100 ng | 线性化载体 |
| 插入片段 | 适量 | 载体:插入 = 1:3-1:10 |
| T4 DNA 连接酶 | 1 U | - |
| ddH₂O | 补至 20 μL | - |

### 4.4 反应条件

| 末端类型 | 温度 | 时间 |
|----------|------|------|
| 粘性末端 | 16°C | 1-4 小时 |
| 平末端 | 25°C | 4-16 小时 |

## 5. 克隆策略

### 5.1 传统克隆策略

**粘性末端克隆**：

1. 使用相同限制酶切割载体和插入片段
2. 凝胶回收纯化
3. 连接反应
4. 转化宿主细胞

**TA 克隆**：

- 利用 Taq 酶在 PCR 产物 3' 端添加 A 尾
- 使用带 T 末端的载体
- 适用于 PCR 产物直接克隆

### 5.2 无缝克隆策略

**Gibson 组装**：

| 特点 | 说明 |
|------|------|
| 原理 | 依赖 20-40 bp 重叠序列 |
| 步骤 | 一步法完成连接 |
| 优势 | 无需酶切位点，可组装多片段 |
| 应用 | 复杂载体构建、代谢工程 |

**Golden Gate 克隆**：

| 特点 | 说明 |
|------|------|
| 原理 | Type IIS 限制酶 + 连接酶循环 |
| 优势 | 无疤痕、可同时组装多片段 |
| 应用 | 模块化组装、标准化构建 |

**Gateway 克隆**：

| 特点 | 说明 |
|------|------|
| 原理 | λ 噬菌体位点特异性重组 |
| 优势 | 快速在不同载体间转移基因 |
| 应用 | 基因功能分析、蛋白表达 |

### 5.3 克隆策略对比

| 策略 | 末端要求 | 片段数量 | 难度 |
|------|----------|----------|------|
| 粘性末端 | 相同酶切位点 | 2 个 | 低 |
| TA 克隆 | A 尾 | 1 个 | 低 |
| Gibson | 重叠序列 | **多片段** | 中 |
| Golden Gate | Type IIS 酶切位点 | **多片段** | 中 |
| Gateway | att 位点 | 1 个 | 中 |

## 7. 转化与转染

### 7.1 大肠杆菌转化

**CaCl₂ 感受态制备**：

1. 培养细胞至 OD600 = 0.4-0.6
2. 冰浴冷却，4°C 离心
3. 重悬于冰冷 0.1 M CaCl₂
4. 冰浴 30 分钟
5. 分装，-80°C 保存

**热激转化**：

1. 冰浴：DNA 与感受态细胞混合，冰浴 30 分钟
2. 热激：42°C，90 秒
3. 冰浴：2 分钟
4. 复苏：37°C 振荡培养 1 小时
5. 涂板：含抗生素的 LB 平板
6. 培养：37°C 过夜

**电转化**：

1. 制备电转化感受态细胞（甘油洗涤）
2. 混合 DNA 和细胞
3. 电击（1.8-2.5 kV）
4. 立即加入 SOC 培养基
5. 复苏培养后涂板

### 7.2 真核细胞转染

| 方法 | 原理 | 特点 |
|------|------|------|
| 脂质体转染 | 脂质体与 DNA 形成复合物 | 操作简单 |
| 电转 | 电击穿孔 | 效率高，细胞损伤大 |
| 磷酸钙法 | 形成 DNA-磷酸钙沉淀 | 经典方法 |
| 病毒感染 | 慢病毒/腺病毒载体 | 稳定表达 |
| PEI 转染 | 高分子聚合物介导 | 高效低成本 |

## 8. 筛选与鉴定

### 8.1 抗性筛选

**原理**：载体携带抗生素抗性基因，只有成功转化的细胞才能在含抗生素的培养基中生长。

**常用抗生素**：

| 抗生素 | 筛选基因 | 工作浓度 |
|--------|----------|----------|
| 氨苄青霉素 | ampR | 100 μg/mL |
| 卡那霉素 | kanR | 50 μg/mL |
| 壮观霉素 | specR | 100 μg/mL |
| 四环素 | tetR | 10 μg/mL |

### 8.2 蓝白斑筛选

**原理**：利用 lacZ 基因的 α-互补现象

**步骤**：

1. 在平板中加入 IPTG 和 X-Gal
2. 涂布转化细胞
3. 37°C 培养过夜

**结果判断**：

| 颜色 | 基因型 | 说明 |
|------|--------|------|
| 蓝色 | lacZ 完整 | 空载体 |
| 白色 | lacZ 被插入片段破坏 | **重组载体** |

### 8.3 菌落 PCR

**原理**：直接以菌落为模板进行 PCR 扩增

**引物设计**：

- 载体通用引物
- 基因特异性引物

**判断**：阳性克隆应扩增出预期大小的片段

### 8.4 酶切鉴定

1. 提取质粒 DNA
2. 使用限制性内切酶消化
3. 凝胶电泳分析

**验证要点**：

- 插入片段大小是否正确
- 载体是否线性化
- 是否有自连或串联插入

### 8.5 测序验证

**金标准**：DNA 测序确认序列正确性

**测序引物**：

- 载体通用引物（M13、T7 等）
- 基因特异性引物

## 9. 实验流程示例

### 9.1 常规基因克隆流程

```
获取目的基因 (PCR/合成)
        ↓
design 引物 (添加酶切位点)
        ↓
PCR 扩增目的基因
        ↓
凝胶电泳检测并回收
        ↓
双酶切载体和插入片段
        ↓
凝胶回收纯化
        ↓
T4 DNA 连接酶连接
        ↓
转化大肠杆菌感受态
        ↓
抗性筛选 (含抗生素平板)
        ↓
蓝白斑筛选 (如适用)
        ↓
菌落 PCR 鉴定
        ↓
酶切鉴定
        ↓
测序验证
        ↓
正确克隆
```

### 9.2 Gibson 组装流程

```
设计引物 (添加 20-40 bp 重叠序列)
        ↓
PCR 扩增各片段
        ↓
凝胶回收纯化
        ↓
 Gibson 组装反应
        ↓
直接转化或 Gibson 产物纯化
        ↓
转化感受态细胞
        ↓
筛选鉴定
        ↓
测序验证
```

## 10. 常见问题与解决方案

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 无克隆生长 | 连接失败、转化效率低 | 检查连接反应、提高转化效率 |
| 假阳性高 | 载体自连 | 去磷酸化处理、优化比例 |
| 片段插入错误 | 酶切不完全 | 优化酶切条件 |
| 测序失败 | 质粒质量差 | 重新提取质粒 |
| 扩增片段不对 | 引物问题 | 重新设计引物 |

> :::tip 学习提示
> 基因克隆是基因工程的基础技术。掌握各种载体的特点、克隆策略的选择、以及筛选鉴定方法，是进行任何基因工程操作的必备技能。建议通过实际操作加深理解。
> :::

> :::warning 注意
> 实验中务必设置阳性对照和阴性对照，以便及时发现和解决问题。同时注意无菌操作，避免交叉污染。
> :::
