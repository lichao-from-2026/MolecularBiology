# 三、PCR 技术

## 1. 定义与核心概念

### 1.1 PCR 的定义

**PCR (Polymerase Chain Reaction)**，即聚合酶链式反应，是一种在体外模拟 DNA 天然复制过程的核酸扩增技术，能够在短时间内将特定的 DNA 片段进行指数级扩增。

**发明历史**：

- 1983 年：Kary Mullis 发明 PCR
- 1985 年：首次公开发表
- 1993 年：Kary Mullis 因 PCR 获诺贝尔化学奖

### 1.2 核心概念

| 术语 | 英文 | 定义 |
|------|------|------|
| 变性 | Denaturation | DNA 双链受热解开为单链 |
| 退火 | Annealing | 引物与模板 DNA 结合 |
| 延伸 | Extension/Elongation | DNA 聚合酶合成新链 |
| 扩增子 | Amplicon | PCR 扩增的产物片段 |
| Ct 值 | Cycle Threshold | qPCR 中荧光达到阈值的循环数 |
| Tm 值 | Melting Temperature | DNA 双链解链温度 |

## 2. PCR 反应原理

### 2.1 PCR 反应成分

| 成分 | 作用 | 终浓度 |
|------|------|--------|
| 模板 DNA | 提供扩增序列 | 1-100 ng |
| 正向引物 | 决定扩增起始点 | 0.1-1 μM |
| 反向引物 | 决定扩增终止点 | 0.1-1 μM |
| dNTPs | 合成原料 | 0.2-0.5 mM |
| DNA 聚合酶 | 催化 DNA 合成 | 1-5 U |
| 缓冲液 | 提供反应环境 | 1× |
| Mg²⁺ | 酶活性辅因子 | 1.5-2.5 mM |

### 2.2 PCR 反应循环

**标准三步法 PCR**：

| 步骤 | 温度 | 时间 | 作用 |
|------|------|------|------|
| 初始变性 | 94-98°C | 2-5 分钟 | DNA 完全变性 |
| 变性 | 94-98°C | 15-30 秒 | 双链解开 |
| 退火 | 50-65°C | 30-60 秒 | 引物结合模板 |
| 延伸 | 72°C | 1 分钟/kb | 合成新链 |
| 最终延伸 | 72°C | 5-10 分钟 | 确保完全延伸 |
| 保存 | 4°C | ∞ | 临时保存 |

**两步法 PCR**：

- 退火和延伸合并为一个步骤
- 适用于引物 Tm 值较高或简并引物

### 2.3 指数扩增原理

**扩增曲线**：

| 循环数 | DNA 分子数 | 说明 |
|--------|------------|------|
| 0 | 2 | 起始双链 DNA |
| 1 | 4 | 2 倍 |
| 5 | 32 | 2⁵ |
| 10 | 1,024 | 约 10³ |
| 20 | 1,048,576 | 约 10⁶ |
| 25 | 33,554,432 | 约 3×10⁷ |
| 30 | 1,073,741,824 | 约 10⁹ |
| 35 | 34,359,738,368 | 约 3×10¹⁰ |

**实际扩增效率影响因素**：

- 酶活性
- 引物特异性
- 模板质量
- 反应条件

## 3. DNA 聚合酶

### 3.1 常用 PCR 酶

| 酶 | 来源 | 保真度 | 特点 |
|------|------|--------|------|
| Taq | *Thermus aquaticus* | 中等 | 无 3'-5' 外切酶 |
| Pfu | *Pyrococcus furiosus* | 高 | 有 3'-5' 外切酶 |
| Phusion | 工程改造 | 极高 | 高保真，高速度 |
| Q5 | 工程改造 | 极高 | 低错配率 |
| Vent | *Thermococcus litoralis* | 高 | 耐高温，长片段 |

### 3.2 Hot Start 酶

**原理**：室温下抗体或配体抑制酶活性，PCR 初始高温步骤使抑制剂失活

**优点**：

- 减少非特异性扩增
- 减少引物二聚体
- 提高扩增灵敏度

### 3.3 酶的选择指南

| 应用 | 推荐酶 | 原因 |
|------|--------|------|
| 常规 PCR | Taq | 经济实用 |
| 高保真克隆 | Pfu/Phusion/Q5 | 需要高保真度 |
| 长片段 PCR (>5 kb) | Vent/PrimeSTAR | 扩增能力强 |
| qPCR | Hot Start Taq | 减少非特异性 |

## 4. 引物设计

### 4.1 引物设计原则

| 参数 | 推荐范围 | 说明 |
|------|----------|------|
| 长度 | 18-25 bp | 过短降低特异性 |
| GC 含量 | 40-60% | 影响退火温度 |
| Tm 值 | 55-65°C | 两引物 Tm 值差异 < 5°C |
| 3' 端 | G/C 结尾 | 有利于延伸 |
| 连续相同碱基 | ≤ 4 个 | 避免聚 G 或聚 C |

### 4.2 引物设计检查

- **特异性检查**：BLAST 搜索确认唯一性
- **内部二级结构**：避免发夹结构
- **引物二聚体**：检查两引物间互补
- **跨外显子设计**：RNA 检测时避免基因组污染

### 4.3 常用引物设计工具

| 工具 | 来源 | 特点 |
|------|------|------|
| Primer3 | 公开 | 经典设计工具 |
| NCBI Primer-BLAST | 在线 | 含特异性检查 |
| Benchling | 在线 | 综合设计平台 |
| SnapGene | 商业 | 可视化设计 |

## 5. PCR 反应条件优化

### 5.1 温度优化

**变性温度**：

- 标准：94-98°C
- GC 含量高时可提高温度或延长时间

**退火温度**：

- 计算公式：Tm = 4×(G+C) + 2×(A+T)
- 原则：低于 Tm 值 3-5°C
- 优化方法：梯度 PCR 确定最佳温度

**延伸温度**：

- 标准：72°C（Taq 最适温度）
- 时间：1 分钟/kb 扩增片段

### 5.2 特殊 PCR 技术

**Touchdown PCR**：

- 初始退火温度高于预期
- 每个循环降低 1°C
- 直至达到合适温度
- 有助于提高特异性

**Nested PCR**：

- 两对引物先后扩增
- 第一对扩增初步产物
- 第二对在第一对产物内部扩增
- 大幅提高特异性

**Multiplex PCR**：

- 同时使用多对引物
- 扩增多个目标片段
- 需要优化各引物浓度

## 6. qPCR 技术

### 6.1 qPCR 定义

**qPCR (Quantitative Real-time PCR)** 是指在 PCR 反应体系中加入荧光染料或探针，通过实时监测荧光信号强度来定量分析核酸含量的技术。

**与普通 PCR 区别**：

| 特性 | 普通 PCR | qPCR |
|------|----------|------|
| 检测时间 | 反应结束后 | **实时监测每轮循环** |
| 定量能力 | 半定量 | **精确定量** |
| 产物检测 | 凝胶电泳 | 荧光信号 |
| 污染风险 | 高 | 低（封闭体系） |

### 6.2 荧光检测方法

**SYBR Green I 法**：

| 特点 | 说明 |
|------|------|
| 原理 | 荧光染料与双链 DNA 小沟结合 |
| 优点 | 操作简单、成本低、通用性强 |
| 缺点 | 非特异性结合（与所有 dsDNA 结合） |
| 验证 | 需要熔解曲线确认特异性 |

**TaqMan 探针法**：

| 特点 | 说明 |
|------|------|
| 原理 | 探针 5' 端荧光基团 + 3' 端淬灭基团 |
| 优点 | 特异性高、可多重 PCR |
| 缺点 | 成本高、探针设计复杂 |
| 检测 | 探针切割后产生荧光 |

**Molecular Beacon 法**：

- 发夹结构探针
- 与模板结合后发夹打开
- 荧光基团与淬灭基团分离

### 6.3 qPCR 反应体系

| 成分 | 作用 | 浓度 |
|------|------|------|
| 模板 DNA/cDNA | 扩增模板 | 1-100 ng |
| 正向引物 | 结合目标序列 | 0.1-0.5 μM |
| 反向引物 | 结合目标序列 | 0.1-0.5 μM |
| 荧光染料/探针 | 检测产物 | 按说明书 |
| DNA 聚合酶 | 催化合成 | 1-2 U |
| dNTPs | 合成原料 | 0.2-0.5 mM |
| 缓冲液 | 反应环境 | 1× |

### 6.4 定量分析方法

**相对定量 (ΔΔCt 法)**：

```
1. 计算 ΔCt = Ct(目标基因) - Ct(内参基因)
2. 计算 ΔΔCt = ΔCt(样品) - ΔCt(对照)
3. 相对表达 = 2^(-ΔΔCt)
```

**绝对定量 (标准曲线法)**：

1. 制备已知浓度的标准品
2. 梯度稀释（5-6 个浓度点）
3. 绘制标准曲线
4. 根据 Ct 值计算样品浓度

### 6.5 数据质量控制

| 指标 | 合格标准 | 说明 |
|------|----------|------|
| 熔解曲线 | 单峰 | 特异性扩增 |
| 扩增效率 | 90-110% | 标准曲线斜率 -3.1~-3.6 |
| R² 值 | > 0.99 | 曲线拟合度 |
| 重复性 | Ct 差异 < 0.5 | 同一样品重复间 |

## 7. PCR 产物分析

### 7.1 凝胶电泳

**琼脂糖凝胶浓度**：

| DNA 片段大小 | 凝胶浓度 |
|--------------|----------|
| 100 bp - 1 kb | 1.5-2% |
| 1 - 5 kb | 1-1.2% |
| 5 - 10 kb | 0.6-0.8% |
| > 10 kb | 0.4-0.5% |

**电泳条件**：

- 电压：100-120 V
- 时间：根据片段大小调整
- 染色：EB 或 SYBR Green

### 7.2 产物纯化

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| 柱纯化 | 硅胶柱吸附 | 快速、回收率高 | 成本较高 |
| 割胶回收 | 从凝胶中回收 | 可分离目标片段 | 回收率较低 |
| 磁珠纯化 | 磁珠结合 DNA | 自动化、高通量 | 需要设备 |

## 8. PCR 技术应用

### 8.1 基因克隆

**目的基因获取**：

- 从基因组 DNA 扩增
- 从 cDNA 扩增
- 从文库中筛选

**基因改造**：

- 定点突变
- 基因融合
- 序列修饰

### 8.2 医学诊断

**病原体检测**：

| 病原体 | 检测方法 | 临床应用 |
|--------|----------|----------|
| 病毒 | RT-PCR/qPCR | HIV、HBV、HCV、SARS-CoV-2 |
| 细菌 | PCR | 结核杆菌、幽门螺杆菌 |
| 真菌 | PCR | 念珠菌、曲霉 |
| 寄生虫 | PCR | 疟原虫 |

**遗传病诊断**：

- 单基因遗传病：囊性纤维化、地中海贫血
- 染色体异常：唐氏综合征
- 肿瘤：基因突变检测

### 8.3 法医学

**DNA 指纹分析**：

- STR 分析：个人识别
- SNP 分析：种群鉴定

**亲子鉴定**：

- 常染色体 STR
- Y 染色体 STR
- 线粒体 DNA 分析

### 8.4 基因表达分析

| 技术 | 应用 |
|------|------|
| RT-PCR | 半定量基因表达检测 |
| qPCR | 精确定量基因表达 |
|芯片定量 PCR | 多基因同时检测 |

### 8.5 农业应用

- 转基因检测
- 品种鉴定
- 疾病检测
- 亲子关系鉴定

### 8.6 环境监测

- 微生物检测
- 生物多样性分析
- 食品检测

## 9. 常见问题与解决方案

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 无扩增产物 | 模板不足、引物问题、酶失活 | 增加模板、重新设计引物、更换酶 |
| 非特异性扩增 | 退火温度过低 | 提高退火温度 |
| 引物二聚体 | 引物互补、退火温度低 | 重新设计引物、提高退火温度 |
| 产物弥散 | 模板降解、酶量过多 | 使用新鲜模板、减少酶量 |
| qPCR 无 Ct | 模板量不足、扩增失败 | 增加模板量、检查引物 |
| 熔解曲线多峰 | 非特异性扩增 | 提高退火温度、重新设计引物 |

## 10. 数字 PCR (dPCR)

### 10.1 dPCR 原理

**数字 PCR (Digital PCR)** 是一种绝对定量的 PCR 技术，将样品分散到大量独立反应单元中，通过终点检测计算阳性反应单元的比例来计算初始模板浓度。

**技术特点**：

| 特点 | 说明 |
|------|------|
| 绝对定量 | 无需标准曲线 |
| 高灵敏度 | 可检测极低拷贝数 |
| 高耐受性 | 对抑制剂容忍度高 |
| 精确度高 | 可检测微小差异 |

### 10.2 dPCR vs qPCR

| 特性 | qPCR | dPCR |
|------|------|------|
| 定量方式 | 相对定量 | **绝对定量** |
| 标准曲线 | 需要 | 不需要 |
| 灵敏度 | 高 | **更高** |
| 耐受性 | 中等 | **高** |
| 成本 | 较低 | 较高 |
| 通量 | 高 | 中等 |

### 10.3 dPCR 应用

- 稀有突变检测
- 拷贝数变异 (CNV) 分析
- 病原体绝对定量
- 基因表达绝对定量
- 液体活检

> :::tip 学习提示
> PCR 技术从发明至今不断发展和完善。理解 PCR 的基本原理是掌握所有衍生技术（qPCR、dPCR、RT-PCR 等）的基础。建议系统学习反应参数的优化，理解引物设计和反应条件对结果的影响。
> :::

> :::warning 注意
> PCR 实验中最重要的是避免污染。使用专用耗材、操作分区设置、设置阴性对照是防止假阳性结果的关键措施。
> :::
