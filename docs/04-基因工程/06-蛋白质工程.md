# 六、蛋白质工程

## 1. 定义与核心概念

### 1.1 蛋白质工程的定义

**蛋白质工程 (Protein Engineering)** 是指在基因水平上对蛋白质的氨基酸序列进行改造，或通过重组 DNA 技术大量表达改造后的蛋白质，以获得具有特定功能或改进性能的蛋白质的学科。

### 1.2 核心概念

| 术语 | 英文 | 定义 |
|------|------|------|
| 定向进化 | Directed Evolution | 模拟自然选择人为加速蛋白质进化的方法 |
| 理性设计 | Rational Design | 基于蛋白质结构信息设计改造策略 |
| 融合蛋白 | Fusion Protein | 两个或多个蛋白连接而成的重组蛋白 |
| 蛋白质折叠 | Protein Folding | 氨基酸序列形成特定三维结构的过程 |
| 酶活性 | Enzyme Activity | 蛋白质催化化学反应的能力 |

## 2. 蛋白质结构基础

### 2.1 蛋白质结构层次

| 层次 | 定义 | 决定因素 |
|------|------|----------|
| 一级结构 | 氨基酸序列 | 基因序列 |
| 二级结构 | α-螺旋、β-折叠 | 氢键 |
| 三级结构 |  polypeptide 折叠 | 疏水作用、离子键 |
| 四级结构 | 多亚基组合 | 亚基相互作用 |

### 2.2 蛋白质功能的关键因素

- **活性位点结构**：酶的催化中心
- **底物结合口袋**：特异性识别底物
- **构象变化**：蛋白质发挥功能时的动态变化
- **翻译后修饰**：磷酸化、糖基化等

## 3. 蛋白质改造策略

### 3.1 定向进化

**定向进化 (Directed Evolution)** 是模拟自然选择过程，在实验室中对蛋白质基因进行随机突变或重组，然后通过筛选获得性能改进的蛋白质突变体。

**基本流程**：

```
1. 创建蛋白质突变库
   ↓
2. 表达突变蛋白
   ↓
3. 筛选功能改进的突变体
   ↓
4. 分离突变基因
   ↓
5. 迭代循环
```

**突变库创建方法**：

| 方法 | 原理 | 特点 |
|------|------|------|
| 易错 PCR | 使用低保真度 DNA 聚合酶 | 随机引入点突变 |
| DNA 改组 | 片段随机重组 | 交换有益突变 |
| 饱和突变 | 特定位点所有氨基酸替换 | 系统扫描 |
| 随机插入/缺失 | 随机插入或删除片段 | 破坏/改变结构域 |

**易错 PCR**：

- 使用低保真度聚合酶（如 Taq）
- 调整 Mg²⁺、Mn²⁺ 浓度
- 控制突变频率（1-2% 氨基酸变化）

**DNA 改组 (DNA Shuffling)**：

1. 用 DNase I 随机切割目的基因
2. 无引物 PCR 重新组装
3. 产物片段相互配对重组
4. 筛选改进的突变体

**饱和突变 (Site-Saturation Mutagenesis)**：

- 针对特定位点
- 使用简并引物
- 可引入所有 19 种氨基酸替换

### 3.2 理性设计

**理性设计 (Rational Design)** 是基于蛋白质三维结构信息，计算预测氨基酸替换对蛋白质功能的影响，从而设计改造策略。

**前提条件**：

- 蛋白质三维结构已知
- 结构与功能关系清楚
- 活性位点等信息明确

**设计策略**：

| 策略 | 说明 |
|------|------|
| 活性位点改造 | 改变底物特异性 |
| 二硫键引入 | 增加蛋白质稳定性 |
| 糖基化位点移除 | 避免不需要的修饰 |
| 表面电荷改造 | 改变蛋白质稳定性或相互作用 |

**计算工具**：

| 工具 | 用途 |
|------|------|
| PyMOL | 蛋白质结构可视化 |
| Rosetta | 蛋白质设计 |
| AutoDock | 分子对接 |
| MD simulation | 分子动力学模拟 |

### 3.3 定向进化 vs 理性设计

| 特性 | 定向进化 | 理性设计 |
|------|----------|----------|
| 结构信息需求 | 不需要 | 需要 |
| 突变范围 | 随机，大范围 | 定点，精确 |
| 适用场景 | 结构未知 | 结构已知 |
| 筛选规模 | 需要高通量筛选 | 少量验证 |
| 成功率 | 较高 | 取决于结构信息质量 |

## 4. 融合蛋白技术

### 4.1 融合蛋白定义

**融合蛋白 (Fusion Protein)** 是通过基因工程手段将两个或多个蛋白编码区连接在一起，表达产生的具有新功能的重组蛋白。

### 4.2 常用标签蛋白

**纯化标签**：

| 标签 | 大小 | 特点 | 纯化方法 |
|------|------|------|----------|
| His-tag | 6-10 个氨基酸 | 小，不影响功能 | Ni-NTA 亲和层析 |
| GST | 26 kDa | 促进溶解 | 谷胱甘肽亲和层析 |
| MBP | 40 kDa | 促进折叠 | 淀粉酶亲和层析 |
| FLAG | 8 个氨基酸 | 亲水性 | 抗 FLAG 抗体 |

**检测标签**：

| 标签 | 功能 |
|------|------|
| GFP | 绿色荧光，可视化 |
| RFP | 红色荧光 |
| HA | 免疫检测 |
| Myc | 免疫检测 |
| V5 | 免疫检测 |

**功能标签**：

| 标签 | 功能 |
|------|------|
| SUMO | 促进溶解，定位 |
| Thioredoxin | 促进溶解 |
| NusA | 促进溶解 |

### 4.3 融合蛋白构建原则

1. **阅读框一致**：确保正确的氨基酸序列
2. **连接肽设计**：防止空间干扰
   - 柔性连接肽：GGGGS
   - 刚性连接肽：EAAAK
3. **表达系统选择**：原核或真核表达
4. **标签位置**：N 端或 C 端

### 4.4 常用连接肽

| 连接肽 | 序列 | 特点 |
|--------|------|------|
| 柔性连接肽 | (GGGGS)ₙ | 灵活，间隔效果好 |
| 刚性连接肽 | (EAAAK)ₙ | 固定距离 |
| 酶切位点连接 | 可被蛋白酶切割 | 便于切除标签 |

## 5. 蛋白质修饰技术

### 5.1 糖基化工程

**糖基化 (Glycosylation)** 是最常见的翻译后修饰之一。

**N-糖基化**：

- 共识序列：Asn-X-Ser/Thr
- 发生在内质网和高尔基体
- 影响蛋白质稳定性、免疫原性

**O-糖基化**：

- 发生在 Ser/Thr 残基
- 主要在粘蛋白中

**糖基化改造**：

- 去除糖基化位点
- 人源化糖基化（真核表达）
- 定点突变改变糖链结构

### 5.2 磷酸化工程

**磷酸化 (Phosphorylation)** 是重要的信号调控机制。

- 丝氨酸/苏氨酸磷酸化
- 酪氨酸磷酸化
- 通过定点突变模拟磷酸化状态
- 用于研究蛋白质功能

### 5.3 二硫键工程

**二硫键 (Disulfide Bond)** 稳定蛋白质结构。

**策略**：

- 引入新的二硫键增加稳定性
- 突变半胱氨酸破坏不需要的二硫键
- 设计新的 Cys 残基对

## 6. 蛋白质表达系统

### 6.1 原核表达系统

**大肠杆菌表达系统**：

| 优点 | 缺点 |
|------|------|
| 表达量高 | 无翻译后修饰 |
| 操作简单 | 可能形成包涵体 |
| 成本低 | 密码子偏好性 |

**优化策略**：

- 密码子优化
- 伴侣蛋白共表达
- 诱导条件优化
- 溶解性标签融合

### 6.2 真核表达系统

**酵母表达系统**：

| 系统 | 特点 | 应用 |
|------|------|------|
| 酿酒酵母 | 简单 | 基础研究 |
| 毕赤酵母 | 高表达，分泌表达 | 工业蛋白生产 |

**昆虫细胞表达系统**：

- 杆状病毒表达系统
- 高表达，正确折叠
- 适用于大分子蛋白

**哺乳动物细胞表达系统**：

| 系统 | 特点 | 应用 |
|------|------|------|
| HEK293 | 高转染效率 | 科研 |
| CHO | 工业生产 | 抗体、重组蛋白 |

### 6.3 无细胞表达系统

- 体外翻译系统
- 不受细胞毒性影响
- 快速表达
- 便于标记和修饰

## 7. 蛋白质稳定性工程

### 7.1 增加蛋白质稳定性的策略

| 策略 | 方法 |
|------|------|
| 二硫键引入 | 突变引入 Cys，形成二硫键 |
| 疏水核心优化 | 增加疏水相互作用 |
| 表面电荷优化 | 改善热稳定性 |
| 糖基化 | 增加水溶性，防止聚集 |
| 截短/删除 | 去除不稳定的区域 |

### 7.2 蛋白质聚集问题

**包涵体形成**：

- 不溶性蛋白质聚集体
- 需要变性复性处理

**解决方案**：

- 降低表达温度
- 减少诱导时间
- 使用融合标签
- 优化表达条件

## 8. 酶工程

### 8.1 酶活性改造

| 改造目标 | 策略 |
|----------|------|
| 提高活性 | 优化活性位点，增加催化效率 |
| 改变底物特异性 | 改造底物结合口袋 |
| 提高稳定性 | 定向进化或理性设计 |
| 改变最适温度/pH | 进化或设计 |

### 8.2 工业酶改造

| 酶类 | 应用 | 改造目标 |
|------|------|----------|
| 蛋白酶 | 洗涤剂 | 碱性稳定性 |
| 淀粉酶 | 食品加工 | 热稳定性 |
| 纤维素酶 | 纺织 | 活性提高 |
| 脂肪酶 | 制药 | 立体选择性 |

### 8.3 定向进化应用案例

**脂肪酶定向进化**：

- 提高有机溶剂耐受性
- 改变立体选择性
- 提高热稳定性

**抗体工程**：

- 提高亲和力
- 优化Fc功能
- 人源化

## 10. 实验技术与方法

### 10.1 定点突变

**PCR 方法**：

1. 设计突变引物（含突变位点）
2. 延伸 PCR
3. DpnI 消化模板
4. 转化鉴定

### 10.2 蛋白质纯化策略

| 方法 | 原理 | 应用 |
|------|------|------|
| Ni-NTA 亲和层析 | His-tag 与 Ni²⁺ 结合 | His 标签蛋白 |
| 谷胱甘肽亲和层析 | GST 与 谷胱甘肽结合 | GST 标签蛋白 |
| 淀粉酶亲和层析 | MBP 与 淀粉酶结合 | MBP 标签蛋白 |
| 离子交换层析 | 电荷差异 | 初步纯化 |
| 凝胶过滤层析 | 分子大小 | 进一步纯化/缓冲液置换 |

### 10.3 蛋白质分析

| 方法 | 检测内容 |
|------|----------|
| SDS-PAGE | 分子量、纯度 |
| Western blot | 特定蛋白检测 |
| 质谱 | 分子量、修饰分析 |
| 圆二色谱 | 二级结构 |
| DSC | 热稳定性 |
