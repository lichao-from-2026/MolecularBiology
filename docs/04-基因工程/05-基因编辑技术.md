# 五、基因编辑技术

## 1. 定义与核心概念

### 1.1 基因编辑的定义

**基因编辑 (Gene Editing)** 是一种能够在基因组特定位点进行精确修饰的技术，包括 DNA 片段的插入、删除、替换等操作。

### 1.2 核心概念

| 术语 | 英文 | 定义 |
|------|------|------|
| 核酸酶 | Nuclease | 切割 DNA 的酶 |
| 双链断裂 | DSB | DNA 两条链同时断裂 |
| NHEJ | Non-Homologous End Joining | 非同源末端连接，细胞修复 DSB 的机制 |
| HDR | Homology-Directed Repair | 同源定向修复，利用模板精确修复 |
| 脱靶效应 | Off-target Effect | 核酸酶切割非目标位点的现象 |
| sgRNA | Single Guide RNA | CRISPR-Cas9 系统中的引导 RNA |

### 1.3 DNA 修复机制

**非同源末端连接 (NHEJ)**：

- 细胞修复 DSB 的主要机制
- 容易产生插入/缺失突变（indels）
- 适用于基因敲除
- 无需同源模板

**同源定向修复 (HDR)**：

- 需要同源模板提供修复信息
- 可实现精确的基因敲入或点突变
- 效率低于 NHEJ
- 主要在细胞分裂期活跃

## 2. ZFN 技术

### 2.1 ZFN 结构

**ZFN (Zinc Finger Nuclease)** 由锌指蛋白 (ZFP) 和 FokI 核酸酶组成。

**锌指蛋白结构**：

- 每个锌指识别 3 个碱基对
- 约 30 个氨基酸组成
- 通过锌离子稳定结构
- 多个锌指串联可识别更长序列

**FokI 核酸酶**：

- 需要二聚化才具有切割活性
- 切割 spacer 区域 DNA 双链
- 产生粘性末端

### 2.2 ZFN 工作机制

1. 锌指蛋白识别 DNA 目标序列（9-18 bp，每侧）
2. 两个 FokI 结构域形成二聚体
3. FokI 二聚体切割 spacer 区域
4. 细胞通过 NHEJ 或 HDR 修复

### 2.3 ZFN 构建

**模块组装法**：

- 选择已知锌指模块
- 按目标序列组装锌指
- 融合 FokI 核酸酶
- 测试切割活性

**锌指库**：

- Sangamo 商业化锌指库
- Wright Lab 开源库

### 2.4 ZFN 优缺点

| 优点 | 缺点 |
|------|------|
| 特异性高 | 设计复杂 |
| 低脱靶效应 | 构建困难 |
| 已有临床验证 | 成本高 |
| 稳定性好 | 耗时 |

## 3. TALEN 技术

### 3.1 TALEN 结构

**TALEN (Transcription Activator-Like Effector Nuclease)** 由 TALE 蛋白和 FokI 核酸酶组成。

**TALE 蛋白结构**：

- 来自植物病原菌 (*Xanthomonas*)
- 重复单元：34 个氨基酸
- 每个重复单元识别一个 DNA 碱基
- RVD（第 12-13 位氨基酸）决定识别碱基

**RVD 与碱基对应**：

| RVD | 识别的碱基 |
|-----|-----------|
| NI | A |
| HD | C |
| NN | G 或 A |
| NG | T |
| NS | 任意碱基 |

### 3.2 TALEN 工作机制

1. TALE 蛋白识别 DNA 目标序列（15-30 bp，每侧）
2. 两个 FokI 结构域形成二聚体
3. FokI 二聚体切割 spacer 区域
4. 细胞通过 NHEJ 或 HDR 修复

### 3.3 TALEN 构建

**Golden Gate 组装法**：

1. 准备含不同 RVD 的模块
2. 使用 Type IIS 限制性内切酶切割
3. 连接组装到载体中
4. 逐步组装完整 TALE 重复序列

**设计原则**：

| 参数 | 推荐值 |
|------|--------|
| 识别序列长度 | 15-30 bp |
| spacer 长度 | 12-20 bp |
| GC 含量 | 40-60% |

### 3.4 TALEN 优缺点

| 优点 | 缺点 |
|------|------|
| 高特异性 | 构建复杂 |
| 灵活性高 | 蛋白尺寸大，递送困难 |
| 效率较高 | 成本较高 |
| 脱靶率低 | 构建耗时 |

## 4. CRISPR-Cas9 技术

### 4.1 CRISPR-Cas9 概述

**CRISPR-Cas9 (Clustered Regularly Interspaced Short Palindromic Repeats-CRISPR associated protein 9)** 是一种革命性的基因编辑技术，来源于细菌的适应性免疫系统。

**核心组件**：

| 组件 | 功能 |
|------|------|
| Cas9 蛋白 | 核酸酶，切割 DNA |
| sgRNA | 引导 Cas9 到目标位点 |
| PAM 序列 | Cas9 识别的必要序列 (NGG) |

### 4.2 sgRNA 结构

```
5' —— spacer (18-20 nt) —— 3'
      ↓
与目标 DNA 互补
      ↓
tracrRNA 序列 (与 Cas9 结合)
```

**PAM 序列**：

- SpCas9 PAM：5'-NGG-3'
- 位于目标序列下游
- 是 Cas9 识别的必要条件

### 4.3 CRISPR-Cas9 工作机制

1. **复合物形成**：Cas9 与 sgRNA 结合
2. **目标搜索**：复合物搜索 PAM 序列
3. **DNA 切割**：Cas9 在 PAM 上游 3 bp 处切割产生 DSB
4. **DNA 修复**：NHEJ 或 HDR

### 4.4 sgRNA 设计

**设计原则**：

| 参数 | 要求 |
|------|------|
| 长度 | 18-20 nt |
| PAM | 5'-NGG-3' |
| GC 含量 | 40-70% |
| 特异性 | 高，避免脱靶位点 |

**设计工具**：

| 工具 | 来源 |
|------|------|
| CRISPR Design | MIT |
| Benchling | 在线平台 |
| CRISPOR | 在线工具 |

### 4.5 CRISPR-Cas9 递送方法

| 方法 | 特点 | 应用 |
|------|------|------|
| 质粒转染 | 简单，瞬时表达 | 细胞系 |
| 病毒递送 | 效率高，稳定 | 难转染细胞 |
| RNP 递送 | 效率高，脱靶低 | **推荐方法** |
| mRNA 递送 | 无基因组整合 | 体内应用 |
| AAV 递送 | 安全性高 | 基因治疗 |

### 4.6 CRISPR-Cas9 应用

**基因敲除**：

- 原理：NHEJ 产生移码突变
- 应用：基因功能研究、疾病模型

**基因敲入**：

- 原理：HDR 插入外源基因
- 应用：基因标签、基因修复

**转录调控**：

| 系统 | Cas9 变体 | 功能 |
|------|-----------|------|
| CRISPRi | dCas9 | 转录抑制 |
| CRISPRa | dCas9-激活域 | 转录激活 |

**基因筛选**：

- 全基因组敲除文库筛选
- 发现基因功能

### 4.7 CRISPR-Cas9 衍生技术

**碱基编辑器 (Base Editors)**：

| 类型 | 编辑类型 |
|------|----------|
| CBEs | C→T (G→A) |
| ABEs | A→G (T→C) |

**Prime Editor**：

- 原理：Cas9 nickase + 逆转录酶 + pegRNA
- 特点：无须 DSB，无须 HDR 模板
- 可实现所有类型点突变

## 5. 三种基因编辑技术对比

### 5.1 技术原理对比

| 特性 | ZFN | TALEN | CRISPR-Cas9 |
|------|-----|-------|------------|
| 识别机制 | 蛋白质-DNA | 蛋白质-DNA | **RNA-DNA** |
| 识别长度 | 9-18 bp | 15-30 bp | 20 bp + PAM |
| 切割方式 | FokI 二聚体 | FokI 二聚体 | Cas9 双结构域 |
| 末端类型 | 粘性末端 | 粘性末端 | 平末端 |

### 5.2 构建与使用对比

| 特性 | ZFN | TALEN | CRISPR-Cas9 |
|------|-----|-------|------------|
| 设计难度 | **高** | 中 | **低** |
| 构建时间 | 2-4 周 | 1-2 周 | **几天** |
| 构建成本 | **高** | 中 | **低** |
| 脱靶效应 | 低 | 低 | 中等 |

### 5.3 选择建议

| 场景 | 推荐技术 |
|------|----------|
| 新手入门 | **CRISPR-Cas9** |
| 高特异性需求 | TALEN / ZFN |
| 多基因编辑 | **CRISPR-Cas9** |
| 碱基编辑 | **CRISPR-Cas9 衍生** |

## 6. 基因编辑的脱靶效应

### 6.1 脱靶检测方法

| 方法 | 原理 | 灵敏度 |
|------|------|--------|
| GUIDE-seq | 标签 DSB 位点 | 高 |
| CIRCLE-seq | 捕获 DSB 位点 | 高 |
| WGS | 全基因组测序 | 全局 |
| DISCOVER-seq | MRE11 招募 | 中 |

### 6.2 减少脱靶的方法

| 方法 | 说明 |
|------|------|
| 优化 sgRNA | 选择高特异性 sgRNA |
| 高保真 Cas9 | HiFi Cas9 突变体 |
| Cas9 nickase | 单链切割 |
| RNP 递送 | 短时表达 |

## 7. 基因编辑的安全性

### 7.1 安全性问题

- **脱靶效应**：非预期基因突变
- **免疫原性**：蛋白可能引起免疫反应
- **染色体异常**：易位或缺失
- **伦理问题**：生殖细胞编辑争议

### 7.2 安全评估

| 评估内容 | 方法 |
|----------|------|
| 脱靶检测 | WGS、脱靶检测工具 |
| 免疫反应 | ELISA、细胞因子检测 |
| 染色体异常 | Karyotyping、FISH |

### 7.3 降低风险策略

- 使用高保真 Cas9
- sgRNA 优化选择
- RNP 递送减少表达时间
- 严格筛选检测

## 8. 基因编辑实验流程

### 8.1 CRISPR-Cas9 实验流程

```
1. sgRNA 设计
   ↓
2. 载体构建或 RNP 制备
   ↓
3. 递送
   ↓
4. 筛选
   ↓
5. 鉴定 (PCR、测序)
   ↓
6. 功能验证
```

### 8.2 HDR 敲入流程

```
1. 设计 sgRNA
   ↓
2. 制备 HDR 模板 (同源臂)
   ↓
3. 共转染
   ↓
4. 筛选
   ↓
5. 单克隆分离
   ↓
6. 鉴定验证
```

### 8.3 检测方法

| 方法 | 检测内容 |
|------|----------|
| T7E1 assay | 突变检测 |
| TIDE | 突变类型和频率 |
| PCR-RFLP | 特定突变 |
| 测序 | 金标准 |

## 9. 应用实例

### 9.1 基因敲除细胞系构建

1. 设计靶向 sgRNA
2. 构建 Cas9-sgRNA 或制备 RNP
3. 转染细胞
4. 筛选单克隆
5. PCR + 测序鉴定
6. WB 验证敲除
7. 功能实验

### 9.2 疾病治疗

| 疾病 | 靶基因/策略 | 技术 |
|------|-------------|------|
| 地中海贫血 | BCL11A | CRISPR |
| 镰刀型细胞贫血症 | HBG | CRISPR |
| 艾滋病 | CCR5 | ZFN |
| 遗传性失明 | CEP290 | CRISPR |

### 9.3 农业应用

- 抗虫作物
- 抗病作物
- 品质改良

> :::tip 学习提示
> CRISPR-Cas9 是目前最主流的基因编辑技术，建议从 Cas9 入手学习。在掌握基本原理和操作后，可以进一步学习碱基编辑器和 Prime Editor 等衍生技术。理解 DNA 修复机制对于设计基因编辑实验至关重要。
