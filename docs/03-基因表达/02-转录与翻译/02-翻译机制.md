# 翻译机制

## 1. 定义与核心概念

### 1.1 翻译 (Translation)

**定义**：翻译是指以 mRNA 为模板合成蛋白质的过程，是基因表达的第二步，将 mRNA 中的遗传信息转化为蛋白质的氨基酸序列。

**核心概念**：

- **核糖体**：蛋白质合成的场所
- **tRNA**：携带氨基酸并识别密码子
- **密码子**：mRNA 上编码氨基酸的三个碱基
- **起始因子**：参与翻译起始的蛋白质
- **延伸因子**：参与翻译延伸的蛋白质
- **释放因子**：参与翻译终止的蛋白质
- **氨基酰-tRNA 合成酶**：催化氨基酸与 tRNA 结合
- **肽酰转移酶**：催化肽键形成

## 2. 核糖体结构

### 2.1 原核生物核糖体

- **组成**：30S 小亚基 + 50S 大亚基 = 70S 核糖体
- **rRNA**：16S rRNA (小亚基)、23S rRNA 和 5S rRNA (大亚基)
- **蛋白质**：约 50 种蛋白质
- **沉降系数**：70S

### 2.2 真核生物核糖体

- **组成**：40S 小亚基 + 60S 大亚基 = 80S 核糖体
- **rRNA**：18S rRNA (小亚基)、28S rRNA、5.8S rRNA 和 5S rRNA (大亚基)
- **蛋白质**：约 80 种蛋白质
- **沉降系数**：80S

### 2.3 核糖体活性位点

- **A 位**：氨基酰-tRNA 结合位点
- **P 位**：肽酰-tRNA 结合位点
- **E 位**：退出位点（原核生物）
- **肽酰转移酶中心**：催化肽键形成，位于大亚基
- **mRNA 结合位点**：位于小亚基

### 2.4 核糖体的组装

- **原核生物**：在细胞质中组装
- **真核生物**：小亚基在细胞核中组装，大亚基在核仁中组装，在细胞质中完成最终组装

## 3. 遗传密码

### 3.1 密码子表

- **起始密码子**：AUG (编码甲硫氨酸)
- **终止密码子**：UAA、UAG、UGA
- **密码子数量**：64 个密码子，其中 61 个编码氨基酸
- **氨基酸数量**：20 种标准氨基酸

### 3.2 密码子特性

- **简并性**：多个密码子编码同一个氨基酸
  - 例如：亮氨酸有 6 个密码子
- **通用性**：几乎所有生物使用相同的密码子表
  - 例外：线粒体、某些原核生物有少量密码子使用差异
- **摆动性**：tRNA 反密码子与 mRNA 密码子的配对存在摆动
  - 主要发生在密码子的第三位
- **方向性**：从 5' 到 3' 阅读
- **连续性**：密码子之间没有间隔

### 3.3 密码子偏好性

- **定义**：不同生物对同义密码子的使用频率不同
- **影响因素**：tRNA 丰度、基因表达水平
- **应用**：基因工程中优化密码子以提高表达效率

## 4. tRNA 与氨基酰-tRNA 合成酶

### 4.1 tRNA 结构

- **三叶草结构**：由氨基酸臂、D 臂、反密码子臂、TψC 臂组成
- **反密码子**：与 mRNA 密码子互补
- **氨基酸接受臂**：3' 端 CCA 序列，结合氨基酸

### 4.2 氨基酰-tRNA 合成酶

- **功能**：催化氨基酸与 tRNA 结合
- **种类**：20 种，每种对应一种氨基酸
- **准确性**：通过双重筛选机制确保正确的氨基酸与 tRNA 结合
- **反应过程**：
  1. 氨基酸活化：与 ATP 反应形成氨基酰-AMP
  2. 转移：将氨基酸转移到 tRNA 的 3' 端

## 5. 翻译过程

### 5.1 起始

1. **原核生物起始**：
   - **起始因子**：IF1、IF2、IF3
   - **过程**：
     - IF3 与 30S 小亚基结合，防止与 50S 大亚基过早结合
     - mRNA 的 SD 序列与 16S rRNA 互补结合
     - IF2 携带 fMet-tRNAfMet 进入 P 位
     - IF1 结合到 A 位
     - IF3 释放，50S 大亚基结合，形成 70S 起始复合物
     - IF1 和 IF2 释放
2. **真核生物起始**：
   - **起始因子**：eIF1、eIF1A、eIF2、eIF3、eIF4 家族、eIF5、eIF5B
   - **过程**：
     - eIF4F 复合物（eIF4E、eIF4G、eIF4A）结合到 mRNA 的 5' 帽子
     - eIF3 与 40S 小亚基结合
     - eIF2 携带 Met-tRNAMet 与 GTP 形成三元复合物
     - 40S 小亚基与 mRNA 结合，开始扫描
     - 识别起始密码子 AUG（Kozak 序列：GCCRCCAUGG）
     - eIF5 促进 GTP 水解，释放起始因子
     - 60S 大亚基结合，形成 80S 起始复合物

### 5.2 延伸

1. **进位**：
   - 氨基酰-tRNA 进入 A 位
   - 原核生物：EF-Tu 携带氨基酰-tRNA 与 GTP 形成复合物
   - 真核生物：eEF1α 携带氨基酰-tRNA 与 GTP 形成复合物
   - GTP 水解，EF-Tu/eEF1α 释放
2. **肽键形成**：
   - 肽酰转移酶催化肽键形成
   - P 位上的氨基酸的羧基与 A 位上的氨基酸的氨基形成肽键
   - 反应在大亚基的肽酰转移酶中心进行
   - 肽酰-tRNA 从 P 位移到 A 位
3. **移位**：
   - 核糖体沿 mRNA 5'→3' 移动一个密码子
   - 原核生物：EF-G 与 GTP 结合，促进移位
   - 真核生物：eEF2 与 GTP 结合，促进移位
   - GTP 水解，EF-G/eEF2 释放
   - 肽酰-tRNA 从 A 位移动到 P 位
   - 空载 tRNA 从 P 位移动到 E 位并释放（原核生物）

### 5.3 终止

1. **识别终止密码子**：
   - 释放因子识别终止密码子
   - 原核生物：RF1 识别 UAA、UAG；RF2 识别 UAA、UGA；RF3 协助 RF1/RF2
   - 真核生物：eRF1 识别所有终止密码子；eRF3 协助 eRF1
2. **肽链释放**：
   - 释放因子诱导肽酰转移酶活性转变为水解酶活性
   - 肽链从 tRNA 上释放
   - GTP 水解提供能量
3. **核糖体解离**：
   - 核糖体亚基解离
   - mRNA 和 tRNA 释放
   - 原核生物：IF3 参与核糖体解离
   - 真核生物：eIF3 参与核糖体解离

## 6. 翻译后加工

### 6.1 蛋白质折叠

- **分子伴侣**：
  - 热休克蛋白（Hsp70、Hsp90）
  - 伴侣蛋白（GroEL/GroES）
  - 协助蛋白质正确折叠
- **内质网**：
  - 真核生物中蛋白质折叠的场所
  - 钙网蛋白、钙联蛋白等协助折叠
- **二硫键形成**：
  - 由二硫键异构酶催化
  - 稳定蛋白质结构
- **脯氨酸异构化**：
  - 由脯氨酰异构酶催化
  - 加速折叠过程

### 6.2 蛋白质修饰

| 修饰类型     | 酶         | 功能             |
| -------- | --------- | -------------- |
| 磷酸化      | 激酶        | 调控酶活性、信号传导     |
| 糖基化      | 糖基转移酶     | 蛋白质定位、稳定性、识别   |
| 甲基化      | 甲基转移酶     | 调控蛋白质功能、DNA 结合 |
| 乙酰化      | 乙酰转移酶     | 调控蛋白质活性、核定位    |
| 泛素化      | 泛素连接酶     | 标记蛋白质降解        |
| 脂酰化      | 脂酰转移酶     | 蛋白质膜定位         |
| 羟基化      | 羟化酶       | 胶原蛋白稳定性        |
| ADP-核糖基化 | ADP-核糖转移酶 | 调控酶活性          |

### 6.3 蛋白质定位

- **信号肽**：
  - 引导蛋白质到特定位置
  - 由信号识别颗粒（SRP）识别
- **核定位信号**：
  - 引导蛋白质进入细胞核
  - 由核孔复合物识别
- **线粒体靶向序列**：
  - 引导蛋白质进入线粒体
  - 由线粒体膜上的受体识别
- **叶绿体靶向序列**：
  - 引导蛋白质进入叶绿体
  - 由叶绿体膜上的受体识别
- \*\* peroxisome 靶向序列\*\*：
  - 引导蛋白质进入 peroxisome
  - 由 peroxisome 膜上的受体识别

### 6.4 蛋白质降解

- **蛋白酶体**：
  - 降解泛素化的蛋白质
  - 26S 蛋白酶体：由 19S 调节颗粒和 20S 核心颗粒组成
- **溶酶体**：
  - 降解细胞内蛋白质和细胞器
  - 酸性环境，含有多种水解酶
- **自噬**：
  - 降解细胞器和蛋白质聚集体
  - 形成自噬体，与溶酶体融合

## 7. 翻译调控

### 7.1 翻译起始调控

- **mRNA 结构**：
  - 5' UTR 长度和二级结构
  - 5' 帽子结构的完整性
  - 3' UTR 中的调控元件
- **起始密码子上下文**：
  - 原核生物：SD 序列与 16S rRNA 的互补性
  - 真核生物：Kozak 序列
- **起始因子活性**：
  - 磷酸化状态（如 eIF2α 磷酸化抑制翻译）
  - 起始因子的表达水平
- **mRNA 稳定性**：
  - 3' UTR 中的 AU 富含元件
  - miRNA 介导的 mRNA 降解
- **上游开放阅读框 (uORF)**：
  - 调控下游 ORF 的翻译
  - 如 ATF4 基因的翻译调控

### 7.2 翻译延伸调控

- **延伸因子活性**：
  - 磷酸化状态
  - 延伸因子的表达水平
- **tRNA 可用性**：
  - 氨基酸供应
  - tRNA 合成和修饰
- **核糖体停滞**：
  - mRNA 二级结构
  - 稀有密码子
  - 氨基酸饥饿
- **抗生素**：
  - 如氯霉素、四环素抑制延伸

### 7.3 翻译终止调控

- **释放因子活性**：
  - 调控终止效率
  - 释放因子的表达水平
- **通读**：
  - 抑制终止，产生更长的蛋白质
  - 如某些病毒的翻译调控
- **无义介导的 mRNA 降解 (NMD)**：
  - 识别提前终止密码子
  - 降解异常 mRNA

## 8. 翻译的应用

### 8.1 基因表达研究

- **Western blot**：检测特定蛋白质表达
- **ELISA**：定量蛋白质表达
- **免疫荧光**：检测蛋白质在细胞中的定位
- **蛋白质组学**：全蛋白质组分析
- **核糖体分析 (Ribosome profiling)**：研究翻译活性

### 8.2 基因工程

- **重组蛋白表达**：
  - 原核表达系统（大肠杆菌）
  - 真核表达系统（酵母、昆虫细胞、哺乳动物细胞）
  - 无细胞表达系统
- **融合蛋白**：
  - 标签蛋白（His、GST、Flag 等）便于纯化
  - 荧光蛋白（GFP 等）便于检测
- **蛋白质工程**：
  - 定点突变
  - 定向进化
  - 融合蛋白设计

### 8.3 药物研发

- **抗生素**：
  - 抑制细菌翻译（如四环素、氯霉素、红霉素）
  - 靶向原核核糖体的特定位点
- **蛋白质抑制剂**：
  - 调控特定蛋白质的功能
  - 如蛋白酶体抑制剂
- **mRNA 疫苗**：
  - 利用翻译表达抗原
  - 如 COVID-19 mRNA 疫苗
- **翻译调控药物**：
  - 调控特定基因的翻译
  - 如 eIF4E 抑制剂

### 8.4 合成生物学

- **人工核糖体**：
  - 设计改造核糖体
  - 扩展遗传密码
- **生物传感器**：
  - 通过翻译调控检测环境信号
- **细胞工厂**：
  - 优化蛋白质表达
  - 生产药用蛋白

## 9. 翻译过程的比较

### 9.1 原核生物 vs 真核生物翻译

| 特征          | 原核生物              | 真核生物                                    |
| ----------- | ----------------- | --------------------------------------- |
| **核糖体**     | 70S (30S + 50S)   | 80S (40S + 60S)                         |
| **起始密码子**   | AUG (编码甲酰甲硫氨酸)    | AUG (编码甲硫氨酸)                            |
| **起始因子**    | IF1、IF2、IF3       | eIF1、eIF1A、eIF2、eIF3、eIF4 家族、eIF5、eIF5B |
| **核糖体结合位点** | SD 序列             | 5' 帽子 + Kozak 序列                        |
| **翻译起始**    | 30S 小亚基直接结合到 mRNA | 40S 小亚基从 5' 帽子扫描到 AUG                   |
| **翻译速度**    | 较快，约 15-20 氨基酸/秒  | 较慢，约 2-5 氨基酸/秒                          |
| **翻译场所**    | 细胞质               | 细胞质（游离核糖体或粗面内质网）                        |
| **与转录的关系**  | 同时进行              | 先转录后翻译                                  |

### 9.2 病毒翻译特点

- **利用宿主核糖体**：所有病毒都利用宿主核糖体进行翻译
- **翻译策略多样**：
  - **帽子依赖翻译**：如大多数 DNA 病毒
  - **IRES 介导的翻译**：如脊髓灰质炎病毒、丙型肝炎病毒
  - **通读**：如呼肠孤病毒
  - **核糖体移码**：如 HIV、冠状病毒
  - **多聚蛋白切割**：如小 RNA 病毒
- **病毒编码的翻译调控因子**：
  - 如 IRES、病毒蛋白酶
- **特点**：
  - 适应宿主翻译环境
  - 部分病毒可抑制宿主翻译
  - 翻译策略与病毒生命周期相关

## 10. 翻译研究的前沿技术

- **核糖体分析**：研究翻译组
- **单分子翻译研究**：实时观察翻译过程

- **CRISPR-Cas9**：研究翻译调控因子的功能
- **人工智能**：预测蛋白质结构和翻译效率
