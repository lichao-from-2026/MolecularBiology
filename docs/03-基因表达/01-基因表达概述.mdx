# 一、基因表达概述

## 1. 定义与核心概念

### 1.1 基因表达 (Gene Expression)

**定义**：基因表达是指从基因到蛋白质的整个过程，包括转录（DNA→RNA）和翻译（RNA→蛋白质）两个主要步骤，以及相关的调控机制。

**核心概念**：
- **中心法则**：DNA → RNA → 蛋白质的信息流
- **基因**：编码蛋白质或功能RNA的DNA序列
- **转录**：以DNA为模板合成RNA的过程
- **翻译**：以mRNA为模板合成蛋白质的过程
- **基因表达调控**：控制基因表达的时间、空间和强度
- **翻译后修饰**：蛋白质合成后的化学修饰

## 2. 基因表达的基本流程

### 2.1 原核生物基因表达

```
DNA → 转录 → mRNA → 翻译 → 蛋白质
```

**特点**：
- **转录和翻译在同一空间进行**：无核膜分隔
- **转录和翻译同时进行**：边转录边翻译
- **基因表达调控主要在转录水平**：以操纵子为单位调控
- **mRNA 不需要加工**：直接作为翻译模板
- **无内含子**：基因结构简单

### 2.2 真核生物基因表达

```
DNA → 转录 → 前体mRNA → 转录后加工 → 成熟mRNA → 核输出 → 翻译 → 蛋白质 → 翻译后加工
```

**特点**：
- **转录在细胞核，翻译在细胞质**：核膜分隔
- **转录和翻译不同步**：先转录后翻译
- **存在复杂的转录后加工**：5'加帽、3'加尾、剪接
- **基因表达调控发生在多个水平**：DNA、转录、转录后、翻译、翻译后
- **存在内含子**：基因结构复杂

### 2.3 病毒基因表达

**DNA病毒**：
- 双链DNA病毒：如疱疹病毒，在细胞核内转录
- 单链DNA病毒：如 parvovirus，先合成双链DNA再转录

**RNA病毒**：
- 正链RNA病毒：如冠状病毒，直接作为mRNA翻译
- 负链RNA病毒：如流感病毒，先合成正链RNA再翻译
- 双链RNA病毒：如轮状病毒，通过RNA聚合酶转录
- 逆转录病毒：如HIV，通过逆转录酶合成DNA，整合到宿主基因组后转录

**特点**：
- 依赖宿主细胞的转录和翻译 machinery
- 基因表达策略多样
- 部分病毒具有自身的转录酶

## 3. 转录过程的比较

### 3.1 原核生物转录

- **RNA聚合酶**：单一RNA聚合酶（α₂ββ'σ）
- **启动子**：-10区（TATAAT）和-35区（TTGACA）
- **转录因子**：σ因子识别启动子
- **转录终止**：rho因子依赖或非依赖
- **转录产物**：多顺反子mRNA

### 3.2 真核生物转录

- **RNA聚合酶**：三种RNA聚合酶（I、II、III）
- **启动子**：TATA盒、CAAT盒、GC盒等
- **转录因子**：通用转录因子和特异性转录因子
- **转录终止**：poly(A)信号依赖
- **转录产物**：单顺反子mRNA

### 3.3 病毒转录

- **DNA病毒**：利用宿主RNA聚合酶或自身携带的RNA聚合酶
- **RNA病毒**：利用自身携带的RNA依赖RNA聚合酶
- **逆转录病毒**：利用逆转录酶合成DNA，再利用宿主RNA聚合酶转录

## 4. 翻译过程的比较

### 4.1 原核生物翻译

- **核糖体**：70S（30S + 50S）
- **起始密码子**：AUG（编码甲酰甲硫氨酸）
- **起始因子**：IF1、IF2、IF3
- **SD序列**：mRNA上的核糖体结合位点
- **翻译起始**：30S小亚基直接结合到mRNA的SD序列
- **翻译速度**：较快，约15-20氨基酸/秒

### 4.2 真核生物翻译

- **核糖体**：80S（40S + 60S）
- **起始密码子**：AUG（编码甲硫氨酸）
- **起始因子**：eIF1、eIF1A、eIF2、eIF3、eIF4家族、eIF5、eIF5B
- **Kozak序列**：起始密码子周围的保守序列
- **翻译起始**：40S小亚基从5'帽子开始扫描到AUG
- **翻译速度**：较慢，约2-5氨基酸/秒

### 4.3 病毒翻译

- **策略多样**：
  - 帽子依赖翻译
  - IRES（内部核糖体进入位点）介导的翻译
  - 通读
  - 核糖体移码
  - 多聚蛋白切割
- **利用宿主核糖体**：所有病毒都利用宿主核糖体进行翻译

## 5. 基因表达调控

### 5.1 原核生物基因表达调控

- **转录水平调控**：
  - 操纵子调控（如乳糖操纵子、色氨酸操纵子）
  - 转录因子调控
  - 衰减子调控
- **翻译水平调控**：
  - mRNA稳定性
  - 翻译起始调控
- **特点**：主要在转录水平，调控迅速

### 5.2 真核生物基因表达调控

- **DNA水平**：
  - DNA甲基化
  - 组蛋白修饰
  - 染色质重塑
- **转录水平**：
  - 转录因子
  - 启动子
  - 增强子
  - 沉默子
- **转录后水平**：
  - 可变剪接
  - mRNA稳定性
  - miRNA调控
- **翻译水平**：
  - 翻译起始调控
  - 翻译延伸调控
- **翻译后水平**：
  - 蛋白质修饰
  - 蛋白质降解
- **特点**：多层次调控，精细复杂

### 5.3 病毒基因表达调控

- **利用宿主调控机制**：
  - 利用宿主转录因子
  - 利用宿主翻译 machinery
- **自身调控机制**：
  - 病毒编码的转录因子
  - 病毒编码的 miRNAs
  - 病毒RNA结构调控
- **特点**：适应宿主环境，策略多样

## 6. 翻译后修饰

### 6.1 原核生物翻译后修饰

- **相对简单**：
  - 甲基化
  - 磷酸化
  - 乙酰化
  - 蛋白酶切割
- **特点**：修饰种类较少

### 6.2 真核生物翻译后修饰

- **复杂多样**：
  - **磷酸化**：调控酶活性、信号传导
  - **糖基化**：蛋白质定位、稳定性
  - **甲基化**：调控蛋白质功能
  - **乙酰化**：调控蛋白质活性
  - **泛素化**：标记蛋白质降解
  - **脂酰化**：蛋白质膜定位
  - **羟基化**：胶原蛋白稳定性
  - **ADP-核糖基化**：调控酶活性
- **特点**：修饰种类多，调控精细

### 6.3 病毒蛋白质修饰

- **利用宿主修饰系统**：
  - 糖基化
  - 磷酸化
  - 棕榈酰化
- **自身修饰**：
  - 某些病毒编码自身的修饰酶
- **特点**：修饰有助于病毒蛋白质的功能和免疫逃逸

## 7. 基因表达的意义

### 7.1 细胞功能的实现

- **蛋白质合成**：基因表达产生细胞所需的各种蛋白质
- **酶的合成**：催化细胞内的生化反应
- **结构蛋白**：维持细胞结构
- **信号分子**：传递细胞内外信号

### 7.2 细胞分化与发育

- **细胞命运决定**：通过基因表达调控实现细胞分化
- **组织器官形成**：特定基因在特定组织中表达
- **发育阶段控制**：基因在特定发育阶段表达

### 7.3 环境适应

- **应激反应**：环境变化时调整基因表达
- **代谢适应**：根据营养状况调整代谢相关基因表达
- **免疫应答**：病原体感染时激活免疫相关基因

### 7.4 病毒生命周期

- **病毒复制**：通过基因表达合成病毒蛋白质
- **病毒组装**：表达结构蛋白组装病毒颗粒
- **病毒释放**：表达裂解蛋白促进病毒释放

## 8. 基因表达的研究方法

### 8.1 转录水平研究

- **RT-PCR**：检测特定基因的转录
- **qPCR**：定量基因表达水平
- **RNA-seq**：全转录组分析
- **Northern blot**：检测特定RNA
- **原位杂交**：检测RNA在组织中的定位

### 8.2 翻译水平研究

- **Western blot**：检测特定蛋白质
- **ELISA**：定量蛋白质表达
- **免疫荧光**：检测蛋白质在细胞中的定位
- **蛋白质组学**：全蛋白质组分析
- **核糖体分析**：研究翻译活性

### 8.3 调控机制研究

- **染色质免疫沉淀 (ChIP)**：研究蛋白质与DNA的相互作用
- **EMSA**：研究转录因子与DNA的结合
- **报告基因 assay**：研究启动子活性
- **CRISPR-Cas9**：研究基因功能

### 8.4 病毒基因表达研究

- **病毒转录组分析**：研究病毒基因表达谱
- **病毒蛋白质组分析**：研究病毒蛋白质表达
- **病毒-宿主相互作用**：研究病毒对宿主基因表达的影响

## 9. 基因表达异常与疾病

### 9.1 遗传性疾病

- **单基因疾病**：如囊性纤维化、镰状细胞贫血
- **多基因疾病**：如糖尿病、心血管疾病

### 9.2 癌症

- **原癌基因激活**：如Ras基因突变
- **抑癌基因失活**：如p53基因突变
- **染色体重排**：如慢性粒细胞白血病的费城染色体

### 9.3 神经退行性疾病

- **蛋白质错误折叠**：如阿尔茨海默病的淀粉样蛋白
- **基因表达异常**：如帕金森病的α-突触核蛋白表达

### 9.4 病毒感染

- **病毒基因表达异常**：导致病毒持续感染
- **宿主基因表达异常**：导致免疫功能障碍
- **病毒诱导的癌症**：如HPV导致宫颈癌

## 10. 基因表达的应用

### 10.1 基因治疗

- **基因替代**：用正常基因替代缺陷基因
- **基因沉默**：抑制异常基因的表达
- **基因增强**：增强有益基因的表达

### 10.2 药物研发

- **靶向转录调控**：开发转录因子抑制剂
- **mRNA药物**：如mRNA疫苗
- **蛋白质药物**：如重组胰岛素
- **抗病毒药物**：抑制病毒基因表达

### 10.3 合成生物学

- **基因回路**：设计人工基因表达网络
- **生物传感器**：检测环境信号
- **代谢工程**：优化生物合成途径

### 10.4 农业应用

- **转基因作物**：调控基因表达提高产量
- **基因编辑**：修饰作物基因
- **植物生长调控**：调控植物发育基因

## 11. 前沿研究

### 11.1 非编码RNA研究

- **miRNA**：调控基因表达
- **lncRNA**：参与基因调控
- **circRNA**：作为 miRNA 海绵

### 11.2 表观遗传学

- **DNA甲基化**：调控基因表达
- **组蛋白修饰**：影响染色质结构
- **染色质重塑**：改变基因可及性

### 11.3 单细胞转录组学

- **单细胞RNA-seq**：研究细胞异质性
- **空间转录组学**：研究基因表达的空间分布

### 11.4 病毒基因表达研究

- **病毒转录调控**：研究病毒基因表达的调控机制
- **病毒-宿主相互作用**：研究病毒如何影响宿主基因表达
- **抗病毒策略**：基于基因表达调控的抗病毒方法
