# 一、载体构建

## 1. 定义与核心概念

### 1.1 载体 (Vector)

**定义**：载体是一种能够携带外源 DNA 进入宿主细胞并进行复制和表达的 DNA 分子。

**核心概念**：
- **克隆载体**：用于扩增外源 DNA 的载体
- **表达载体**：用于表达外源基因的载体
- **穿梭载体**：能在多种宿主细胞中复制的载体
- **启动子**：控制基因表达的 DNA 序列
- **选择标记**：用于筛选含载体的细胞

## 2. 克隆载体的类型

### 2.1 质粒载体

- **结构**：
  - 复制起点 (ori)
  - 选择标记（如抗生素抗性基因）
  - 多克隆位点 (MCS)
  - 启动子（表达载体）

- **常用质粒**：
  - pUC 系列：高拷贝数，用于基因克隆
  - pBR322：经典克隆载体
  - pGEM 系列：含 T7/SP6 启动子，用于 RNA 转录

### 2.2 噬菌体载体

- **λ 噬菌体载体**：
  - 容量：约 25 kb
  - 应用：克隆较大的 DNA 片段

- **M13 噬菌体载体**：
  - 单链 DNA
  - 应用：测序、定点突变

### 2.3 cosmids 载体

- **结构**：质粒 + λ 噬菌体 cos 位点
- **容量**：约 45 kb
- **应用**：构建基因组文库

### 2.4 BAC 载体

- **结构**：基于 F 质粒
- **容量**：约 300 kb
- **应用**：构建大片段基因组文库

### 2.5 YAC 载体

- **结构**：酵母染色体元件
- **容量**：100-2000 kb
- **应用**：构建大型基因组文库

## 3. 表达载体的类型

### 3.1 原核表达载体

- **特点**：
  - 原核启动子（如 lac、tac、T7）
  - 核糖体结合位点 (RBS)
  - 多克隆位点
  - 选择标记

- **常用载体**：
  - pET 系列：T7 启动子，高效表达
  - pGEX 系列：融合 GST 标签
  - pMAL 系列：融合 MBP 标签

### 3.2 真核表达载体

- **特点**：
  - 真核启动子（如 CMV、SV40）
  -  poly(A) 信号
  - 选择标记
  - 复制起点

- **常用载体**：
  - pcDNA 系列：哺乳动物细胞表达
  - pEGFP 系列：融合 GFP 标签
  - pYES 系列：酵母表达

## 4. 载体构建步骤

### 4.1 1. 目的基因获取

- **PCR 扩增**：
  - 设计引物，添加酶切位点
  - 扩增目的基因
  - 纯化 PCR 产物

- **从 cDNA 文库获取**：
  - 筛选含目的基因的克隆
  - 提取质粒

- **化学合成**：
  - 对于短片段
  - 含有特殊修饰的序列

### 4.2 2. 载体和插入片段的制备

- **载体酶切**：
  - 选择合适的限制性内切酶
  - 酶切载体，产生粘性末端
  - 去磷酸化，防止自连

- **插入片段制备**：
  - 酶切 PCR 产物或 cDNA
  - 确保与载体末端互补

### 4.3 3. 连接反应

- **连接酶**：
  - T4 DNA 连接酶
  - 最适温度：16°C
  - 反应时间：1-16 小时

- **连接体系**：
  - 载体：插入片段 = 1:3-1:10（摩尔比）
  - 连接缓冲液
  - T4 DNA 连接酶

### 4.4 4. 转化

- **感受态细胞制备**：
  - CaCl₂ 法
  - 电转化法

- **转化步骤**：
  - 冰浴混合连接产物和感受态细胞
  - 热激（42°C，90 秒）
  - 冰浴冷却
  - 加入培养基，复苏
  - 涂布抗生素平板

### 4.5 5. 筛选与鉴定

- **抗生素筛选**：
  - 含抗生素的平板筛选
  - 挑取单克隆

- **菌落 PCR**：
  - 快速鉴定阳性克隆
  - 验证插入片段

- **酶切鉴定**：
  - 提取质粒
  - 酶切验证插入片段大小

- **测序验证**：
  - 确认插入片段序列正确

## 5. 载体构建策略

### 5.1 常规克隆策略

- **粘性末端连接**：
  - 使用相同的限制性内切酶
  - 高效，定向克隆

- **平末端连接**：
  - 使用产生平末端的酶
  - 效率较低，非定向克隆

- **TA 克隆**：
  - 利用 Taq 酶的 3' 加 A 特性
  - 快速克隆 PCR 产物

### 5.2 无缝克隆策略

- **Gibson 组装**：
  - 依赖重叠序列
  - 一步法组装多个片段
  - 适用于复杂载体构建

- **Golden Gate 克隆**：
  - 使用 Type IIS 限制性内切酶
  - 可同时组装多个片段
  - 无疤痕克隆

- **Gateway 克隆**：
  - 基于 λ 噬菌体的位点特异性重组
  - 快速在不同载体间转移基因

## 6. 载体构建注意事项

### 6.1 引物设计

- **添加酶切位点**：
  - 5' 端添加保护碱基
  - 确保酶切位点在正确位置

- **避免内部酶切位点**：
  - 检查目的基因内部是否含有相同的酶切位点

### 6.2 酶切条件

- **酶切温度**：根据酶的最适温度
- **酶切时间**：通常 1-3 小时
- **酶量**：1 U/μg DNA

### 6.3 连接条件

- **载体与插入片段比例**：优化摩尔比
- **连接温度**：16°C 过夜或 25°C 1 小时
- **连接酶量**：根据说明书

### 6.4 转化效率

- **感受态细胞质量**：高转化效率
- **连接产物量**：适量，避免过多
- **复苏时间**：确保细胞恢复

## 7. 应用

### 7.1 基因功能研究

- **过表达研究**：
  - 构建表达载体
  - 转染细胞
  - 检测基因功能

- **基因敲除**：
  - 构建敲除载体
  - 转染细胞
  - 筛选敲除克隆

### 7.2 蛋白质表达与纯化

- **重组蛋白生产**：
  - 构建融合表达载体
  - 诱导表达
  - 纯化蛋白质

- **蛋白质结构研究**：
  - 表达重组蛋白
  - 结晶分析

### 7.3 基因治疗

- **载体构建**：
  - 构建治疗性基因载体
  - 包装病毒
  - 体内递送

### 7.4 疫苗开发

- **抗原表达**：
  - 构建抗原表达载体
  - 表达抗原
  - 制备疫苗
