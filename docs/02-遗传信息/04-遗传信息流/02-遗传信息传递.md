# 遗传信息传递

## 1. 定义与核心概念

### 1.1 遗传信息传递概述

**定义**：遗传信息传递是指生物体内遗传信息从 DNA 到 RNA 到蛋白质的流动过程，包括 DNA 复制（信息保存）、转录（信息转录）和翻译（信息表达）三个核心环节。

**核心概念**：
- **模板链**：DNA 双链中作为转录模板的那条链
- **编码链**：与模板链互补的另一条 DNA 链
- **转录单位**：从启动子到终止子的完整转录区域
- **开放阅读框**：连续编码氨基酸的 DNA 序列
- **遗传密码**：mRNA 上决定氨基酸的三联密码子

:::tip
遗传信息传递的忠实性决定了生物性状的稳定性。任何环节的错误都可能导致突变或疾病。
:::

## 2. DNA 复制

### 2.1 DNA 复制的基本原则

| 原则 | 说明 |
|------|------|
| **半保留复制** | 每条新合成的 DNA 分子包含一条母链和一条新链 |
| **双向复制** | 复制从复制起点向两个方向进行 |
| **半不连续复制** | 前导链连续合成，后随链不连续合成 |
| **保真性** | DNA 聚合酶具有校对功能，确保复制准确性 |

### 2.2 DNA 复制过程

#### 2.2.1 复制起始

| 步骤 | 酶/蛋白 | 功能 |
|------|---------|------|
| **1. 复制起点识别** | DnaA 蛋白 | 识别复制起点 OriC |
| **2. 解旋酶装载** | DnaB、DnaC | 解旋酶装载到 DNA 上 |
| **3. 双链打开** | 解旋酶 | 利用 ATP 能量解开双螺旋 |
| **4. 单链稳定** | 单链结合蛋白 (SSB) | 稳定单链 DNA |
| **5. 引物合成** | 引物酶 (DnaG) | 合成 RNA 引物 |

#### 2.2.2 复制延伸

| 链类型 | 合成方式 | 引物 | 方向 |
|--------|----------|------|------|
| **前导链** | 连续合成 | 1 个引物 | 5' → 3' |
| **后随链** | 不连续合成 | 多个引物 | 5' → 3' |

#### 2.2.3 复制终止

| 步骤 | 酶 | 功能 |
|------|-----|------|
| **1. 引物去除** | DNA 聚合酶 I | 切除 RNA 引物 |
| **2. 缺口填补** | DNA 聚合酶 I | 填补缺口 |
| **3. 片段连接** | DNA 连接酶 | 连接相邻 DNA 片段 |
| **4. 超螺旋解决** | 拓扑异构酶 | 缓解复制产生的超螺旋 |

### 2.3 DNA 复制的酶

| 酶 | 功能 | 备注 |
|----|------|------|
| **解旋酶** | 解开 DNA 双螺旋 | 利用 ATP 能量 |
| **DNA 聚合酶 III** | 合成新 DNA 链 | 原核生物主要复制酶 |
| **DNA 聚合酶 I** | 切除引物、填补缺口 | 原核生物 |
| **DNA 连接酶** | 连接 DNA 片段 | 需要 NAD+ 或 ATP |
| **拓扑异构酶** | 缓解超螺旋张力 | 拓扑异构酶 I 和 II |
| **引物酶** | 合成 RNA 引物 | 短 RNA 片段 |
| **单链结合蛋白** | 稳定单链 DNA | 防止复性 |

## 3. 转录

### 3.1 转录的基本原则

| 原则 | 说明 |
|------|------|
| **不对称转录** | 只有模板链被转录 |
| **基因特异性** | 不同基因在不同组织中表达 |
| **可调控性** | 转录水平是基因表达调控的主要节点 |
| **方向性** | RNA 合成方向为 5' → 3' |

### 3.2 原核生物转录

#### 3.2.1 原核生物基因结构
- **启动子**：-35 区和 -10 区 (Pribnow 盒)
- **结构基因**：连续的编码序列
- **终止子**：终止信号

#### 3.2.2 原核生物转录过程

| 阶段 | 酶 | 关键事件 |
|------|-----|----------|
| **起始** | RNA 聚合酶全酶 | σ 因子识别启动子，形成起始复合物 |
| **延伸** | RNA 聚合酶核心酶 | RNA 链延伸，核心酶沿 DNA 移动 |
| **终止** | RNA 聚合酶 | 识别终止子，释放 RNA 和 DNA |

### 3.3 真核生物转录

#### 3.3.1 真核生物基因结构
- **启动子**：TATA 盒、CAAT 盒、GC 盒
- **5'UTR**：前导序列
- **外显子**：编码序列
- **内含子**：非编码序列（转录后剪接去除）
- **3'UTR**：尾部序列

#### 3.3.2 真核生物转录过程

| 阶段 | 酶 | 关键事件 |
|------|-----|----------|
| **起始** | RNA 聚合酶 II | 转录因子协助结合启动子 |
| **延伸** | RNA 聚合酶 II | RNA 链延伸，伴随 RNA 加工 |
| **终止** | RNA 聚合酶 II | 识别终止信号 |

#### 3.3.3 真核生物 RNA 聚合酶

| 类型 | 转录产物 | 位置 |
|------|----------|------|
| **RNA 聚合酶 I** | rRNA (28S, 18S, 5.8S) | 核仁 |
| **RNA 聚合酶 II** | mRNA、miRNA、snRNA | 核质 |
| **RNA 聚合酶 III** | tRNA、5S rRNA、U6 snRNA | 核质 |

### 3.4 转录后加工

#### 3.4.1 mRNA 加工

| 加工步骤 | 说明 | 酶/因子 |
|----------|------|---------|
| **5' 端加帽** | 添加 7-甲基鸟苷帽子 | 加帽酶复合物 |
| **3' 端加尾** | 添加 poly(A) 尾 | Poly(A) 聚合酶 |
| **剪接** | 去除内含子，连接外显子 | 剪接体 (spliceosome) |
| **编辑** | 碱基替换或插入/缺失 | 腺苷脱氨酶等 |

#### 3.4.2 tRNA 加工

| 加工步骤 | 说明 |
|----------|------|
| **5' 端切割** | 切除前体 tRNA 的 5' 端序列 |
| **3' 端切割** | 切除前体 tRNA 的 3' 端序列 |
| **CCA 添加** | 添加氨基酸结合位点 |
| **碱基修饰** | 甲基化、假尿嘧啶化等 |

## 4. 翻译

### 4.1 翻译的基本原则

| 原则 | 说明 |
|------|------|
| **三联密码子** | 每 3 个核苷酸决定 1 个氨基酸 |
| **简并性** | 多个密码子可编码同一氨基酸 |
| **方向性** | 翻译从起始密码子到终止密码子 |
| **连续性** | 密码子连续阅读，无间隔 |
| **通用性** | 几乎所有生物使用同一套遗传密码 |

### 4.2 遗传密码表

| 密码子特性 | 说明 |
|------------|------|
| **64 种密码子** | 61 种编码氨基酸，3 种终止信号 |
| **起始密码子** | AUG (甲硫氨酸) |
| **终止密码子** | UAA、UAG、UGA |
| **简并性** | 除 Trp 和 Met 各 1 个密码子外，其他氨基酸有 2-6 个 |
| **摆动性** | 反密码子第一位可与密码子第三位非标准配对 |

### 4.3 翻译过程

#### 4.3.1 原核生物翻译

| 阶段 | 关键事件 |
|------|----------|
| **起始** | 30S 小亚基结合 mRNA，起始 fMet-tRNA^fMet 结合起始密码子，50S 大亚基结合形成 70S 起始复合物 |
| **延伸** | 进位、成肽、转位循环 |
| **终止** | 释放因子识别终止密码子，多肽链释放 |

#### 4.3.2 真核生物翻译

| 阶段 | 关键事件 |
|------|----------|
| **起始** | 40S 小亚基扫描 mRNA 寻找起始密码子，Met-tRNA^Met 结合，起始 Met 与 AUG 配对后 60S 大亚基结合 |
| **延伸** | 类似原核生物的进位、成肽、转位循环 |
| **终止** | 释放因子识别终止密码子，多肽链释放 |

### 4.4 翻译的关键成分

| 成分 | 功能 |
|------|------|
| **mRNA** | 蛋白质合成的模板 |
| **tRNA** | 转运氨基酸，识别密码子 |
| **核糖体** | 蛋白质合成的场所 |
| **起始因子** | 协助翻译起始 |
| **延伸因子** | 协助翻译延伸 |
| **释放因子** | 识别终止密码子 |
| **氨基酰-tRNA 合成酶** | 氨基酰化反应 |

## 5. 遗传信息传递的调控

### 5.1 DNA 复制调控

| 调控层面 | 机制 |
|----------|------|
| **复制起始调控** | 复制起点激活、细胞周期调控 |
| **复制叉进程** | 复制速度与染色质状态相关 |
| **复制终止调控** | 复制叉碰撞、端粒复制 |

### 5.2 转录调控

| 调控层面 | 机制 |
|----------|------|
| **染色质结构** | 组蛋白修饰、DNA 甲基化 |
| **转录因子** | 激活或抑制转录 |
| **增强子与启动子** | 远距离调控基因表达 |
| **RNA 加工** | 可变剪接调控信息流 |

### 5.3 翻译调控

| 调控层面 | 机制 |
|----------|------|
| **翻译起始** | eIF 磷酸化、mTOR 通路 |
| **mRNA 稳定性** | 3'UTR 元件、miRNA 调控 |
| **核糖体活性** | 核糖体蛋白磷酸化 |

## 6. 遗传信息传递与疾病

| 疾病类型 | 涉及的传递环节 | 举例 |
|----------|----------------|------|
| **癌症** | 转录调控异常 | MYC 扩增、p53 突变 |
| **遗传病** | 基因突变 | 囊性纤维化、镰刀型贫血 |
| **病毒感染** | 利用宿主翻译机制 | 流感病毒、HIV |
| **神经退行性疾病** | 翻译错误 | 朊病毒病 |

:::warning
遗传信息传递的任何环节出现问题都可能导致疾病。理解这些机制对于药物开发和疾病治疗至关重要。
:::
