# 遗传变异

## 1. 定义与核心概念

### 1.1 遗传变异 (Genetic Variation)

**定义**：遗传变异是指生物个体间基因组序列的差异，这些差异可以来源于 DNA 序列的改变、染色体结构的改变或基因组数量的改变。遗传变异是生物进化的原材料，也是导致个体间性状差异的遗传学基础。

**核心概念**：
- **突变 (Mutation)**：DNA 序列的永久性改变
- **多态性 (Polymorphism)**：群体中存在的常见遗传变异
- **等位基因 (Allele)**：同一基因的不同版本
- **基因型 (Genotype)**：个体的遗传组成
- **表型 (Phenotype)**：可观察到的物理特征

:::tip
遗传变异是自然选择的基础。没有变异，就没有进化。突变是遗传变异的根本来源。
:::

## 2. 突变的类型

### 2.1 按 DNA 序列改变分类

| 突变类型 | 定义 | 举例 | 结果 |
|----------|------|------|------|
| **点突变** | 单个碱基的替换 | A→G | 可能改变编码的氨基酸 |
| **插入突变** | 碱基插入 | AAA→AAAA | 移码突变 |
| **缺失突变** | 碱基缺失 | AAA→AA | 移码突变 |
| **重复突变** | 序列片段重复 | ABC→ABCABC | 改变蛋白长度 |
| **倒位突变** | 序列片段方向反转 | ABC→CBA | 可能破坏基因 |

### 2.2 按对蛋白质功能影响分类

| 突变类型 | 对蛋白的影响 | 功能后果 |
|----------|--------------|----------|
| **同义突变** | 氨基酸不变 | 通常无影响 |
| **错义突变** | 氨基酸改变 | 可能改变蛋白功能 |
| **无义突变** | 产生终止密码子 | 蛋白截短 |
| **移码突变** | 阅读框改变 | 蛋白序列大幅改变 |
| **终止密码子突变** | 终止→氨基酸 | 蛋白延长 |

### 2.3 点突变的分子机制

| 突变类型 | 原因 | 举例 |
|----------|------|------|
| **转换** | 嘌呤→嘌呤，或嘧啶→嘧啶 | A→G，或 T→C |
| **颠换** | 嘌呤→嘧啶，或嘧啶→嘌呤 | A→T，或 G→C |

### 2.4 突变效应示意图

```
DNA:     ATG-Code:  ATG GAA AAA CCT
         ↓突变     ATG GAA AAA CTT   (错义)
         ↓突变     ATG GAA AAB CCT   (无义)
         ↓突变     ATGA GAA AAA CCT  (插入/移码)
```

## 3. 染色体变异

### 3.1 染色体数目变异

#### 3.1.1 整倍体变异

| 类型 | 染色体数目 | 举例 |
|------|------------|------|
| **单倍体** | n | 蜜蜂雄性 |
| **二倍体** | 2n | 人类、果蝇 |
| **三倍体** | 3n | 无籽西瓜 |
| **四倍体** | 4n | 小麦、棉花 |

#### 3.1.2 非整倍体变异

| 类型 | 染色体数目 | 举例 |
|------|------------|------|
| **单体** | 2n-1 | Turner 综合征 (45,X) |
| **三体** | 2n+1 | Down 综合征 (47,XX,+21) |
| **双三体** | 2n+1+1 | 某些遗传病患者 |
| **四体** | 2n+2 | 较少见 |

### 3.2 染色体结构变异

| 变异类型 | 定义 | 细胞学特征 | 遗传效应 |
|----------|------|------------|----------|
| **缺失** | 染色体片段丢失 | 末端缺失、中间缺失 | 基因剂量效应 |
| **重复** | 染色体片段复制 | 串联重复、插入重复 | 基因剂量增加 |
| **倒位** | 片段方向反转 | 臂内倒位、臂间倒位 | 基因顺序改变 |
| **易位** | 片段转移到另一染色体 | 相互易位、罗伯逊易位 | 基因融合 |
| **环状染色体** | 染色体两端连接成环 | 常见于肿瘤 | 基因功能改变 |
| **等臂染色体** | 染色体臂复制融合 | 少见 | 基因剂量改变 |

### 3.3 结构变异示意图

```
缺失:   ABCDEF → ABCF    (中间缺失)
重复:   ABCDEF → ABCDEFDEF
倒位:   ABCDEF → ADCBEF  (臂内倒位)
易位:   ABCDEF + XYZ → ABZ XY CDEF  (相互易位)
```

## 4. 突变的产生机制

### 4.1 自发突变

| 来源 | 机制 | 举例 |
|------|------|------|
| **复制错误** | DNA 聚合酶错配 | 微卫星不稳定 |
| **DNA 稳定性** | 碱基互变异构 | 复制错误 |
| **氧化损伤** | 活性氧攻击 DNA | 8-oxoG 形成 |
| **烷基化损伤** | 化学修饰碱基 | O⁶-甲基鸟嘌呤 |
| **脱氨基** | 胞嘧啶→尿嘧啶 | C→T 突变 |

### 4.2 诱发突变

| 诱变因素 | 作用机制 | 举例 |
|----------|----------|------|
| **物理诱变** | 射线损伤 DNA | X射线、紫外线 |
| **化学诱变** | 化学物质与 DNA 反应 | 亚硝基胍、EMS |
| **生物诱变** | 病毒或转座子插入 | 逆转录病毒 |

### 4.3 诱变剂类型

| 类型 | 作用方式 | 常见诱变剂 |
|------|----------|------------|
| **碱基类似物** | 掺入 DNA 后引起配对错误 | 5-溴尿嘧啶、2-氨基嘌呤 |
| **碱基修饰剂** | 化学修饰碱基 | 亚硝酸、羟胺 |
| **嵌合剂** | 插入碱基对之间 | 溴化乙锭、吖啶橙 |
| **烷化剂** | 烷基化碱基 | 甲基磺酸乙酯 (EMS)、亚硝基胍 |
| **紫外线** | 形成嘧啶二聚体 | UV-A、UV-B |

## 5. 突变的生物学意义

### 5.1 突变与进化

| 意义 | 说明 |
|------|------|
| **提供原材料** | 突变是进化的原始材料 |
| **增加多样性** | 增加群体的遗传多样性 |
| **适应环境** | 某些突变可能提供选择优势 |
| **物种形成** | 积累突变可导致新物种形成 |

### 5.2 突变与疾病

| 疾病类型 | 突变类型 | 举例 |
|----------|----------|------|
| **遗传病** | 先天突变 | 囊性纤维化、镰刀型贫血 |
| **癌症** | 体细胞突变 | p53 突变、BRCA1/2 突变 |
| **代谢病** | 酶缺陷突变 | 苯丙酮尿症、白化病 |
| **神经退行性疾病** | 蛋白错误折叠 | 亨廷顿病、阿尔茨海默病 |

### 5.3 著名突变案例

| 突变 | 效应 | 研究价值 |
|------|------|----------|
| **镰刀型贫血** | β-珠蛋白基因点突变 | 分子病理学模型 |
| **BRCA1/2** | 乳腺癌易感 | 癌症遗传学 |
| **p53** | 多种癌症发生 | 肿瘤抑制基因 |
| **CCR5-Δ32** | HIV 抗性 | 疾病抵抗机制 |

## 6. 突变检测与研究方法

### 6.1 序列变异检测

| 方法 | 原理 | 应用 |
|------|------|------|
| **Sanger 测序** | 双脱氧链终止法 | 验证特定突变 |
| **PCR-SSCP** | 单链构象差异 | 筛查未知突变 |
| **DGGE** | 解链温度差异 | 检测点突变 |
| **基因芯片** | 杂交原理 | 高通量筛查 |
| **NGS** | 高通量测序 | 全基因组分析 |

### 6.2 染色体变异检测

| 方法 | 原理 | 应用 |
|------|------|------|
| **核型分析** | 染色体形态观察 | 数目和结构异常 |
| **FISH** | 荧光探针杂交 | 定位特定序列 |
| **CGH** | 比较杂交 | 检测拷贝数变异 |
| **SNP 芯片** | 基因分型 | 高通量分析 |

:::warning
突变研究需要结合多种方法才能全面了解遗传变异的类型和功能。不同检测方法有各自的优缺点和适用范围。
:::

## 7. 突变与精准医学

### 7.1 药物基因组学

| 应用 | 说明 | 举例 |
|------|------|------|
| **个体化用药** | 根据基因型选择药物 | 华法林剂量预测 |
| **疗效预测** | 基因型预测药物反应 | EGFR 突变与靶向药 |
| **毒性预测** | 避免不良反应 | 硫唑嘌呤与 TPMT |

### 7.2 基因治疗

| 策略 | 说明 | 挑战 |
|------|------|------|
| **基因替换** | 导入正常基因 | 递送效率 |
| **基因修正** | 定点修复突变 | 编辑精度 |
| **基因沉默** | 抑制异常基因表达 | 特异性 |
| **基因编辑** | CRISPR-Cas9 定点改造 | 脱靶效应 |
