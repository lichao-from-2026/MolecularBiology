# 基因结构

## 1. 定义与核心概念

### 1.1 基因 (Gene)

**定义**：基因是一段具有功能的 DNA 序列，能够编码特定的蛋白质或 RNA 分子，是遗传信息的基本单位。

**核心概念**：
- **结构基因**：编码蛋白质或功能性 RNA 的基因
- **调控序列**：控制基因表达的 DNA 序列
- **内含子**：基因中不编码蛋白质的序列
- **外显子**：基因中编码蛋白质的序列
- **启动子**：RNA 聚合酶结合的区域

## 2. 基因的基本结构

### 2.1 原核生物基因结构

- **操纵子**：由启动子、操作基因和结构基因组成的基因表达单位
- **启动子**：RNA 聚合酶结合位点
- **操作基因**：调控蛋白结合位点
- **结构基因**：编码蛋白质的序列
- **终止子**：转录终止信号

### 2.2 真核生物基因结构

| 区域 | 功能 |
|------|------|
| 启动子 | RNA 聚合酶结合，转录起始 |
| 5' 非翻译区 (5' UTR) | 调控翻译起始 |
| 外显子 | 编码蛋白质的序列 |
| 内含子 | 非编码序列，转录后被切除 |
| 3' 非翻译区 (3' UTR) | 调控 mRNA 稳定性和翻译 |
| 终止子 | 转录终止信号 |

## 3. 调控序列

### 3.1 启动子

- **核心启动子**：TATA 盒、起始子、下游启动子元件
- **上游调控元件**：CAAT 盒、GC 盒
- **增强子**：远距离调控基因表达的序列
- **沉默子**：抑制基因表达的序列

### 3.2 转录因子结合位点

- **通用转录因子**：TFIID、TFIIB、TFIIF、TFIIE、TFIIH
- **特异性转录因子**：根据细胞类型和环境信号调节基因表达

### 3.3 染色质结构调控

- **组蛋白修饰**：乙酰化、甲基化、磷酸化等
- **DNA 甲基化**：抑制基因表达
- **染色质重塑**：改变染色质结构，影响基因表达

## 4. 基因表达调控层次

### 4.1 转录前调控
- **染色质结构**： euchromatin vs heterochromatin
- **DNA 甲基化**：CpG 岛甲基化
- **组蛋白修饰**：组蛋白密码

### 4.2 转录调控
- **启动子活性**：RNA 聚合酶结合效率
- **转录因子**：激活或抑制转录
- **增强子与启动子相互作用**：形成转录复合物

### 4.3 转录后调控
- **mRNA 加工**：剪接、加帽、加尾
- **mRNA 稳定性**：3' UTR 中的 AU 富含元件
- **mRNA 运输**：从细胞核到细胞质

### 4.4 翻译调控
- **翻译起始**：5' UTR 结构、起始密码子识别
- **翻译延伸**：核糖体功能
- **翻译终止**：终止密码子识别

### 4.5 翻译后调控
- **蛋白质修饰**：磷酸化、糖基化、 ubiquitination
- **蛋白质折叠**：分子伴侣协助
- **蛋白质定位**：信号肽引导
- **蛋白质降解**：蛋白酶体降解

## 5. 基因家族

### 5.1 定义
- 一组具有相似序列和功能的基因

### 5.2 类型
- **多基因家族**：如血红蛋白基因家族
- **假基因**：无功能的基因拷贝
- **串联重复基因**：如 rRNA 基因

## 6. 基因变异

### 6.1 突变类型
- **点突变**：单个碱基的改变
- **插入/缺失**：碱基的插入或缺失
- **染色体畸变**：缺失、重复、倒位、易位

### 6.2 突变效应
- **同义突变**：不改变氨基酸序列
- **错义突变**：改变氨基酸序列
- **无义突变**：产生终止密码子
- **移码突变**：改变阅读框架

## 7. 应用

### 7.1 基因克隆
- **目的基因获取**：从基因组 DNA 或 cDNA 中扩增
- **载体构建**：将目的基因插入载体
- **转化/转染**：将重组载体导入宿主细胞

### 7.2 基因编辑
- **CRISPR-Cas9**：精确编辑基因
- **TALEN**：转录激活因子样效应物核酸酶
- **ZFN**：锌指核酸酶

### 7.3 基因治疗
- **基因替代**：用正常基因替代缺陷基因
- **基因增强**：增强现有基因的表达
- **基因沉默**：抑制异常基因的表达

### 7.4 基因诊断
- **PCR**：扩增特定基因片段
- **基因测序**：检测基因突变
- **基因芯片**：分析基因表达和突变
