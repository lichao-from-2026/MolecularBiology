# 质粒与载体

## 1. 定义与核心概念

### 1.1 质粒 (Plasmid)

**定义**：质粒是存在于细菌细胞质中的小型、闭合环状双链 DNA 分子，能够独立于染色体 DNA 进行自主复制，并在细胞分裂时随分裂分配到子细胞中。

**核心概念**：
- **复制起点 (Origin of Replication, Ori)**：质粒自主复制的起始位点
- **复制子 (Replicon)**：质粒 DNA 复制的基本单位
- **拷贝数 (Copy Number)**：每个细菌细胞中质粒的数目
- **不相容性 (Incompatibility)**：同一复制子类型的质粒不能稳定共存于同一细胞中

:::tip
质粒是分子克隆中最常用的载体工具，几乎所有基因工程操作都离不开质粒载体。
:::

## 2. 质粒的类型与结构

### 2.1 按功能分类

| 类型 | 功能 | 应用 |
|------|------|------|
| **克隆质粒 (Cloning Vector)** | 复制和保存外源 DNA 片段 | 基因克隆、构建重组 DNA |
| **表达质粒 (Expression Vector)** | 在宿主细胞中表达外源基因 | 蛋白质生产、基因功能研究 |
| **报告质粒 (Reporter Vector)** | 监测基因表达活性 | 启动子分析、基因调控研究 |
| **shRNA/miRNA 质粒** | 调控基因表达 | 基因沉默、RNA 干扰研究 |
| **荧光报告质粒** | 可视化细胞过程 | 蛋白定位、活细胞成像 |

### 2.2 质粒的基本元件

#### 复制相关元件
| 元件 | 说明 |
|------|------|
| **复制起点 (Ori)** | 如 ColE1、pBR322 Ori 等 |
| **拷贝数控制基因** | 调控质粒复制频率 |

#### 选择标记
| 标记类型 | 抗生素 | 应用 |
|----------|--------|------|
| **氨苄青霉素抗性** | Amp | 常规克隆 |
| **卡那霉素抗性** | Kan | 大肠杆菌、哺乳动物细胞 |
| **四环素抗性** | Tet | 大肠杆菌、酵母 |
| **潮霉素抗性** | Hyg | 哺乳动物细胞 |
| **杀稻瘟菌素抗性** | BSD | 哺乳动物细胞 |

### 2.3 质粒的拓扑结构

| 结构类型 | 特性 | 电泳迁移 |
|----------|------|----------|
| **超螺旋质粒** | 最稳定构型 | 最快 |
| **开环质粒** | 一条链断裂 | 最慢 |
| **线性质粒** | 双链断裂 | 中等 |

## 3. 载体类型

### 3.1 按宿主分类

| 载体类型 | 宿主 | 特点 |
|----------|------|------|
| **原核载体** | 大肠杆菌 | 复制快、易操作 |
| **真核载体** | 酵母、哺乳动物细胞 | 含真核表达元件 |
| **穿梭载体** | 多宿主 | 可在多种生物中复制 |

### 3.2 常用载体

| 载体 | 特点 | 应用 |
|------|------|------|
| **pUC 系列** | 高拷贝、常用 | 常规克隆 |
| **pET 系列** | T7 启动子 | 蛋白表达 |
| **pGEX 系列** | GST 融合标签 | 蛋白纯化 |
| **pCMV 系列** | CMV 启动子 | 哺乳动物细胞表达 |
| **Ti 质粒** | 农杆菌介导转化 | 植物转基因 |

## 4. 质粒的复制机制

### 4.1 复制过程

| 阶段 | 关键事件 |
|------|----------|
| **起始** | 识别 Ori 位点，解旋酶打开 DNA |
| **延伸** | DNA 聚合酶合成新链 |
| **终止** | 复制叉相遇，片段连接 |

### 4.2 拷贝数调控

| 调控类型 | 机制 |
|----------|------|
| **负调控** | 通过反义 RNA 或抑制蛋白 |
| **正调控** | 通过激活蛋白促进复制 |

## 5. 质粒提取与纯化

### 5.1 碱裂解法原理

| 步骤 | 说明 |
|------|------|
| **细胞裂解** | NaOH 处理，DNA 变性 |
| **中和** | SDS 和钾盐沉淀蛋白质 |
| **离心** | 去除沉淀，上清含质粒 |
| **纯化** | 乙醇或柱纯化 |

### 5.2 质量检测

| 方法 | 检测指标 |
|------|----------|
| **凝胶电泳** | 拓扑结构、纯度 |
| **分光光度计** | 浓度、纯度 (A260/A280) |
| **酶切检测** | 正确性验证 |
| **测序** | 序列验证 |
