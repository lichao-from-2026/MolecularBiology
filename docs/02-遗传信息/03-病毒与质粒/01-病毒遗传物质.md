# 病毒遗传物质

## 1. 定义与核心概念

### 1.1 病毒 (Virus)

**定义**：病毒是严格细胞内寄生的非细胞生物体，由核酸基因组和蛋白质外壳组成，不能独立进行新陈代谢，必须依赖宿主细胞进行复制和传播。

**核心概念**：
- **病毒粒子 (Virion)**：完整的病毒颗粒，包括基因组和蛋白质外壳
- **衣壳 (Capsid)**：由蛋白质组成的病毒外壳
- **衣壳蛋白 (Capsid Protein)**：构成衣壳的蛋白质亚基
- **包膜 (Envelope)**：某些病毒外包被的脂质膜
- **宿主特异性**：病毒只能感染特定类型的宿主细胞
- **复制策略**：病毒基因组在宿主细胞内的复制方式

:::tip
病毒是分子生物学研究的重要工具，也是基因治疗和疫苗开发的关键载体。病毒的研究推动了分子生物学多个领域的发展。
:::

## 2. 病毒遗传物质的分类

### 2.1 按核酸类型分类

| 核酸类型 | 病毒举例 | 特点 |
|----------|----------|------|
| **双链 DNA (dsDNA)** | 腺病毒、噬菌体、疱疹病毒 | 复制方式类似于宿主 DNA |
| **单链 DNA (ssDNA)** | 细小病毒、M13 噬菌体 | 需要先合成双链中间体 |
| **双链 RNA (dsRNA)** | 轮状病毒、呼肠孤病毒 | 病毒自己携带 RNA 聚合酶 |
| **正义单链 RNA (+ssRNA)** | 脊髓灰质炎病毒、登革病毒 | 可直接作为 mRNA 翻译 |
| **反义单链 RNA (-ssRNA)** | 流感病毒、狂犬病毒 | 需要先合成互补正义链 |
| **逆转录 RNA (rtRNA)** | 逆转录病毒、HIV | 通过逆转录过程复制 |

### 2.2 按基因组结构分类

| 结构类型 | 说明 | 病毒举例 |
|----------|------|----------|
| **连续基因组** | 基因组序列无间隔 | 脊髓灰质炎病毒 |
| **分段基因组** | 基因组分成多个片段 | 流感病毒 (8 segments)、呼肠孤病毒 (10-12 segments) |
| **环状基因组** | 基因组形成闭合环状 | 多瘤病毒、细小病毒 |
| **聚合基因组** | 基因组通过共价连接形成多联体** | 某些病毒粒子中呈递串联体 |

## 3. DNA 病毒

### 3.1 双链 DNA 病毒 (dsDNA Virus)

#### 3.1.1 代表病毒与特征

| 病毒科 | 代表病毒 | 宿主 | 基因组大小 | 包膜 |
|--------|----------|------|------------|------|
| **腺病毒科** | 腺病毒 | 哺乳动物 | 35-40 kb | 无 |
| **疱疹病毒科** | 单纯疱疹病毒、水痘-带状疱疹病毒 | 哺乳动物 | 120-200 kb | 有 |
| **痘病毒科** | 天花病毒、牛痘病毒 | 哺乳动物 | 130-300 kb | 有 |
| **乳头瘤病毒科** | HPV | 哺乳动物 | 7-8 kb | 无 |
| **多瘤病毒科** | SV40、JC 病毒 | 哺乳动物 | 5 kb | 无 |

#### 3.1.2 复制机制

| 阶段 | 过程 |
|------|------|
| **吸附** | 病毒表面蛋白与宿主细胞受体结合 |
| **穿入** | 病毒颗粒进入细胞（受体介导内吞或膜融合） |
| **脱衣壳** | 病毒基因组释放到宿主细胞 |
| **早期转录** | 早期基因转录，合成复制相关蛋白 |
| **基因组复制** | 利用宿主 DNA 复制机制合成病毒 DNA |
| **晚期转录** | 晚期基因转录，合成结构蛋白 |
| **组装** | 病毒基因组与蛋白组装成新病毒颗粒 |
| **释放** | 病毒颗粒通过裂解或出芽方式释放 |

#### 3.1.3 dsDNA 病毒的生物学意义

- **基因治疗载体**：改造的腺病毒和腺相关病毒用于基因治疗
- **疫苗开发**：天花疫苗使用牛痘病毒，HPV 疫苗预防宫颈癌
- **肿瘤研究**：SV40 等病毒用于肿瘤发生机制研究

### 3.2 单链 DNA 病毒 (ssDNA Virus)

#### 3.2.1 代表病毒与特征

| 病毒科 | 代表病毒 | 宿主 | 基因组大小 | 包膜 |
|--------|----------|------|------------|------|
| **细小病毒科** | B19 病毒、腺相关病毒 | 哺乳动物 | 4-6 kb | 无 |
| **噬菌体科** | M13、ΦX174 | 细菌 | 5-7 kb | 无 |
| **圆环病毒科** | 猪圆环病毒 | 哺乳动物 | 1.7-2.3 kb | 无 |

#### 3.2.2 复制机制

1. **ssDNA 转为 dsDNA**：利用宿主 DNA 聚合酶或病毒携带的酶合成互补链
2. **滚动复制**：以双链中间体为模板进行滚环复制
3. **合成病毒蛋白**：利用宿主转录翻译机制
4. **组装与释放**：子代病毒颗粒的组装和释放

#### 3.2.3 临床意义

:::warning
细小病毒 B19 是唯一已知引起人类疾病的单链 DNA 病毒，可导致第五疾病（传染性红斑）、再生障碍性贫血危象等。
:::

| 疾病 | 病毒 | 症状 |
|------|------|------|
| **第五疾病** | B19 细小病毒 | 儿童面部红斑 |
| **再生障碍性贫血** | B19 细小病毒 | 骨髓抑制 |
| **关节疾病** | B19 细小病毒 | 成人关节炎 |

## 4. RNA 病毒

### 4.1 双链 RNA 病毒 (dsRNA Virus)

#### 4.1.1 代表病毒与特征

| 病毒科 | 代表病毒 | 宿主 | 基因组片段数 | 包膜 |
|--------|----------|------|--------------|------|
| **呼肠孤病毒科** | 轮状病毒 | 哺乳动物 | 10-12 | 无 |
| **双 RNA 病毒科** | 传染性法氏囊病病毒 | 鸟类 | 2 | 无 |
| **野田病毒科** | 稻瘟病病毒 | 植物 | 4-5 | 无 |

#### 4.1.2 复制机制特点

| 特点 | 说明 |
|------|------|
| **病毒编码聚合酶** | 病毒粒子携带 RNA 依赖性 RNA 聚合酶 (RdRp) |
| **胞质复制** | 复制和转录在胞质中进行 |
| **不对称转录** | 病毒 mRNA 合成远多于互补链 RNA 合成 |
| **保守复制中间体** | dsRNA 中间体不暴露于胞质中 |

#### 4.1.3 临床意义

| 疾病 | 病毒 | 说明 |
|------|------|------|
| **胃肠炎** | 轮状病毒 | 婴幼儿严重腹泻的主要病因 |
| **呼吸道感染** | 呼肠孤病毒 | 普通感冒病因之一 |

### 4.2 正义单链 RNA 病毒 (+ssRNA Virus)

#### 4.2.1 代表病毒与特征

| 病毒科 | 代表病毒 | 宿主 | 基因组大小 | 包膜 |
|--------|----------|------|------------|------|
| **小 RNA 病毒科** | 脊髓灰质炎病毒、 rhinovirus | 哺乳动物 | 7-8.5 kb | 无 |
| **黄病毒科** | 登革病毒、寨卡病毒、丙型肝炎病毒 | 哺乳动物 | 9.5-12 kb | 有 |
| **冠状病毒科** | SARS-CoV、MERS-CoV、SARS-CoV-2 | 哺乳动物 | 27-31 kb | 有 |
| **杯状病毒科** | 诺如病毒 | 哺乳动物 | 7-8 kb | 无 |

#### 4.2.2 复制机制

1. **直接翻译**：病毒 RNA 进入胞质后直接作为 mRNA 翻译
2. **合成聚合酶**：翻译产生 RNA 依赖性 RNA 聚合酶 (RdRp)
3. **负链合成**：RdRp 以正链为模板合成负链 RNA
4. **基因组复制**：负链 RNA 复制产生新的正链基因组
5. **亚基因组转录**：某些病毒产生多个亚基因组 mRNA

#### 4.2.3 复制策略对比

| 特征 | 脊髓灰质炎病毒 | 冠状病毒 |
|------|----------------|----------|
| **基因组结构** | 连续 | 连续 + 多个亚基因组 |
| **翻译产物** | 多聚蛋白 | 多聚蛋白 + 亚基因组 mRNA |
| **复制场所** | 胞质 | 胞质 |
| **聚合酶** | 3Dpol | nsp12 RdRp |

### 4.3 反义单链 RNA 病毒 (-ssRNA Virus)

#### 4.3.1 代表病毒与特征

| 病毒科 | 代表病毒 | 宿主 | 基因组片段数 | 包膜 |
|--------|----------|------|--------------|------|
| **正黏病毒科** | 流感病毒 | 哺乳动物、鸟类 | 8 | 有 |
| **副黏病毒科** | 麻疹病毒、呼吸道合胞病毒 | 哺乳动物 | 1 | 有 |
| **弹状病毒科** | 狂犬病毒、 vesicular stomatitis virus | 哺乳动物、昆虫 | 1 | 有 |
| **丝状病毒科** | 埃博拉病毒、马尔堡病毒 | 哺乳动物 | 1 | 有 |

#### 4.3.2 复制机制

1. **携带 RdRp**：病毒粒子携带 RNA 依赖性 RNA 聚合酶
2. **合成正义链**：RdRp 以负链为模板合成互补正链 mRNA
3. **转录与复制切换**：早期转录产生 mRNA，后期复制产生完整基因组
4. **组装与出芽**：在特定细胞部位组装并出芽释放

#### 4.3.3 临床意义

| 疾病 | 病毒 | 说明 |
|------|------|------|
| **流感** | 流感病毒 | 季节性呼吸道感染，大流行风险 |
| **麻疹** | 麻疹病毒 | 高传染性病毒性疾病 |
| **狂犬病** | 狂犬病毒 | 致死性脑炎，可通过疫苗预防 |
| **埃博拉出血热** | 埃博拉病毒 | 高致死率出血热 |

## 5. 逆转录病毒

### 5.1 定义与特征

- **定义**：逆转录病毒是一类 RNA 病毒，其复制过程涉及将 RNA 基因组逆转录为 DNA 中间体
- **关键酶**：逆转录酶 (Reverse Transcriptase, RT)
- **特点**：病毒基因组整合到宿主染色体成为前病毒

### 5.2 代表病毒与分类

| 科 | 代表病毒 | 宿主 | 相关疾病 |
|---|----------|------|----------|
| **逆转录病毒科** | HIV、HTLV | 人类 | 艾滋病、T 细胞白血病 |
| **慢病毒属** | HIV-1、HIV-2 | 人类 | 获得性免疫缺陷综合征 |
| **泡沫病毒属** | 泡沫病毒 | 哺乳动物 | 无明显致病性 |

### 5.3 HIV 病毒结构与基因组

#### 5.3.1 病毒结构

| 成分 | 说明 |
|------|------|
| **包膜** | 来自宿主细胞膜，含 gp120 和 gp41 糖蛋白 |
| **衣壳** | 由 p24 蛋白组成，内含病毒基因组和酶 |
| **酶类** | 逆转录酶、整合酶、蛋白酶 |
| **RNA 基因组** | 两个相同的正链 RNA 分子 |

#### 5.3.2 HIV 基因组结构

| 基因 | 功能 |
|------|------|
| **gag** | 编码结构蛋白 (MA, CA, NC, p6) |
| **pol** | 编码酶类 (RT, IN, PR) |
| **env** | 编码包膜糖蛋白 (gp120, gp41) |
| **tat** | 转录激活因子 |
| **rev** | 调控 RNA 剪接和核输出 |
| **nef, vif, vpr, vpu** | 辅助蛋白 |

### 5.4 逆转录病毒复制机制

| 阶段 | 过程 |
|------|------|
| **吸附** | gp120 与 CD4 受体结合 |
| **融合** | gp41 介导病毒包膜与细胞膜融合 |
| **逆转录** | 逆转录酶将 RNA 逆转录为 DNA |
| **整合** | 整合酶将前病毒 DNA 整合到宿主染色体 |
| **转录** | 宿主转录机制产生病毒 RNA |
| **翻译** | 病毒 mRNA 翻译产生病毒蛋白 |
| **组装** | 病毒基因组 RNA 与蛋白组装 |
| **出芽与成熟** | 病毒颗粒出芽，蛋白酶切割成熟 |

### 5.5 逆转录病毒的应用

| 应用领域 | 应用内容 |
|----------|----------|
| **基因治疗** | 逆转录病毒载体用于稳定转导宿主细胞 |
| ** CAR-T 细胞治疗** | 逆转录病毒载体用于 CAR-T 细胞制备 |
| **科学研究** | 作为研究基因功能的工具病毒 |

## 6. 亚病毒因子

### 6.1 类病毒 (Viroid)

#### 6.1.1 定义与特征
- **定义**：类病毒是仅由闭合环状单链 RNA 组成的感染性颗粒，无蛋白质外壳
- **大小**：约 246-401 个核苷酸
- **宿主**：植物（唯一已知天然宿主）
- **复制方式**：依赖宿主 RNA 聚合酶 II 进行复制

#### 6.1.2 类病毒的结构与功能

| 特性 | 说明 |
|------|------|
| **RNA 结构** | 高度碱基配对的杆状结构 |
| **无编码能力** | 不编码任何蛋白质 |
| **催化活性** | 某些类病毒具有核糖酶活性 |
| **复制机制** | 滚环复制，产生多联体中间体 |

#### 6.1.3 类病毒的危害

:::warning
类病毒可引起多种植物病害，如马铃薯纺锤块茎病、柑橘裂皮病等，造成重大农业损失。
:::

| 病害 | 类病毒 | 影响 |
|------|--------|------|
| **马铃薯纺锤块茎病** | PSTVd | 块茎变形、减产 |
| **柑橘裂皮病** | CEVd | 树皮裂开、生长不良 |
| **黄瓜白果病** | CCCVd | 果实白化 |

### 6.2 朊病毒 (Prion)

#### 6.2.1 定义与特征
- **定义**：朊病毒是仅由异常折叠的蛋白质 (PrPSc) 组成的感染性颗粒，不含核酸
- **正常蛋白**：细胞型朊蛋白 (PrPC)
- **异常蛋白**： Scrapie 型朊蛋白 (PrPSc)
- **传播方式**：通过 PrPSc 诱导正常 PrPC 转变为异常构象

#### 6.2.2 朊病毒相关疾病

| 疾病 | 宿主 | 传播方式 |
|------|------|----------|
| **羊瘙痒病** | 绵羊、山羊 | 自然传播 |
| **牛海绵状脑病 (BSE)** | 牛 | 感染性饲料 |
| **克雅氏病 (CJD)** | 人类 | 散发性、遗传性、获得性 |
| **变异型克雅氏病 (vCJD)** | 人类 | 食用感染 BSE 的牛肉 |
| **库鲁病** | 人类 | 食人习俗 |

#### 6.2.3 致病机制

| 假说 | 说明 |
|------|------|
| **构象诱导假说** | PrPSc 与 PrPC 结合，诱导其构象改变 |
| **种子假说** | PrPSc 作为"种子"引发 PrPC 的错误折叠 |
| **聚合体形成** | 错误折叠蛋白聚集形成淀粉样纤维 |

## 7. 病毒遗传物质的比较总结

| 病毒类型 | 核酸形式 | 复制场所 | 关键酶 | 代表病毒 |
|----------|----------|----------|--------|----------|
| **dsDNA** | 双链 DNA | 胞核/胞质 | 宿主 DNA 聚合酶 | 腺病毒、疱疹病毒 |
| **ssDNA** | 单链 DNA | 胞核 | 宿主 DNA 聚合酶 | 细小病毒 |
| **dsRNA** | 双链 RNA | 胞质 | 病毒 RdRp | 轮状病毒 |
| **+ssRNA** | 正链 RNA | 胞质 | 病毒 RdRp | 脊髓灰质炎病毒、冠状病毒 |
| **-ssRNA** | 反义 RNA | 胞质 | 病毒 RdRp | 流感病毒、狂犬病毒 |
| **逆转录病毒** | RNA→DNA | 胞质+胞核 | 逆转录酶 | HIV |
| **类病毒** | 环状 RNA | 胞核 | 宿主 RNA 聚合酶 II | PSTVd |
| **朊病毒** | 无核酸 | -- | -- | PrPSc |

## 8. 病毒遗传物质与宿主的关系

### 8.1 病毒与宿主基因组的相互作用

| 相互作用类型 | 说明 |
|--------------|------|
| **整合** | 逆转录病毒和某些 DNA 病毒将基因组整合到宿主染色体 |
| **基因重组** | 病毒与宿主基因组之间可能发生遗传物质交换 |
| **基因获取** | 宿主可能获取病毒来源的基因（如逆转录酶基因） |
| **免疫逃逸** | 病毒通过突变和抗原漂移逃避免疫监视 |

### 8.2 病毒在分子生物学中的应用

| 应用领域 | 应用内容 |
|----------|----------|
| **基因克隆载体** | 改造的病毒用于递送外源基因 |
| **基因治疗** | 腺相关病毒 (AAV)、慢病毒载体 |
| **疫苗开发** | 减毒活疫苗、病毒样颗粒 (VLP) 疫苗 |
| **基因功能研究** | 利用病毒研究基因调控机制 |
| **蛋白质表达** | 杆状病毒表达系统用于重组蛋白生产 |
