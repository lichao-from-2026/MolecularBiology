# DNA 损伤与修复

## 1. 定义与核心概念

### 1.1 DNA 损伤 (DNA Damage)

**定义**：DNA 损伤是指 DNA 分子结构遭受破坏或改变的现象，包括碱基错配、碱基脱落、碱基修饰、单链断裂、双链断裂等多种类型。细胞通过复杂的 DNA 修复机制来维持基因组的完整性。

**核心概念**：
- **DNA 损伤**：DNA 结构的任何改变
- **修复机制**：细胞识别和修复 DNA 损伤的系统
- **基因组稳定性**：保持 DNA 序列完整性的能力
- **检查点**：细胞周期中确保修复完成的控制点
- **突变**：未修复的 DNA 损伤在复制时产生的永久性改变

:::tip
DNA 修复机制是生命的基本防御系统。修复缺陷会导致基因组不稳定，引发癌症、衰老和多种遗传疾病。
:::

## 2. DNA 损伤的类型

### 2.1 按损伤来源分类

| 来源 | 损伤类型 | 常见例子 |
|------|----------|----------|
| **内源性损伤** | 代谢产物损伤 | 活性氧、烷化剂、脱氨基 |
| **外源性损伤** | 环境因素损伤 | UV 辐射、电离辐射、化学诱变剂 |

### 2.2 按损伤结构分类

| 损伤类型 | 结构改变 | 后果 |
|----------|----------|------|
| **碱基错配** | 错误碱基配对 | 复制错误 |
| **嘧啶二聚体** | 相邻嘧啶共价结合 | 阻断复制和转录 |
| **碱基脱落** | 脱嘌呤/脱嘧啶 | 复制叉停滞 |
| **单链断裂** | DNA 链断裂 | 复制叉解体 |
| **双链断裂** | 两链同时断裂 | 染色体断裂 |
| **烷基化损伤** | 碱基被烷基修饰 | 错配或复制停止 |
| **氧化损伤** | 碱基被氧化 | 突变 |

### 2.3 常见 DNA 损伤详解

| 损伤类型 | 形成机制 | 生物学后果 |
|----------|----------|------------|
| **8-氧代鸟嘌呤 (8-oxoG)** | 活性氧攻击鸟嘌呤 | G→T 颠换突变 |
| **O⁶-甲基鸟嘌呤** | 烷化剂修饰 | G→A 突变 |
| **嘧啶二聚体 (CPD、6-4PP)** | UV 辐射 | 阻断复制和转录 |
| **无碱基位点 (AP 位点)** | 碱基脱落 | 复制叉停滞 |
| **单链断裂** | 氧化或酶切 | 染色体不稳定 |
| **双链断裂** | 电离辐射等 | 最严重的损伤类型 |

## 3. DNA 修复机制概述

### 3.1 主要修复途径

| 修复途径 | 修复对象 | 修复机制 |
|----------|----------|----------|
| **直接修复** | 特定损伤 | 直接逆转损伤 |
| **碱基切除修复 (BER)** | 小损伤 | 切除受损碱基 |
| **核苷酸切除修复 (NER)** | 大损伤 | 切除含损伤的寡核苷酸 |
| **错配修复 (MMR)** | 复制错误 | 切除错配碱基 |
| **双链断裂修复** | 双链断裂 | 同源重组或 NHEJ |
| **复制后修复** | 复制叉崩溃 | 重组修复 |

### 3.2 修复途径的选择

| 因素 | 影响 |
|------|------|
| **损伤类型** | 不同损伤由不同途径修复 |
| **细胞周期** | S 期和 G2 期优先同源重组 |
| **损伤程度** | 严重损伤需要复杂修复机制 |
| **染色质状态** | 开放染色质更易修复 |

## 4. 直接修复

### 4.1 光复活修复 (Photoreactivation)

| 特性 | 说明 |
|------|------|
| **修复对象** | 嘧啶二聚体 (CPD、6-4PP) |
| **酶** | 光复活酶 (DNA photolyase) |
| **能量来源** | 光能 (300-500 nm) |
| **机制** | 嘧啶二聚体→两个嘧啶 |
| **分布** | 主要在低等生物和植物中 |

### 4.2 O⁶-甲基鸟嘌呤-DNA 甲基转移酶 (MGMT)

| 特性 | 说明 |
|------|------|
| **修复对象** | O⁶-甲基鸟嘌呤、O⁶-氯乙基鸟嘌呤 |
| **机制** | 接受甲基/氯乙基，自身失活 |
| **特点** | 一分子修复酶修复一个损伤 |
| **意义** | 是细胞对烷化剂的主要防御 |

### 4.3 直接修复示意图

```
嘧啶二聚体修复:
    UV损伤: T-T(CPD) → [光复活酶 + 光] → T=T (分离)

O⁶-甲基鸟嘌呤修复:
    O⁶-MeG → [MGMT] → G (MGMT失活)
```

## 5. 碱基切除修复 (Base Excision Repair, BER)

### 5.1 BER 的基本过程

| 步骤 | 酶 | 过程 |
|------|-----|------|
| **1. 损伤识别** | DNA 糖基化酶 | 识别并切除受损碱基 |
| **2. 链切割** | AP 内切酶 | 在 AP 位点切割 DNA |
| **3. 缺口处理** | 磷酸二酯酶 | 去除 AP 位点 |
| **4. DNA 合成** | DNA 聚合酶 β | 填补缺口 |
| **5. 连接** | DNA 连接酶 | 连接断口 |

### 5.2 DNA 糖基化酶的类型

| 糖基化酶 | 识别损伤 | 产物 |
|----------|----------|------|
| **UNG** | 尿嘧啶 | AP 位点 |
| **OGG1** | 8-氧代鸟嘌呤 | AP 位点 |
| **MPG** | 3-甲基腺嘌呤等 | AP 位点 |
| **TDG** | T/G 错配 | AP 位点 |
| **MUTYH** | A/8-oxoG | AP 位点 |

### 5.3 BER 示意图

```
损伤碱基:     G    (受损)
    ↓ DNA糖基化酶
AP位点:       X    (无碱基)
    ↓ AP内切酶
切割:         X|   (单链断裂)
    ↓ DNA聚合酶β
填补:         G    (新合成)
    ↓ DNA连接酶
完整:         G-C  (修复完成)
```

## 6. 核苷酸切除修复 (Nucleotide Excision Repair, NER)

### 6.1 NER 的基本过程

| 步骤 | 关键蛋白 | 过程 |
|------|----------|------|
| **1. 损伤识别** | XPC-HR23B 等 | 识别扭曲的 DNA |
| **2. 损伤验证** | TFIIH、XPA、XPD | 解旋打开 DNA |
| **3. 切割** | XPG、XPF-ERCC1 | 切除含损伤的寡核苷酸 |
| **4. DNA 合成** | DNA 聚合酶 δ/ε | 填补缺口 |
| **5. 连接** | DNA 连接酶 | 连接断口 |

### 6.2 NER 的两种类型

| 类型 | 识别方式 | 发挥作用 |
|------|----------|----------|
| **全基因组 NER (GG-NER)** | 全局扫描 | 修复转录沉默区的损伤 |
| **转录偶联 NER (TC-NER)** | RNA 聚合酶停滞 | 优先修复转录链的损伤 |

### 6.3 NER 缺陷与疾病

| 疾病 | 缺陷 | 症状 |
|------|------|------|
| **着色性干皮病 (XP)** | XPA-XPG 突变 | UV敏感、皮肤癌高发 |
| **Cockayne 综合征** | CSA、CSB 突变 | 发育迟缓、神经退行性 |
| **三核苷酸重复疾病** | NER 相关 | 脆性 X、 Huntington |

## 7. 错配修复 (Mismatch Repair, MMR)

### 7.1 MMR 的基本过程

| 步骤 | 蛋白 | 过程 |
|------|------|------|
| **1. 识别** | MutS 同源二聚体 | 识别错配碱基 |
| **2. 招募** | MutL | 招募其他蛋白 |
| **3. 切除** | MutH、解旋酶、外切酶 | 切除包含错配的 DNA 片段 |
| **4. 合成** | DNA 聚合酶 III | 填补缺口 |
| **5. 连接** | DNA 连接酶 | 连接断口 |

### 7.2 原核与真核 MMR 蛋白比较

| 原核生物 | 真核生物 | 功能 |
|----------|----------|------|
| **MutS** | MSH2、MSH6 (MutSα) | 识别错配 |
| **MutL** | MLH1、PMS2 (MutLα) | 介导修复 |
| **MutH** | 无 | 切除信号 |

### 7.3 MMR 的生物学意义

| 意义 | 说明 |
|------|------|
| **复制保真性** | 将复制错误率从 10⁻⁷ 降至 10⁻⁹ |
| **基因组稳定性** | 防止突变积累 |
| **肿瘤抑制** | MMR 缺陷导致微卫星不稳定 (MSI) |

## 8. 双链断裂修复

### 8.1 同源重组修复 (Homologous Recombination, HR)

| 特性 | 说明 |
|------|------|
| **模板** | 需要同源 DNA 序列作为模板 |
| **细胞周期** | 主要在 S/G2 期 |
| **保真性** | 高保真修复 |
| **关键蛋白** | Rad51、BRCA1、BRCA2 |

#### 8.1.1 HR 的基本过程

| 步骤 | 蛋白/酶 | 过程 |
|------|---------|------|
| **1. 末端加工** | MRN 复合物、CtIP | 产生 3' 单链 DNA |
| **2. Rad51 装载** | BRCA2 | 形成 Rad51 核蛋白丝 |
| **3. 链入侵** | Rad51 | 入侵同源双链 DNA |
| **4. D-loop 形成** | -- | 形成十字结构 |
| **5. DNA 合成** | DNA 聚合酶 | 以同源链为模板合成 |
| **6. 交叉形成** | -- | 形成 Holiday 结构 |
| **7. 拆分连接** | -- | 分离连接 |

#### 8.1.2 HR 示意图

```
双链断裂:
    ---ABCDEF---
    ---ABCDEF---

末端加工:
    ---ABCDEF---
    ----abc----

Rad51装载:
    ---ABCDEF~~~
    ~~~~~abc---

链入侵:
    ---ABCDEF---
    ---ABcdef---
        ↑
    D-loop

DNA合成:
    ---ABCDEF---
    ---ABcDE---
        ↓
    交叉结构
```

### 8.2 非同源末端连接 (Non-Homologous End Joining, NHEJ)

| 特性 | 说明 |
|------|------|
| **模板** | 不需要同源序列 |
| **细胞周期** | 整个细胞周期 |
| **保真性** | 可能引入突变或缺失 |
| **关键蛋白** | Ku70/80、DNA-PKcs、XRCC4、DNA 连接酶 IV |

#### 8.2.1 NHEJ 的基本过程

| 步骤 | 蛋白 | 过程 |
|------|------|------|
| **1. 结合** | Ku70/80 | 结合 DNA 末端 |
| **2. 招募** | DNA-PKcs | 招募 DNA-PKcs |
| **3. 加工** | 核酸酶 | 去除突出末端 |
| **4. 连接** | XRCC4、DNA 连接酶 IV | 直接连接末端 |

### 8.3 HR 与 NHEJ 的比较

| 特性 | 同源重组 (HR) | 非同源末端连接 (NHEJ) |
|------|---------------|----------------------|
| **同源序列要求** | 需要 | 不需要 |
| **细胞周期** | S/G2 期 | 整个周期 |
| **保真性** | 高 | 可能出错 |
| **修复时间** | 较慢 | 快速 |
| **主要功能** | 复制叉崩溃修复、减数分裂 | 双链断裂修复 |

## 9. DNA 损伤检查点

### 9.1 检查点机制

| 检查点 | 细胞周期位置 | 激活条件 |
|--------|--------------|----------|
| **G1/S 检查点** | G1→S | DNA 损伤 |
| **S 期检查点** | S 期 | 复制压力、DNA 损伤 |
| **G2/M 检查点** | G2→M | DNA 损伤 |

### 9.2 检查点通路

| 蛋白 | 功能 |
|------|------|
| **ATM** | 感受双链断裂，激活检查点 |
| **ATR** | 感受单链断裂和复制压力 |
| **Chk1** | ATR 下游效应器 |
| **Chk2** | ATM 下游效应器 |
| **p53** | 检查点核心调控因子 |
| **Cdc25** | 抑制 CDK，阻止细胞周期 |

## 10. DNA 修复与疾病

### 10.1 修复缺陷相关疾病

| 疾病 | 缺陷的修复途径 | 症状 |
|------|----------------|------|
| **着色性干皮病 (XP)** | NER | 皮肤癌高发、光敏感 |
| **遗传性乳腺癌/卵巢癌** | HR (BRCA1/2) | 癌症易感 |
| **Lynch 综合征** | MMR | 结直肠癌、子宫内膜癌 |
| **Fanconi 贫血** | DNA 交联修复 | 骨髓衰竭、癌症 |
| **Ataxia-telangiectasia** | ATM | 神经退行性、运动失调 |

### 10.2 DNA 修复与癌症

| 关系 | 说明 |
|------|------|
| **突变积累** | 修复缺陷导致突变积累 |
| **基因组不稳定** | 是癌症的重要特征 |
| **靶向治疗** | PARP 抑制剂用于 BRCA 缺陷肿瘤 |
| **放化疗增敏** | 抑制修复增强治疗效果 |

### 10.3 DNA 修复与衰老

| 机制 | 说明 |
|------|------|
| **突变积累** | 修复能力下降导致突变增加 |
| **端粒缩短** | 复制性衰老 |
| **线粒体 dysfunction** | 增加氧化损伤 |
| **衰老相关疾病** | 与多种年龄相关疾病相关 |

:::warning
DNA 修复机制是维持基因组完整性的关键系统。了解这些机制对于理解癌症、衰老和遗传疾病具有重要意义。
:::

## 11. DNA 修复与治疗

### 11.1 放化疗增敏

| 策略 | 原理 | 药物/方法 |
|------|------|-----------|
| **抑制 BER** | 增加氧化损伤毒性 | PARP 抑制剂 |
| **抑制 NHEJ** | 增加双链断裂毒性 | DNA-PKcs 抑制剂 |
| **抑制 HR** | 合成致死 | PARP 抑制剂 (BRCA缺陷) |

### 11.2 合成致死策略

| 靶点 | 适用人群 | 药物 |
|------|----------|------|
| **PARP** | BRCA1/2 缺陷 | Olaparib、Rucaparib |
| **DNA-PKcs** | 特定肿瘤 | cc-115 |
| **ATM** | ATM 缺陷肿瘤 | -- |

### 11.3 基因治疗前景

| 方向 | 说明 |
|------|------|
| **基因补偿** | 导入正常基因补偿缺陷 |
| **编辑修复** | 使用 CRISPR 精确修复突变 |
| **药物增强** | 小分子增强特定修复途径 |
