# 非编码 RNA

## 1. 定义与核心概念

### 1.1 非编码 RNA (Non-coding RNA, ncRNA)

**定义**：非编码 RNA 是指不编码蛋白质的 RNA 分子，它们在细胞内直接发挥功能，参与基因表达调控、蛋白质合成、RNA 加工等多种生命活动。

**核心概念**：
- **非编码 RNA**：不携带蛋白质翻译信息的 RNA 总称
- **小 RNA**：长度约 20-30 个核苷酸的 RNA 分子
- **长非编码 RNA**：长度超过 200 个核苷酸的非编码 RNA
- **RNA 干扰**：小 RNA 诱导的基因沉默现象

:::tip
人类基因组中约 98% 的转录产物是非编码 RNA，只有约 2% 编码蛋白质。这一发现彻底改变了人们对基因功能的认识。
:::

## 2. 小 RNA 的类型与功能

### 2.1 微小 RNA (MicroRNA, miRNA)

#### 2.1.1 定义与特征
- **长度**：约 20-25 个核苷酸
- **来源**：由较长的初级转录本（pri-miRNA）经过 Drosha 和 Dicer 酶加工生成
- **功能**：通过与靶 mRNA 的 3' UTR 区域不完全互补配对，抑制翻译或促进 mRNA 降解

#### 2.1.2 miRNA 合成途径

| 步骤 | 位置 | 关键酶 | 产物 |
|------|------|--------|------|
| **转录** | 细胞核 | RNA 聚合酶 II | pri-miRNA |
| **第一次加工** | 细胞核 | Drosha + DGCR8 | pre-miRNA (发夹结构) |
| **核输出** | 细胞核→细胞质 | Exportin-5 | pre-miRNA |
| **第二次加工** | 细胞质 | Dicer + TRBP | miRNA 双链体 |
| **链分离** | 细胞质 | -- | 成熟 miRNA |

#### 2.1.3 miRNA 的作用机制

| 作用模式 | 配对程度 | 结果 |
|----------|----------|------|
| **翻译抑制** | 不完全配对 | 抑制 mRNA 翻译 |
| **mRNA 降解** | 完全配对 | 促进 mRNA 降解 |
| **转录抑制** | -- | 参与转录沉默 |

#### 2.1.4 miRNA 的生物学功能

- **发育调控**：调控细胞分化、组织发育
- **细胞周期调控**：控制细胞增殖和凋亡
- **代谢调控**：参与糖代谢、脂代谢调控
- **疾病相关**：与癌症、代谢疾病密切相关

### 2.2 小干扰 RNA (Small Interfering RNA, siRNA)

#### 2.2.1 定义与特征
- **长度**：约 20-23 个核苷酸
- **来源**：通常由外源双链 RNA（如病毒、转基因）加工而成
- **功能**：与靶 mRNA 完全互补配对，诱导 RNA 诱导的沉默复合物 (RISC) 切割降解靶 mRNA

#### 2.2.2 siRNA 与 miRNA 的对比

| 特性 | siRNA | miRNA |
|------|-------|-------|
| **来源** | 外源双链 RNA | 基因组编码 |
| **前体** | 长双链 RNA | 发夹结构 pre-miRNA |
| **靶向** | 高度特异性 | 可能靶向多个 mRNA |
| **作用方式** | 完全配对，切割降解 | 不完全配对，抑制翻译 |
| **功能** | 防御外源核酸 | 内源性基因调控 |

#### 2.2.3 siRNA 的应用

:::warning
siRNA 技术已广泛应用于基因功能研究和药物开发，但面临递送效率低、脱靶效应等挑战。
:::

| 应用领域 | 应用内容 |
|----------|----------|
| **基因功能研究** | 敲低特定基因表达，研究基因功能 |
| **药物开发** | 治疗病毒感染、癌症等疾病 |
| **农业应用** | 抗病毒转基因作物 |

### 2.3 PIWI 互作 RNA (PIWI-interacting RNA, piRNA)

#### 2.3.1 定义与特征
- **长度**：约 24-31 个核苷酸
- **来源**：由较长的单链前体加工生成
- **功能**：与 PIWI 蛋白结合，调控转座子沉默和配子形成

#### 2.3.2 piRNA 的生物学功能

| 功能 | 说明 |
|------|------|
| **转座子沉默** | 保护基因组完整性，防止转座子跳跃 |
| **生殖细胞发育** | 参与精子形成和卵子发育 |
| **表观遗传调控** | 参与建立和维持表观遗传修饰 |

## 3. 其他非编码 RNA

### 3.1 转运 RNA (Transfer RNA, tRNA)

#### 3.1.1 定义与结构
- **长度**：约 76-90 个核苷酸
- **功能**：在蛋白质合成过程中转运氨基酸
- **结构特征**：三叶草形，包含受体臂、D 环、反密码子环、可变环、TΨC 环

#### 3.1.2 tRNA 的功能

| 功能 | 说明 |
|------|------|
| **氨基酸转运** | 结合特定氨基酸，运送到核糖体 |
| **密码子识别** | 反密码子与 mRNA 密码子配对 |
| **翻译起始** | 起始 tRNA 识别起始密码子 |

### 3.2 核糖体 RNA (Ribosomal RNA, rRNA)

#### 3.2.1 定义与组成
- **功能**：构成核糖体的主要组成部分，催化蛋白质合成
- **组成**：原核生物 23S + 5S rRNA，真核生物 28S + 5.8S + 5S rRNA

#### 3.2.2 rRNA 的功能

| 功能 | 说明 |
|------|------|
| **核糖体组装** | 与蛋白质组装成核糖体亚基 |
| **肽酰转移酶活性** | 催化肽键形成 |
| **mRNA 结合** | 结合 mRNA 进行翻译 |

### 3.3 信使 RNA (Messenger RNA, mRNA)

#### 3.3.1 定义与特征
- **功能**：携带遗传信息，从 DNA 到蛋白质的信使
- **结构**：5'帽、5' UTR、编码区、3' UTR、3' poly(A) 尾

#### 3.3.2 mRNA 的加工

| 加工步骤 | 说明 |
|----------|------|
| **5' 加帽** | 形成 7-甲基鸟苷帽，保护 RNA，促进翻译 |
| **剪接** | 去除内含子，连接外显子 |
| **3' 加尾** | 添加 poly(A) 尾，增强稳定性 |

### 3.4 核糖核酸酶 P (RNase P)

- **功能**：切割前体 tRNA 的 5' 端前导序列
- **催化活性**：具有核糖酶活性，是重要的核糖核蛋白复合物

### 3.5 核糖核酸酶 MRP

- **功能**：切割前体 rRNA 的 ITS1 区域，参与核糖体生物合成
- **位置**：主要存在于细胞核

### 3.6 小核 RNA (Small Nuclear RNA, snRNA)

- **功能**：参与前体 mRNA 剪接
- **主要类型**：U1、U2、U4、U5、U6 snRNA
- **位置**：主要存在于细胞核（剪接体）

### 3.7 小核仁 RNA (Small Nucleolar RNA, snoRNA)

- **功能**：指导 rRNA 和其他 RNA 的化学修饰（甲基化、假尿苷化）
- **分类**：C/D box snoRNA（指导 2'-O-甲基化）、H/ACA box snoRNA（指导假尿苷化）

## 4. 长非编码 RNA (Long Non-coding RNA, lncRNA)

### 4.1 定义与分类

| 类型 | 长度 | 说明 |
|------|------|------|
| **基因间 lncRNA** | >200 nt | 位于基因之间的区域 |
| **正义 lncRNA** | >200 nt | 与蛋白质编码基因同义方向 |
| **反义 lncRNA** | >200 nt | 与蛋白质编码基因反向转录 |
| **增强子 RNA** | >200 nt | 来自增强子区域的转录物 |

### 4.2 lncRNA 的作用机制

| 机制 | 说明 |
|------|------|
| **染色质修饰** | 招募染色质修饰复合物，调控基因沉默或激活 |
| **转录调控** | 作为转录因子或辅助因子调控基因转录 |
| **mRNA 加工** | 影响可变剪接、运输、翻译 |
| **蛋白质定位** | 影响蛋白质在细胞内的定位 |

### 4.3 重要的 lncRNA 实例

| lncRNA | 功能 |
|--------|------|
| **Xist** | 介导 X 染色体失活 |
| **HOTAIR** | 调控 HOX 基因表达，与癌症转移相关 |
| **MALAT1** | 参与基因表达调控，与多种癌症相关 |
| **NEAT1** | 形成核散斑体，参与基因表达调控 |

## 5. 环状 RNA (Circular RNA, circRNA)

### 5.1 定义与特征

- **定义**：由前体 mRNA 通过反向剪接形成的闭合环状 RNA 分子
- **特征**：无 5' 帽和 3' poly(A) 尾，环状结构使其具有高稳定性
- **功能**：主要作为 miRNA 海绵，调控基因表达

### 5.2 circRNA 的生物学功能

| 功能 | 说明 |
|------|------|
| **miRNA 海绵** | 吸附 miRNA，解除 miRNA 对靶 mRNA 的抑制 |
| **蛋白质海绵** | 结合特定蛋白质，调控其功能 |
| **翻译模板** | 部分 circRNA 可作为翻译模板产生肽段 |
| **免疫调节** | 参与天然免疫应答调控 |

### 5.3 circRNA 与疾病的关联

| 疾病 | circRNA 作用 |
|------|-------------|
| **癌症** | 调控肿瘤细胞增殖、迁移、侵袭 |
| **心血管疾病** | 参与心肌梗死、心力衰竭等过程 |
| **神经系统疾病** | 与阿尔茨海默病、帕金森病相关 |

## 6. 非编码 RNA 的实验研究方法

### 6.1 小 RNA 的检测与定量

| 方法 | 说明 |
|------|------|
| **Northern blot** | 检测特定小 RNA 的表达 |
| **RT-qPCR** | 定量检测小 RNA 表达水平 |
| **RNA-seq** | 高通量测序分析小 RNA 表达谱 |
| **微阵列** | 检测特定小 RNA 的表达变化 |

### 6.2 miRNA 功能研究

| 方法 | 应用 |
|------|------|
| **miRNA mimic** | 增强 miRNA 功能 |
| **miRNA inhibitor (antimiR)** | 抑制 miRNA 功能 |
| **Target protector** | 保护特定 mRNA 的 miRNA 结合位点 |
| **双荧光素酶报告基因** | 验证 miRNA 与靶 mRNA 的相互作用 |

### 6.3 lncRNA 功能研究

| 方法 | 应用 |
|------|------|
| **RNAi** | 敲低 lncRNA 表达 |
| **CRISPR-Cas13** | 特异性降解 lncRNA |
| **ASO (反义寡核苷酸)** | 靶向降解 lncRNA |
| **ChIRP** | 鉴定 lncRNA 的基因组结合位点 |

## 7. 非编码 RNA 的比较总结

| 类型 | 长度 | 主要功能 | 代表性分子 |
|------|------|----------|-----------|
| **miRNA** | 20-25 nt | 基因表达调控 | let-7, miR-21 |
| **siRNA** | 20-23 nt | 防御外源核酸 | -- |
| **piRNA** | 24-31 nt | 转座子沉默 | piR-823 |
| **tRNA** | 76-90 nt | 转运氨基酸 | -- |
| **rRNA** | 120-5000 nt | 蛋白质合成 | 28S, 18S, 5S |
| **snRNA** | 100-200 nt | mRNA 剪接 | U1, U2 |
| **snoRNA** | 60-300 nt | rRNA 修饰 | -- |
| **lncRNA** | >200 nt | 基因表达调控 | Xist, HOTAIR |
| **circRNA** | 环状 | miRNA 海绵 | ciRS-7 |
