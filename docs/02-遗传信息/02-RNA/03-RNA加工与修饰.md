# RNA 加工与修饰

## 1. 定义与核心概念

### 1.1 RNA 加工 (RNA Processing)

**定义**：RNA 加工是指真核生物中，前体 RNA (pre-mRNA) 在转录后经历的一系列化学修饰和剪切过程，包括 5' 端加帽、3' 端加尾、剪接等，最终形成成熟的功能性 RNA 分子。

**核心概念**：
- **转录后加工**：RNA 从前体到成熟的过程
- **可变剪接**：同一基因产生不同蛋白亚型
- **RNA 编辑**：碱基的化学改变
- **核输出**：成熟 RNA 从细胞核到细胞质的转运

:::tip
RNA 加工是真核生物基因表达调控的重要层次，也是产生蛋白质多样性的关键机制之一。
:::

## 2. mRNA 的加工

### 2.1 5' 端加帽 (5' Capping)

| 特征 | 说明 |
|------|------|
| **结构** | 7-甲基鸟苷 (m7GpppN) 帽子 |
| **位置** | mRNA 的 5' 端 |
| **酶** | 加帽酶复合物 (RNA triphosphatase, guanylyltransferase, methyltransferase) |
| **功能** | 稳定 mRNA、促进翻译、核输出、防止降解 |

#### 加帽过程

| 步骤 | 酶 | 反应 |
|------|-----|------|
| **1. 去磷酸** | RNA triphosphatase | 去除 γ-磷酸 |
| **2. 加 GMP** | Guanylyltransferase | 添加 GMP |
| **3. 甲基化** | Guanine-7-methyltransferase | 鸟嘌呤甲基化 |
| **4. 核糖甲基化** | Ribose-2'-O-methyltransferase | 第一个核苷酸核糖甲基化 |

### 2.2 3' 端加尾 (3' Polyadenylation)

| 特征 | 说明 |
|------|------|
| **结构** | poly(A) 尾 (约 150-250 个腺苷酸) |
| **位置** | mRNA 的 3' 端 |
| **酶** | Poly(A) 聚合酶 (PAP) |
| **信号** | AAUAAA 序列 |

#### 加尾过程

| 步骤 | 关键事件 |
|------|----------|
| **1. 信号识别** | CPSF 识别 AAUAAA 信号 |
| **2. 切割** | CstF 协助在信号下游切割 |
| **3. 加尾** | Poly(A) 聚合酶添加 poly(A) |
| **4. 保护** | PABP 结合 poly(A) 尾 |

### 2.3 剪接 (Splicing)

| 特征 | 说明 |
|------|------|
| **定义** | 去除内含子，连接外显子 |
| **场所** | 剪接体 (spliceosome) |
| **关键序列** | 5' 剪接位点 (GU)、分支点 (A)、3' 剪接位点 (AG) |

#### 剪接过程

| 阶段 | 事件 |
|------|------|
| **组装** | U1、U2 小核RNA 结合 |
| **催化前** | U4/U6.U5 三联体结合 |
| **催化** | 两步转酯化反应 |
| **解离** | 剪接体释放 mRNA |

#### 可变剪接类型

| 类型 | 结果 | 举例 |
|------|------|------|
| **盒式外显子** | 外显子跳跃 | 心肌肌钙蛋白 |
| **可变 5' 剪接位点** | 外显子长度变化 | 原肌球蛋白 |
| **可变 3' 剪接位点** | 外显子长度变化 | 免疫球蛋白 |
| **互斥外显子** | 二选一 | FGF 受体 |

## 3. tRNA 的加工

### 3.1 加工步骤

| 步骤 | 酶 | 产物 |
|------|-----|------|
| **5' 端切割** | RNase P | 切除前导序列 |
| **3' 端切割** | RNase Z | 切除尾部序列 |
| **CCA 添加** | tRNA 核苷酸转移酶 | 添加 3' CCA |
| **碱基修饰** | 多种修饰酶 | 修饰碱基 |

### 3.2 tRNA 修饰类型

| 修饰 | 位置 | 功能 |
|------|------|------|
| **假尿嘧啶** | D 环、TΨC 环 | 稳定性 |
| **甲基化** | 碱基 | 稳定性、识别 |
| **硫代** | U34 | 密码子识别 |
| **二氢尿嘧啶** | D 环 | 折叠 |

## 4. rRNA 的加工

### 4.1 原核生物 rRNA 加工

| 前体 | 产物 |
|------|------|
| **30S rRNA 前体** | 16S rRNA |
| **50S rRNA 前体** | 23S rRNA, 5S rRNA |

### 4.2 真核生物 rRNA 加工

| 前体 | 产物 | 场所 |
|------|------|------|
| **45S pre-rRNA** | 18S, 5.8S, 28S rRNA | 核仁 |
| **5S rRNA** | 直接成熟 | 核质 |

#### 加工过程
1. **转录**：RNA 聚合酶 I 转录 45S pre-rRNA
2. **甲基化**：主要发生在核糖 2' 位
3. **切割**：分级切割产生成熟 rRNA
4. **组装**：与核糖体蛋白组装

## 5. RNA 编辑 (RNA Editing)

### 5.1 碱基替换型编辑

| 类型 | 机制 | 结果 |
|------|------|------|
| **C→U 编辑** | 胞嘧啶脱氨 | C→U (如 ApoB) |
| **A→I 编辑** | 腺嘌呤脱氨 | A→I (如 ADAR) |

### 5.2 插入/缺失型编辑

| 类型 | 说明 | 实例 |
|------|------|------|
| **U 插入** | 引导 RNA 指导 U 插入 | 锥虫线粒体 |
| **C 插入** | 引导 RNA 指导 C 插入 | 锥虫 |

## 6. RNA 核输出

### 6.1 mRNA 核输出

| 蛋白 | 功能 |
|------|------|
| **NXF1/TAP** | 主要出口受体 |
| **CBC** | 帽结合复合物 |
| **EJC** | 剪接外显子连接复合物 |

### 6.2 tRNA 核输出

| 蛋白 | 功能 |
|------|------|
| **Exportin-t** | tRNA 出口受体 |
| **Ran-GTP** | 能量依赖调控 |

:::warning
RNA 加工异常与多种疾病相关，包括癌症、遗传病和神经退行性疾病。了解 RNA 加工机制对于疾病研究和药物开发具有重要意义。
:::

## 7. 非编码 RNA 的加工

### 7.1 miRNA 加工

| 步骤 | 酶 | 产物 |
|------|-----|------|
| **转录** | RNA 聚合酶 II | pri-miRNA |
| **第一次加工** | Drosha + DGCR8 | pre-miRNA |
| **第二次加工** | Dicer + TRBP | miRNA 双链体 |
| **链分离** | -- | 成熟 miRNA |

### 7.2 siRNA 加工

| 步骤 | 说明 |
|------|------|
| **来源** | 长双链 RNA 或发夹结构 |
| **加工** | Dicer 酶切割 |
| **功能** | RISC 介导基因沉默 |

### 7.3 piRNA 加工

| 特征 | 说明 |
|------|------|
| **长度** | 24-31 nt |
| **来源** | 单链前体 |
| **功能** | 与 PIWI 蛋白结合，转座子沉默 |
