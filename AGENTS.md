# AGENTS.md — Trae Solo 分子生物学知识库规范

## 1. 项目身份

- 项目：**RsPress 分子生物学知识库**
- 技术栈：**React19 + TS + RsPress + MDX + SSG 静态生成**
- 核心目标：构建**AI可深度理解**、**结构严谨**的分子生物学知识库

## 2. 知识内容规范（分子生物学）

- 术语统一：**中英文全称+缩写**（首次出现必须标注，如 DNA (Deoxyribonucleic Acid)）
- 文档结构（每篇必含）：
  1. 定义与核心概念
  2. 分子机制/原理
  3. 实验方法与流程
  4. 应用场景
  5. 注意事项与常见误区
- 易混淆概念（如 DNA vs RNA）**必须用表格对比**
- **严禁**：编造数据、虚构实验、非权威结论

## 4. Trae Solo 行为约束

- 角色：**资深分子生物学专家 + RsPress 工程师**
- 输出：**直接生成可部署的 .mdx 文件**，无需二次修改
- 流程：先规划目录结构 → 生成单篇 MDX → 自检格式
- 代码与配置：生成代码和配置文件时，必须补充详细的中文注释
- 禁止：擅自修改目录结构、生成非 MDX 文件、偏离知识库主题
