# AGENTS.md — Trae Solo 分子生物学知识库规范

## 1. 项目身份

- 项目：**RsPress 分子生物学知识库**
- 技术栈：**React19 + TS + RsPress + MDX + SSG 静态生成**
- 核心目标：构建**AI可深度理解**、**结构严谨**的分子生物学知识库

## 2. 文档格式铁律（MDX + SSG 强制标准）

### 2.1 强制标题格式

- **一级标题**：`# 一、XXXX`
- **二级标题**：`## 1. XXXX`
- **三级标题**：`### 1.1 XXXX`
- **四级标题**：`#### 1.1.1 XXXX`

### 2.2 格式规则

- 标题层级严格对应 Markdown 层级：`#` → `##` → `###` → `####`
- 编号连续、不跳级、不重复、不乱序
- 段落之间空一行，重点内容加粗
- 英文术语首次出现必须标注中英文全称
- 步骤用有序列表，知识点用无序列表，对比用表格
- 使用 RsPress 内置组件：`:::tip` / `:::warning`
- 所有文档必须使用 **RsPress 兼容 MDX** 格式（.mdx 文件）
- **禁止**：复杂内嵌 HTML、非标准 Markdown 语法、混乱嵌套
- docs下文档对应的文件夹 & 文件名：**递增序号 + 中文**（如：1. DNA结构与功能.mdx）

## 3. 知识内容规范（分子生物学）

- 术语统一：**中英文全称+缩写**（首次出现必须标注，如 DNA (Deoxyribonucleic Acid)）
- 文档结构（每篇必含）：
  1. 定义与核心概念
  2. 分子机制/原理
  3. 实验方法与流程
  4. 应用场景
  5. 注意事项与常见误区
- 易混淆概念（如 DNA vs RNA）**必须用表格对比**
- **严禁**：编造数据、虚构实验、非权威结论

## 4. Trae Solo 行为约束

- 角色：**资深分子生物学专家 + RsPress 工程师**
- 输出：**直接生成可部署的 .mdx 文件**，无需二次修改
- 流程：先规划目录结构 → 生成单篇 MDX → 自检格式
- 代码与配置：生成代码和配置文件时，必须补充详细的中文注释
- 禁止：擅自修改目录结构、生成非 MDX 文件、偏离知识库主题

## 5. RsPress 构建规则

- 开发命令：`npm dev`
- 构建命令：`npm build` (SSG 静态站点生成)
- 目录：文档存放于 `docs/`，按分类建子文件夹。支持 多级文件夹层级架构。

### 多级层级结构示例

```
docs/
├── 1. 基础知识/
│   ├── 1.1 分子生物学/
│   │   ├── 1.1.1 化学基础.mdx
│   │   └── 1.1.2 大分子结构.mdx
│   └── 1.2 细胞生物学/
│       ├── 1.2.1 细胞结构.mdx
│       └── 1.2.2 细胞周期.mdx
├── 2. DNA与RNA/
│   ├── 2.1 DNA/
│   │   └── 2.1.1 DNA结构与功能.mdx
│   └── 2.2 RNA/
│       └── 2.2.1 RNA结构与功能.mdx
└── ...
```

**命名规则**：
- 文件夹与文件名均使用**递增序号 + 中文**格式
- 不同层级用不同序号位数区分：一级(`1.`)、二级(`1.1`)、三级(`1.1.1`)
